PDF Instructions

The following are the instructions for configuring and building a PDF target.

  1. Make sure the first page of each chapter has only a H1 and insert a mini toc. If there is content in this topic, copy and paste it into a new topic (see step 2). Drag and drop the ChapterMiniToc file from the Content Explorer in Resources > Snippets > PageLayouts .
  2. Right click the topic and set the properties as follows:

  1. The next topic (no heading displayed in PDF) contains the content from your first chapter page and no heading (delete the heading if it's there). This topic should be named "<topic content> Content" for consistency and so you can easily recognize it.
  2. Right click the topic and set the properties as follows:

  1. Open the Sample PDF TOC copy the front matter topics and paste into your TOC.
  2. Move the topics to where they belong.
  3. When you build the PDF target set the PDF conditions to include print and exclude screen.
