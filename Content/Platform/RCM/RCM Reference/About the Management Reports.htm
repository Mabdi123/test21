About the Management Reports

The following management reports are available for use within the : Alerts Alert Activity Details: Displays details on audit trail and notes of the selected work items.

  • Alerts: Alert Trends Daily: Provides a historical summary of alert distribution over a selected date range. The data is detailed by work item date within a specified business unit.
  • Alerts: Alert Trends Monthly: Provides a historical summary of work item distribution over a selected date range. The data is summarized in monthly buckets within a specified business unit.
  • Alerts: Alert Trends Weekly: Provides a historical summary of work item distribution weekly buckets within a specified business unit.
  • Alerts: Closed Alerts by Owner: Displays statistics on closed work items by owner. Statistics include counts of work items by issue/non-issue/no determination, average days open and average score. User selects a range of work item dates for the report. Average days open is calculated as total days work items are
    opened / total number of work items.
  • Alerts: Closed Alerts by Type: Displays statistics on closed work items by work item type. User selects a range of work item dates for the report.
  • Alerts: Open Alerts by Owner: Displays statistics on open work items by owner. You also have the option to filter via the work item type.
  • Alerts: Open Alerts by Type: Displays statistics on open work items by work item type. You also have the option to filter via the owner.
  • Alerts: Score Distribution Detailed by Alert Type: Displays work item counts on all distributed work items grouped by score in buckets of 10. All data is reported on a specific business unit over a selected Date Range.
  • Cases: Case Activity Details: Provides a full audit history on cases with the user option to show/hide notes and audit details. Filters can be used to reduce the initial report size.
  • Cases: Closed Cases By Type: Provides summary statistics on all closed cases that were opened during a specific date range.
  • Cases: Open Cases By Type: Provides summary statistics on all currently open cases by case type.
  • Many reports provide links to drill-down pages that can be accessed from graphs or charts, enabling managers to view more detailed information.
