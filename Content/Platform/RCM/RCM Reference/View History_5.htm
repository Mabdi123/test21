View History

Selecting the View History link in the More menu of the Case Items toolbar switches to the View History window for any selected item. This feature allows you to view all actions and updates that were performed for the selected case item. For example, if a user changes the step of a case item or opens a case item, the type of action appears within the Action column and the details about the action appear within the History Line column.

This action may appear in the toolbar. Buttons and action links are customizable depending on your organization implementation.

The Work Item History includes the following columns, by default:

Option

Description

Date

Shows the date when the action occurred.

Action

Shows all actions that were performed on this case item. For example, if you use the Change Step action, this information appears here.

Performed by

Shows which user performed the action.

History Line

Shows more detailed information about the action. For example, if someone assigns this work item to a different user, the details (previous user and new user names) appear here.
