Totals for Open Work Items

This report displays the total number of open work items and the specific number of open work items for each category. It includes the following types of work items: work items, DART, and Ad-hoc Blotters.

The data includes:

  • Open Work Items – number of work items that are in an open state for all categories
  • Closed Work Items - number of work items that are in a closed state for all categories
  • Assigned Work Items – number of work items that have an assigned owner
  • Unassigned Work Items - number of work items that do not have an assigned owner
  • Items in Progress – number of work items that are not in their initial state
