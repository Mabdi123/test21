Sort Lists

You can sort most lists according to one column heading by clicking on the required heading. The screen updates and displays the list according to the sorting order that you request and an arrow appears within the column heading to indicate the sorting preference. Click the same heading again to toggle the sorting preference between ascending and descending order.

The sorting request affects all pages of the current view.

You can define multiple sorting levels for the Work Items List by using an Advanced Sort option (see Advanced Sort). When multiple sorting is in effect you can view the full sorting order within a tool tip that appears when you hover your mouse over the column that has the sorting indicator
