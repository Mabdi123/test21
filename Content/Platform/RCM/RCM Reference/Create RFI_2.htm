Create RFI

The Request for information (RFI) feature allows you to collaborate with external and organizational users via e-mail. It provides you with the ability to select a case and send an email for more information. Once the email is answered it is attached to the case.

This action may not be enabled by your organization and therefore may not appear in the toolbar.

To create an RFI request:

  1. Select a case and click on the Create RFI button in the toolbar.
  2. The Create RFI window appears.

    3. It includes the following fields:

    • RFI Name: Enter a name for the RFI.
    • To, CC, Bcc: Enter the email addresses of the recipients in one or more of these fields. These fields are identical to email message fields. Use a ';' to separate between addresses.
    • Subject: Enter a subject for the RFI. Identical to the subject of an email message.
    • Message: A text box for entering data to the recipients on the RFI. Identical to the body of an email message.

  1. Click the Attachments button to view and add specific case attachments to this RFI. Click OK. Note that you cannot add URL based attachments.

  1. Click Send.
