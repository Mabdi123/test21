Case Contents Subtab

This subtab contains all related work items, notes and documents that have been added to the case.

Sorting

Sorting is available for each list in this subtab.

  • Click a column heading to select this column as a sorting condition.
  • Click a second column heading to indicate that you want to use this as the highest sorting level. Each time you click an additional column heading, the newly selected column is used as the highest sorting level.

When you hover your mouse over the column headings in the list, a tool tip provides information about the sorting order that is in effect, as shown below:

To remove sorting conditions, return to the Case List page and click x in the filter box to remove the search filter and update the display to show all cases within the list, with no filter applied.
