Work Item Indication

A work item that appears in bold characters indicates to the item owner that it has not been read yet.

A work item appears as read when one of the following actions occurs:

  • It is assigned to a specific user and is viewed by that user.
  • It is unassigned and is viewed by any user.
  • Adding a note to the work item.
  • Changing the score of the work item.
  • Adding the work item to a case.

Your organization may also configure the to return a work item that has been read to an unread status when one or more of the following actions occur:

  • One of the field values in the work item have changed either by a user or due to an updated calculation of a value.
  • A child work item is added or removed.
  • A work item is added to a case.
  • A note or attachment is added (this does not include confidential notes or attachments).
  • Sending or receiving RFI messages.
