The More Options Tab

The More Options tab enables you filter according to the creation date of the item as well as search for items related to cases and case items.

Option

Description

Create Date

Enter the range of the date the case item you want to search for was created. Use to select a date.

Note

View only items that contain a note with the specified text. The case sensitivity of this search depends on the database collation. If the collation is case sensitive, the search will be case sensitive, and vice versa.

Contained in Case

Search for a work item related to a specific case.

Contained in Case Item

Search for an item related to a specific case item.

Exact Match

Select this checkbox to search only for items that match the exact filter criteria exclusively.
