Act as Another User

The Act as Another link appears when a delegation has been defined for you by another user. (See Delegate to Another User). Click on the Act as Another User link of the User Info window.

The Act As Another User window opens. If more than one Delegation has been defined for you then select a user from the drop-down list and click OK. If only one delegation has been defined for you, then click OK.

The upper right corner of the screen indicates the name of the user whose credentials are used. A related icon also appears with the tooltip that provides details on this session.

To go back to using your credentials click the Switch Back link of the User Info window.

Two Step Delegation Chains

A two step chain delegation is when a user changes the user credentials via the User Context application link and then changes it again to a third user.

This feature is available only to users that have been provided with the relevant permissions and delegations by the administrator. For details on delegation, substitution and acting as another user, see Application Links and Options .

The following two step delegation chains are available:

  • A user that is in substituting mode can act as another user.
    For example:
    user 10 substitutes for user 11 and then acts as user 12.
  • A user that is currently acting as another user can substitute for another user.
    For example:
    user 10 acts as user 11 and then substitutes for user 12.
