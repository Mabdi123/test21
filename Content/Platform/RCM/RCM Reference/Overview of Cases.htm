Overview of Cases

Typically, a work item that requires further examination is attached to a case. Similar work items, notes, and attachments can then be added to a case.

The Cases tab opens a module, which shows cases that the currently logged on user (the user name that appears on the far right side) has permission to view.

The Cases module appears only if the module has been enabled by RCM implementer and you have the required permissions to view it.
