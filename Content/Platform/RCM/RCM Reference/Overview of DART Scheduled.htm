Overview of DART Scheduled Queries

As a feature of the Detection and Research Tool (DART), authorized users are able to not only run DART Queries interactively, but also schedule queries to run once or on a recurring basis.  The Scheduled Queries Management is used to view and maintain these schedule queries, specifically it enables users to:

  • Quickly find a scheduled query via the Filter by field.
  • View the original query
  • Delete a scheduled query.

To access the DART Scheduled Queries feature, click the Administration link . In the Schedule and Monitor tab click DART Scheduled Queries.
