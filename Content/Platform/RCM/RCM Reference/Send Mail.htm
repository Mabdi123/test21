Send Mail

Sends a link to email recipients to enable them to access work items details for selected work items.

When you send work items details for more than one work item, the recipient clicks the link within the email and the first page of the work item details opens. The recipient can use the standard browse options, available within the Work Items Details page, to view the next work items.

The recipient must be an authorized user and must have permission to view these work items details.
