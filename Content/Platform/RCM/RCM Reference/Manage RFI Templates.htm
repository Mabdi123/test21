Manage RFI Templates

This feature enables you to create and manage RFI (Request for Information) templates for use with work item email correspondence.

To access managing RFI templates:

  • Click the bar to display the activity bar with the associated actions.

The information on the right side of the window updates to show the currently defined list of created templates.

The possible columns for the RFI templates list are:

Option

Description


(checkboxes)

Click on a checkbox (or several checkboxes) to perform an operation on the selected template(s) (for example, Edit RFI Template). Click the checkbox in the title bar to select all templates in the list.

Template name

The template name that is entered by the system administrator.

Identifier

The description of the template as entered by the system administrator.
