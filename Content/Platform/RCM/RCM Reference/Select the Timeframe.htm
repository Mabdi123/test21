Select the Timeframe

The timeframe filter provides the following options:

Option

Description

Between dates

Requires filling in two additional fields that enable you to enter dates. The result displays all the records that occurred between the specified dates.

Rolling Prior Week

The result displays all records that occurred in the last seven days.

Rolling Prior Month

The result displays all records that occurred up to a month ago from the current date. For example, if the search is made on Feb. 23, then results are displayed for all the activity between Jan. 23 and Feb 23.

Rolling Prior year

The result displays all records that occurred from the current date to 365 days ago.
