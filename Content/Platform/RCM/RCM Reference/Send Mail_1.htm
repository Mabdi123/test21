Send Mail

Sends a link to email recipients to enable them to access case items details for selected case items.

When you send case items details for more than one work item, the recipient clicks the link within the email and the first page of the case item details opens. The recipient can use the standard browse options, available within the Case Items Details page, to view the next case items.

The recipient must be an authorized user and must have permission to view these case items details.
