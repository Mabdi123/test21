Validate a Rule

Before running a rule you must verify that it is valid. Initially the rule's validity is unknown. A rule's validity may also be unknown after a metadata update.

Click the Validate button in order to verify that the rule is valid. When the rule is valid, the rule validation bar's color changes to green and its validation is indicated

When the rule is invalid, the rule validation bar's color changes to orange and the specific reason for the rule's invalidity is displayed.

You cannot run rules that are invalid.
