Indications on the Status of the Query Results

The message Results are outdated is displayed and the Go button’s color changes to a darker blue in the following scenarios:

  • Condition(s) have been added but the query’s results are not displayed.
  • A saved query’s conditions have been modified but the query’s results are not displayed.

When the Go button is clicked, the message Results are outdated is removed from the screen and the Go button’s color changes back to its original color.
