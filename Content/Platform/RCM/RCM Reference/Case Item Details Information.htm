Case Item Details Information

The information available within the Case Item Details page depends on the type of solution that was run and the requirements of your organization.

The Case Item details page contains the following:

Element

Description

Header

Displays the case item name and relevant information.

Summary

Displays additional fields for the case item and additional specific information that is appropriate for the case item type of this case item.

Description

The item description. This can be edited in the Edit Item window. For details see Edit Item.

Content Tab

Contains a list of the content in the case item. For details see Content Tab.

Case Work Items Tab

Contains a list of the items added to this case item. For details see Case Work Items Tab.
