Application Links and Options

The following links appear in the upper right corner of the application main window:

Link Description
Administration
  • Click on the Administration link to open the Admin module, which provides options for authorized Administrators to manage users, groups and roles, and to assign permissions. (See Admin)
About

Shows the following information:

  • application and database versions.
  • Current version of your application.
  • License details:
  • Customer name: indicates the name of the company that owns the license.
  • License use: indicates the identifier string for this license.
  • License version: indicates the license version number
  • Named users: indicates the maximum number of registered users.
  • Expiration date: indicates the date when the license expires.
  • List of optional licensed modules and the licensing step of each module (licensed/not licensed). Optional modules are purchased separately by your organization.
Help

Opens the Online Web Help.
Current User Name

Clicking your displayed name link or the arrow opens the User Info window that includes the following:

Change Password: Enables you to change the password

Delegate to Another User: See Delegate to Another User

Sunstitute for Another User: See Substitute for Another User

Personalization: This link lists your default view and enables you to perform layout personalization options. For details, see Personalization Feature.

Logout: Enables you to log out of the application. An authorized user can log in again with a different user name and password.

 

 Change TenantThe license for this feature is purchased separately. This link appears only if you are in a Multi Tenant mode. Click on the Change Tenant link of the User Info window.

Enter the tenant name or select one via the picker . If the tenant is not linked to a business unit then click the Advanced icon and select a business unit for the tenant.

 

 Act As Another User

Appears when a delegation has been defined for you by another user. See Act as another User.
