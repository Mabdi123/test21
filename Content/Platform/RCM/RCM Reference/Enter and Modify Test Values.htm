Enter and Modify Test Values

The Test Values tab displays the solution’s default system values and user-defined values for each setting. When a user-defined value was not defined, the test uses the default system value for the test. When a user-defined value was defined but not tuned, the test uses the original user-defined value. When a user-defined value is tuned for a given test, the test uses the tuned setting value.

All tuned values are applied to the test only and will not overwrite any user-defined values.

To enter or modify test values:

  1. From the RCM application, select Settings tab à Tuning tab à Test Values tab.
  1. Select the setting you need to tune and click Edit.
  1. The Edit Platform List Test Item window for the selected setting appears.
  1. In the Risk Level Value – User field, enter the tuned value.
