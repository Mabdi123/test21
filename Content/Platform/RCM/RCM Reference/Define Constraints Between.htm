Define Constraints Between Events

The default constraint between two Events is then. This constraint can be defined with additional options. To view these options click on the then link.

The following constraints are available:

  • For a same: The two Events must have the same value in the selected field.
  • For a different: The two Events must have different values in the selected field.
  • Within X hours/days: The two Events must have occurred within the defined time frame
  • After more than X hours/days: The two Events must have occurred after the defined interval
  • After X to Y hours/days: The two Events must have occurred after a defined interval in a specific time frame. For example: <then> after 5 to 6 hours <the 2 Events occurred>

You can define more than one constraint between two Events by clicking the Add Constraint link.
