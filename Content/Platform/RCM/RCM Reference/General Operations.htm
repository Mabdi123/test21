General Operations

After opening a report set in JReport Web, you can perform the following general operations:

  • Managing reports

    • A report set can include one or more reports. The Report Set bar is displayed at the top of the report area, listing the display names of all the open reports in the current report set. Clicking the display name of an inactive report will make it active. You can manage reports in a report set easily:

  • Opening and closing a report

    • In a report set, a report can be shown or not. To close (hide) the active report, click File > Close Report, or right-click the report display name and then select Close, or directly click in the tab. If there is one or more reports open other than the active report, the close action will hide the active report; in the case that the active report is the only one report open, the close action will prompt you whether or not to close the report set.

    To open (show) a hidden report, click File > Open.. (or the Open button ) to display the Open Report window, in which the reports opened in the current report set are marked with a check symbol. Check the reports you want to open, uncheck the ones you want to close, and then click OK.

  • Turning the report pages

    • If a report includes more than one page, you can click the first page icon , previous page icon , next page icon , or last page icon to turn between the report pages. You can also input a number into the page box and press Enter to go to that page. In addition, you can use the Turn To submenu on the View menu to accomplish the same purpose.

  • Refreshing the report result

    • To fetch the data of the current report again, you can click View > Refresh, or click the Refresh icon in the relevant view.

  • Undoing/redoing actions

    • You can undo or redo some actions by clicking Edit > Undo or Redo (or the Undo button or Redo button in the relevant view).

  • Setting JReport Web options

    • JReport Web allows you to set the skin and customize toolbars.

To set JReport’s Web Options:

  1. Click View > Options... (or right-click anywhere on the toolbar area and select Options... from the shortcut menu).
  2. In the Option tab, set the skin of JReport Web user interface.
  3. In the Customize tab,
    1. To modify a toolbar, select it in the current Toolbar box, remove those unnecessary items from the Selected Tools box, and add required tools from the Available Tools box. Click or to adjust the order of the tools on the toolbar.
    2. To add a toolbar, click to show the New Toolbar Name window, then specify the toolbar name, click OK to return to the Options window, and set the tools for the new toolbar.
    3. To delete a toolbar, select it and click .
    4. To load the default settings, click the Restore Defaults button.
    5. Click OK to apply the settings.
  • Tuning report page magnification

    • You can zoom in or out the report page by selecting a magnification from the zoom button and options in the Basic view. You can also click View > Zoom... to open the Zoom window, and then specify the magnification.

  • Asking for help

    • At any time, you can click the Help icon on the Basic toolbar to open the index page of JReport Web User's Guide. Furthermore, click the Help button in any dialog to show the online help about the dialog. You can also use the Help menu to open the User's Guide and access Jinfonet Software website for more information.

  • Setting up the page

    • To set up the report page, click File > Page Setup... (or the Page Setup icon on the Basic toolbar). In the Page Properties window, specify the page type, the page panel to set, the orientation, and the margins as required.

  • Printing the report result

    • Print the report result to a PDF file. To do this, click File > Printable Version... (or the Printable Version icon on the Basic toolbar. In the Printable Version window, specify the pages to be printed, and then click OK. The PDF result file will open in an associated program. A PDF viewer plug-in (e.g., Adobe Reader) should be installed in order to properly view any item that has been exported as a PDF file.

  • Exiting the report set

    • To close the current report set and release the resources, click File > Exit (or the Exit icon which is always on the right of the toolbar area). Closing the only report open will also prompt you whether or not to close the report set. If you have modified the report set without saving it, JReport Web will prompt you to save the report set. If you have changed the sort and/or filter criteria, you can check Sort and/or Filter in this window to save these changes with the report set. Click Yes to save the report set and close the report set.

Closing the browser window of JReport Web is not recommended, because in this way, JReport cannot detect the report modifications so as to prompt you to save the changes and JReport will not release the resources right away.
