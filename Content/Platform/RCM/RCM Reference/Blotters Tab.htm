Blotters Tab

The following is an example of the Blotters tab in the New Role window.

Select the Blotters Type or all Blotters in the Blotter drop-down menu.

To add a permitted action, select one of the following actions from the Actions drop-down menu and click the Add button;

  • All - enables users with this role to have all the available permissions for Blotters.
  • Add free text Note to Blotter- enables users with this role to add text to a note for a Blotter.
  • Add free text note to Trade – enables users with this role to add text to a note for a Trade of a Blotter.
  • Add Predefined note to Blotter – enables users with this role to add a predefined note for a Blotter.
  • Add Predefined note to Trade – enables users with this role to add a predefined note for a Trade of a Blotter.
  • Assign to BU – enables users with this role to assign a Blotter to a business unit.
  • Assign to User – enables users with this role to assign a Blotter to a user.
  • Change to under review - enables users with this role to change the step of a Blotter to Under Review.
  • Create Alert - enables users with this role to create a work item from a Trade of a Blotter.
  • Delete Filter - enables users with this role to remove a Trade filter.
  • Delete Private Filter - enables users with this role to remove a private filter.
  • Export - enables users with this role to export a Blotter as a CSV file.
  • Materialize Blotter – enables users with this role to materialize a Blotter into a file that can be archived.
  • Print Blotter Trades – enables users with this role to print Blotter Trades.
  • Review Trade – enables users with this role to mark a Blotter Trade as Reviewed.
  • Save as Alert - enables users with this role to create an work item from a Trade from the Ad-Hoc Blotters module.
  • Save Filter - enables users with this role to save a filter.
  • Save Private Filter – enables users with this role to save a private filter.
  • Sign off – multiple selection - enables users with this role to sign off one or more Blotters at a time.
  • Sign off – single selection - enables users with this role to sign off one Blotter at a time.
  • Un-review Trade – enables users with this role to mark a Blotter Trade as un-reviewed.
  • Update Filter - enables users with this role to update a filter.
  • Update Private Filter – enables users with this role to update a private filter.
  • View Blotters – enables users with this role to view Blotters.
  • View History – enables users with this role to view the history of actions on a Blotter.
  • View Notes – enables users with this role to view the notes of a Blotter.
  • View Trade History – enables users with this role to view the history of actions on a Trade of a Blotter.
  • View Trade Notes – enables users with this role to view the notes of a Trade
  • View Trades - enables users with this role to view the Trades of a Blotter

You have the option to set an action to one of the following in the Assigned to drop-down menu:

  • Any: Blotters that are assigned to any user. This is the default setting.
  • Assigned to me: Blotters that are assigned to you.
  • Assigned to others: Blotters that are assigned to other users.
  • Unassigned: Blotters that are not assigned to any user.

To add another permitted action, select it and click the Add button. You also have the option to select another Blotter Type and set the desired permitted actions for it.
