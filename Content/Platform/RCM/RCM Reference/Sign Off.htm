Sign-Off

Select a Blotter and click the Sign-Off button within the Trade Blotter toolbar to open the Sign-Off Blotter window.

The RCM may be configured to allow signing off only if all trades have been reviewed. For more information about reviewing trades, see Review.

This window includes the following information and options:

  • Total trades: The total number of trades for this Trade Blotter.
  • Reviewed Trades: The number of trades that have been reviewed
  • Alert on trades: The number of work items that are attached to Trades in this Trade Blotter.
  • Add Note: Enables you to add a predefined or free text note that is attached to the Trade Blotter.

Click OK and a green check is displayed next to the Blotter.
