Define Local Admin Users

Some organizations require distributing the user admin workload across multiple local administrators who have control over users within their department. Local administrators are able to manage only users within their own departments and have no user admin privileges in other departments.

Associated terms for this feature:

  • Global administrator: Has control of all user administration objects in the and can function as a backup to the local administrator. This administrator can perform all actions on roles, users and groups.
  • Local administrator: this user can add, change and delete users and groups assigned to departments under their responsibility only. The roles are maintained by global administrators only and the local admin can only associate them only to their users/groups, and only if they are related to the department that is their responsibility.
  • Department: Department is the smaller unit for which one or more local administrations can be assigned. A department is a collection of BUs, users, groups and their attributes like their associated roles. Note that a department may have multiple local administrators and a local admin may have multiple departments.
