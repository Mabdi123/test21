Create an Internal Sub Query

An internal sub query is a nested query whose conditions are defined during its creation. It is a set of conditions that is embedded in the containing query and cannot be referenced from other queries.

To create an internal sub query:

  1. In the condition operator drop-down, select In Sub Query as the filter for the condition.
  1. Select <Internal> as the sub query and click the Edit link.
  2. Define the internal query conditions and once you are done click the button.
  3. The main query and its internal sub query appear on the screen.

You can create an internal query for another (parent) internal query.
