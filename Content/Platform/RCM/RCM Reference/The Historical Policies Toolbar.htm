The Historical Policies Toolbar

The Historical Policies Toolbar includes the following options:

Option

Description
Filter

Enables you to filter the rules that you want to view. After clicking the Filter icon, the Advanced Filter window opens. Enter the relevant filtering criteria in the fields of this window.

Tags must be entered in full. For example, if there is a tag named not NY branch, then if the user enters not NY then the tag is not properly filtered.

The State field has the following options:

  • New: displays only new rules for this policy
  • Changed: displays only the rules that have been changed for this policy.
  • Unchanged: displays only the rules that have not been changed for this policy.
  • Deactivated: displays only the rules that have been deactivated for this policy.
Reset

Disables all the filtering criteria.
Refresh

Refreshes the data on the screen.
Activation History

Opens the Activation History Report window. For more information refer to the previous page.
