Layout Options

The personalization layout options are as follows:

  • Save Layout: Click to personalize your layout after making changes to the view.
    • More Actions:
    • Return to Saved Layout: Click to return the view to the saved layout.
    • Return to Default Layout: Click to return the view to the layout designated as the default.
  • Set as Default View: Click to set the current view as the default.
  • Reset Layout: This button appears when the personalization parameter is deactivated by the implementer in the Designer. It resets the view to the Designer configurations. You can then make changes but cannot save them.
