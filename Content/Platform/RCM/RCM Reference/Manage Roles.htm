Manage Roles

This feature enables the administrator to manage roles and permissions for users.

To access managing roles:

  1. Click the bar to display the activity bar with the associated actions.

The main window displays the currently defined list of roles and associated information for each role.

The possible columns for the roles list are:

Option

Description


(checkboxes)

Select a checkbox to perform an operation on the selected roles (e.g., Update Role).

Role name

The role name that is entered by the system administrator.

Description

The description of the role as entered by the system administrator.
