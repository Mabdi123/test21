View Notes

Select a Trade Blotter and click the View Notes button within the Trade Blotter toolbar to open the View Notes window. This enables users to view the history of all notes associated with the Trade Blotter.

This action can only be performed on one Trade Blotter at a time.

The View Notes window includes the following information and options:

Option

Description

Enables you to add a note to the current list of notes.

Note Date

The date and time that the note was added.

Entered by

The name of the user who added the note.

Related Action

The action that the user was performing when the note was added.

Notes can be added to a Trade Blotter while performing the following actions.

You can add notes from the general Trade Blotter list by selecting any Blotter and then performing one of the following:

  • Add Note: you can add a note directly from the Trade Blotter List or while viewing the history of notes.
  • Sign off: you can add a note when signing off a selected Trade Blotter.
  • Assign to User: you can add a relevant note about this action from the Assign to User window.
  • You can add a note to a specific Trade in the Trade Blotter when you select a specific trade within the Blotter Detail window and change a Status of the trade to Review/Un Review.
  • You can also add notes from the Blotter Details window for any Trade Blotter by selecting Add Note or Sign off at the top of the Details window.

Note

The text of the note.

Notes that are added after Sign Off appear within the View Notes window with blue highlight.
