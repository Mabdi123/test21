The Drawing Canvas

You have the option to increase the viewing size of the canvas by clicking on the Enlarge icon () that is at the top right corner of the VLA window.

The drawing includes the following objects:

  • Node: A visual element representing a logical/physical entity. A drawing includes a number of node types. For example, your organization may display the following node type to represent an account:

Your organization may also include unique nodes that are relevant to its operations. In order to view the name of a Node it is recommended to be at a zoom level of 50 percent or higher. Hovering over a specific Node displays a tooltip that provides more information on it.

Right-click the mouse while hovering over a node may provide a menu for more options. For example, right-clicking the mouse on the following node provides a menu that enables the user to view various related queries:

This menu may also include links to additional graphs that are related to the selected Node.
To move a node, click on the selection arrow of the drawing toolbar and select a specific node. The node can be moved anywhere in the canvas while its relationships and links are maintained.

  • Link: Visual element representing the relation between entities. A drawing may include more than one type of link where each type of link has a unique display and indicates a specific type of relationship. Each link includes text that describes the relationship as shown in the following example:


In order to view the text it is recommended to be at a zoom level of 50 percent or higher.

  • Network: A set of nodes, links and sub graphs that represent a specific solution. There can be more than one Network for a specific drawing.
