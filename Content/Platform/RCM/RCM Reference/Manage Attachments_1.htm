Manage Attachments

The following describes functionality that may not be available depending on your organization’s configuration.

Click the Attachments button in the Case Items toolbar to open the Attachments window.

The Attachments window provides a detailed list of all the attachments for the item. It also enables you to add or remove attachments.

The Attachments window includes the following options:

Add a File

You can add a file as an attachment by clicking the Add File button in the Attachments window.

To add a file:

  1. Click the Add File button and Browse to select the file that you want to add. You can also use Ctrl-Shift when browsing to add multiple files at once. When added, the new icon appears next to the files in the grid.
  1. Select a single file checkbox to add a description.
  1. Click Save to save the attachments to the item. The attachments are added and the window closes automatically.

Add a Link

You can add a link as an attachment by clicking the Add Link button in the Attachments window.

To add a URL:

  1. Click the Add Link button and enter the following in the window that appears:

    • Text to Display – enter a name for the link that you want to appear in the attachments grid. Text cannot be more than 100 characters.
    • URL – Enter the URL of the link (for example, http://www.actimize.com). Length of URL cannot exceed 500 characters.
  1. You can optionally click Test this link to open a separate window of the URL.
  1. Click Add to add the link to the attachments grid.

Drag and Drop Files

You can drag files to the Attachments window in order to attach them to an item.

To drag and drop:

  1. Navigate to the file you want to add as an attachment.
  1. Drag the file to the attachments grid.
  1. The file is added to the attachments grid list.

Paste an Image

You can copy and paste an image in the Attachments window.

To paste an image:

  1. Copy an image using Print Screen.
  1. Press Ctrl-V to attach the image to the item.

  1. The image is added to the attachments grid list. You can optionally select an image and add a description.

Pasting an image is not supported in Safari.

Confidential Attachments

You can specify an attachment as confidential as follows.

To add a confidential attachment:

  1. Select an attachment in the attachments grid.
  1. Select the Confidential Attachment checkbox.

The icon is displayed next to rows that have a confidential attachment.

If the confidential feature is not enabled or you do not have the relevant permissions, you cannot enable or view confidential attachments.

Save Attachments

After you have added attachments to the grid, the new icon  is displayed beside each one. Click Save to save the attachments to the item. Click Cancel to close the Attachments window and discard any added attachments before saving.

You can view the date and time of the attachment as well as the user that added the attachment.

Clicking outside the Attachments window automatically closes the window. All attachments that have not been saved will be lost.
