Browse and Step Options

A navigation and step bar, at the top of the window, includes the following options:

Option

Description

Enables you to return to the page from which you accessed the Case Item Details. For example, if you accessed the Case Items Details from the general Case Items List, this takes you back to this list. If, however you accessed the details from a link within a different case item, this takes you back to that item page.

Clicking the arrow opens a drop-down menu with the name of the previous page that you viewed. Click this name in order to move to that page.

Provides information about the step of this case item. The two icons show whether the case item is open or closed, and whether it is an issue, non-issue or undetermined. (For details about case item steps, see Change Step).

Click to open the View Notes window and view the notes associated with the case item. This icon is only displayed when the case item has notes attached to it and you have permission to view them, (for example, viewing confidential notes). For details see View Notes.

If a file is attached to this case item, a paperclip icon appears in the navigation and step bar. You can click the paperclip to view the links of the attached files. 

The current Item ID.

Indicates the current case item that you are viewing in relation to the total number of items that are within the current view.
