Entity Matching List Thresholds

In order to determine the final decision on whether an incoming entity matches the existing entity, you can set threshold scores for the levels of a match and a doubt on the incoming entity.

Click Edit to configure the relevant list item. The Entity Platform List Item window opens.

The default settings are as follows.

  • Entity Identifier: The entity name. This parameter cannot be changed.
  • Max Doubt Threshold: The threshold score (80 by default) that is the maximum score that is designated as doubt. By default the score threshold for a match is from 90-100. The score for no match is from 0-10.
  • Min Doubt Threshold: The threshold score (20 by default) that is the minimum score that is designated as doubt. The incoming entity with a score from 20-80 is placed in a state of doubt.

It is recommended that only the administrator have permissions to edit the Entity Matching list thresholds.
