Select and View a Data Source

You must select the relevant data source for your query.

To select a data source

  1. Click the arrow next to the Data Source field and choose the data source for the query.

    The data source is displayed as the page heading and the data of this source appears on the screen.

    If the data of the source goes configured row limit of the window then the source is presented in multiple pages.

  1. Use the next and previous page options to jump to the next or previous page.
  1. Click the button to jump directly to the first page. You also have the option of entering a specific line number in order to view its data. For example, entering 89 and pressing the Enter key moves the Data Source to the 89th line.
