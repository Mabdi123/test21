Add External Group

When an external method of authentication is used, such as LDAP, the Administrator must ensure that the includes all relevant groups that are maintained within the external server.

Click the Add External Group link within the Manage Groups activity bar to open the Select External Group window that enables you to search for the LDAP group that you want to add to the .

To select an LDAP group:

  1. Enter the name, or any letter that is included within the name, and click Go to search through all LDAP groups for the group that you require. The list of all LDAP groups that contain the entered letter(s) appears within the Available Groups list box.
  1. Select the group that you want to add to .
  1. Click Next to add it to the main list of groups.

You cannot assign users to external LDAP groups since this information is based on the LDAP server configuration.

Once you have added all LDAP groups to the list of groups, you can continue by synchronizing the user list with the relevant LDAP users. The list of users must be synchronized with those users who are associated with the LDAP groups that you added to the . This can be performed using:

  • The Manage LDAP à Synchronize action. This feature enables you to automatically synchronize the list of RCM users with the LDAP list. The RCM list is synchronized to include all users who belong to the LDAP groups that were added to the . (For more information, see Synchronize List of Users with LDAP Users List.)
  • The Manage Users à Add External Users option. This enables you to select the LDAP users manually. (For more information, see Add External Group.)
