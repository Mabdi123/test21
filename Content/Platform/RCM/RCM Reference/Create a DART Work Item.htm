Create a DART Work Item

This feature provides you with the option to create a manual work item that is based on the results of a DART query. Using the DART collected data, you can save results as an work item.

Data sources that do not belong to a specific scheduled query group cannot be saved as an work item. In this scenario, the Create Alert icon is not displayed. Only The implementer has the option to associate data sources to specific scheduled query groups via the Designer.

To create a work item:

  1. Click the Create Alert button at the bottom of the condition pane. The Create DART Alert window opens.

  1. Populate the fields of this manual work item.
  2. Deselect the Attach Results to created Alert checkbox if you do not want the CSV file that contains the results to be created.
  3. Complete the wizard steps required and click Finish.
  1. Click the Open Item button to view the manual DART Alert.
