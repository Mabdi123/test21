Modules Tab

Enables you to select the checkboxes for the specific modules that you want displayed in the . This means that users, who are assigned the role that you are currently defining, can access that module.

You can use the Mark All Modules button in the upper right corner of the screen to quickly select all modules for the current role.

In the Work Items module, click a Select views link to expand the Work Items section and display all currently defined Work Items views.

Example:

You might define a role called Officers and enable access to all modules within the application with the exception of the Supervision and the Admin modules. In this case, these tabs would not appear at the top of the application for users who are assigned the Officer role, as follows:

If you assign the following Work Items views to this role:

  • My Documents
  • All Documents
  • My Work Items
  • All Work Items
  • My BU Items
  • Unassigned Items
  • Aging - 30 days
  • Closed Items
  • Archived Items

All users who are assigned the role Officer will be able to select the specified views from the following drop-down list within their Work Items List window.
