Open Cases

This report provides data about open cases for any selected case.

You can configure this report to retrieve the data that you want by using the By: drop-down menu that has the following options:

  • View is based on the category.
  • View is based on the case type.
  • View is based on the step.
  • View is based on the owner of the case.
  • View is based on the assigned business unit.

To choose the view, select the sorting option from the drop-down menu and click Go. The top ten values are then displayed.

The Drill Down Option

Click on a specific bar to view its drilled down data.

The results for the specific bar are then displayed. A second filtering menu enables you to select the view on a specific category, case type, step, or owner. Select the specific filtering option from the drop-down menu and click Go.
