Conditional Settings

Select the following actions for a specific Conditional Setting or all Conditional Settings:

get new

  • All - enables users with this role to use all the available permissions for the Conditional Setting.
  • Add - enables users with this role to add a record to the Conditional Setting.
  • Delete - enables users with this role to delete a record from the Conditional Setting.
  • Update - enables users with this role to update a record of the Conditional Setting.

To add another permitted action, select it and click the Add button.
