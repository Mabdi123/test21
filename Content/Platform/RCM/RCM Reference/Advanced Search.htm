Advanced Search

The Advanced Filter window enables you to display only reports that correspond to the search factors that you are interested in. (The criteria selection in this window uses the “and” operand.)

Click the Advanced Filter button to open the Advanced Filter window to search for specified information within the current view.

Use one or more of the following to search through the list of reports for specified information.

Option

Description

Report ID

The unique internal ID of the report.

Report Name

Enables you to search for a specified report name that interests you or to enter the first few letters or words to search for all reports that start with these letters or words. (for example, Highest will give you Highest trades in month, Highest trades in year etc.)

Status

Search for a specified Status. The Status can be:

  • New
  • Purged
  • Archived
  • Signed-off
  • Viewed

For more information about the different statuses, see Report List.

Generated Date: From

Search for report(s) that were generated on or after this date. You can search for a range of dates to view all reports generated within that range. This field enables you to enter the beginning date for a range.

Generated Date: To

Enter the final date for a range of dates that you want to use to filter the display of reports. All reports that were generated between the From and To date appear in the list of reports.

Generated By

Indicate which type of reports you want to view. Reports can be generated by:

  • The Designer (reports that are generated during implementation stage)
  • AIS Model (automatically),
  • On Demand by the users.

(For more information about generating on demand reports, see On-Demand Reports).

Business Unit

Enables you to search for reports that were distributed to a specified business unit.

Owner Name

Enables you to search for reports that are assigned to a specified owner. The owner is the user who is assigned responsibility for this report.

Sign-Off Date: From

Enables you to specify the start date when searching for reports that were signed off. This filter will display all reports that were signed off from this date until the Sign-Off Date: To (below) that is specified.

Sign Off Date: To

Enables you to specify the end date when searching for reports that were signed off. This filter will display all reports that were signed off from this date until the Sign-Off Date: From (above) that is specified.

Sign-Off User

Enables you to search for reports that were signed off by the specified user.

Availability

Enables you to search for reports according to their availability. The options are:

  • Online: The report is saved within the database.
  • Offline: The report was archived and purged. In order to access an archived report you must make a request to the System Administrator who will restore the report.
  • Not available: The report was purged and not archived.

Report Type

Enables you to search for reports of a specified report type.

Business Date: To

Enables you to enter the starting and ending business dates
