View Notes

Select a trade and click the View Notes button within the toolbar opens the View Notes window that enables users to view the history of all notes associated with the Trade.

This action can only be performed on one trade at a time.

The View Notes window includes the following information and options:

Option

Description

Enables you to add a note to the current list of notes.

Note Date

The date and time that the note was added.

Entered by

The name of the user who added the note.

Related Action

The action that the user was performing when the note was added.

Notes can be added to a trade while performing different actions.

You can add notes from the general Trade list by selecting any trade and then performing one of the following:

Add Note: you can add a note directly from the Trades list or while viewing the history of notes.

Sign off:  you can add a note when signing off a selected Trade Blotter.

Create Manual Work Item: you can add a note when creating a manual work item for a selected trade

Review /Un-Review: you can add a note when changing a status of a Trade to Review /Un-Review.

Note

The text of the note

View Notes that Include Tags

The View Notes window provides information on all the relevant tag values and enables you to use the tag values as links to more information about a specific value.

For example, when a user clicks on the John Smith Broker value, all the relevant notes about John Smith are displayed in all related work items and Transactions.
