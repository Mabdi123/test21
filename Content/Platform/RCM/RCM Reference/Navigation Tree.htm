Navigation Tree

Select the following actions for a selected branch and its descendants of the Navigation Tree:

  • All - enables users with this role to use all the available permissions for the Navigation Tree.
  • Add - enables users with this role to add a record of a setting that is under this branch of the tree.
  • Delete - enables users with this role to delete a record of a setting that is under this branch of the tree.
  • Update - enables users with this role to update a branch of the Navigation Tree.
