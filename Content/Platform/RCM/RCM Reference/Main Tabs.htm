Main Tabs

The tabs at the top of the window are defined according to basic categories. Some tabs include drop-down menus containing subtabs for modules. Some subtabs may not be required by your organization and therefore may not be configured to appear.

The following is a list of all standard modules that are available within the application.

Module Description
Cases

 

 

Enables you to group work items, notes, and various attachments under a single business case for further handling. (See Cases)
Workbench

 

The tab includes the following options:

Work Items – Shows the list of work items that the currently logged on user is authorized to view. Users can access Work Items Details from this window. (See also Work Items)

Case Items - A type of work item with cases capabilities. Case Item views show the list of case items for authorized users. See Case Items.

 

Outgoing Forms – Provides a list of all the forms that have been added to cases. It enables you to quickly search and find a specific form and its related cases (See Outgoing Forms.)

 

Reports – Enables authorized users to view and, when appropriate, to sign off management reports and statistical reports about work items, workload and other activities (See Reports).

 

Blotters – The license for this module is purchased separately. Enables authorized users to view and, when appropriate, sign off Trade Blotters. Each Blotter includes a link that enables users to view Trades within the Blotter. (See Blotters and Trades of Trade Blotter).

Research This tab includes the following options:

DART – Enables you to create, save, access, and run queries on external tables that reside outside the repository. See DART - Detection and Research Tool.

On-Demand Reports – Displays a tree showing the Report Types that support the generation of on-demand reports Enables you to generate a variety of reports from predefined report types (See On-Demand Reports).

Tuning – enables defining different thresholds and configurations in order to test them. (See Tuning).
Policy Manager

Enables you to set and manage further policy rules for work item detection. See Policy Manager.
Settings

This tab includes the following options:

Settings Navigator- Enables you to modify when and how work items are distributed and risk scores are derived. Its Settings Tree includes a folders hierarchy for the current solution that you are viewing. (See Settings Navigator.)

Thresholds – Enables users to enter threshold values that are used by AIS analytical objects when generating work items. The thresholds are usually presented according to Item type. (See Thresholds.)

Lists – Enables users to maintain lists that are factored into the AIS analytical process when generating work items, such as exclusion or inclusion lists. These lists are usually maintained according to work item type. (See Lists).

Platform Lists – Enables you to manage and maintain various thresholds and scoring as required by your specific solution. (See Platform Lists.)

References

Read only reference information that application end-users would find useful in the course of their work. Reference data may include processing statistics and definitions on standard reference codes. (Available when configured for your organization.)
