The Draft Policy Toolbar

The Draft Policy toolbar enables you to perform policy rule actions.

Some of the icons for the rules are only enabled when a rule is selected.

The following options are available in the Draft Policy toolbar:

Option

Description

Refresh

Refreshes the display of the rules.

Filter

Enables you to filter the display of the rules. For details, see Filter a Rule.

New

Enables you to create a new rule. For details, see Create a New Rule.

Duplicate

Enables you to duplicate a specific rule. For details, see Duplicate a Rule.

Edit

Enables you to edit a specific rule. For details, see Edit a Rule.

History

Enables you to view the changes that were made to a specific rule. For details, see View the History of a Rule.

Import

Enables you to import rules into the policy type directly from the . The imported APF file must include rules that are related to the Policy Type. For details, see Import Rules.

Reset

Enables you to reset the display of the rules table and remove any sorting criteria from the rules table. For details, see Sort the Table.
