JReport DHTML Window Screen Elements

The following information describes the basic options that are available when you generate a report using the On-Demand Reports – Generate feature. These topics were extracted from the JReport online help.

The main page of DHTML consists of the user information bar, menu bar, toolbar, report set bar, Toolbox, Resource View panel, TOC Browser, and report area. The options for browsing or controlling a DHTML report are as follows:

Basic View Toolbar

 

Export

Opens the Open Report window for you to open/close reports in current report set.

Displays the Printable Version window for you to print the current report result to a PDF file.

Undoes the last operation.

 

Redo

Reverses the operation of Undo.

 

Filter

Displays the Filter window, with which you can filter the report records according to the filter criteria you specify.

 

Sort

Displays the Sort window, with which you can sort the report records or groups in ascending or descending order on the field(s) you select.

 

Search

Displays the Search window for you to find specific text.

Enables you to enlarge or reduce the size of the report. Click the arrow to display sizing options.

Goes to the first page of the current report.
