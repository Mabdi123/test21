Print Details

Select one or more cases within the Case List and click to print the Case Details for the selected case(s). Depending on the supported options for the selected case, this link may open a window that lets you select additional information that you may want to include within the printed version of the details. By default, you will have an option to include:

  • Notes
  • Attachments (See Attachments, page 1)
  • History (See History Subtab , page 1)

The Print Details action item presents a printable version of the Case Details for the selected cases, including all selected additional information.
