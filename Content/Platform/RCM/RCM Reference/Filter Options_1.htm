Filter Options

The following standard search field enables you to search for work items according to fields.

Enter any string in the field and press the Enter key to search through all the work items within the current view.

This filters the display and shows only those work items that contain the search string within the work items list summary. If the current view includes multiple pages, the search is conducted on all pages of the view.

When a filter is in effect an icon and information about the filter appears within the title bar. Following is an example of a filtered Work Items view:

  • Click to open the filter box for more options.

  • The Field: drop-down list enables you to filter according to specific fields and columns (for example, Item ID).

  • Select the Exact match only checkbox to filter according to filter according to your exact entry.
  • Click x to remove the search filter and update the display to show all items within the list, with no filter applied.
  • Click Reset to refresh the Field parameter.
  • Click Close to close the filter box.
  • Click in the top right corner to update the data on display, without changing any filter selection that may be in effect. For example: if you search for all work items whose owner is Admin, clicking Refresh updates the display to include the latest work items, while maintaining your search request.
