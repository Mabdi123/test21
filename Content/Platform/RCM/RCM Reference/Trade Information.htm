Trade Information

The information available on a specific trade depends on the requirements of your organization. There may be, for example, a Review Date column and then a Create Date column that is appropriate for the Trade Type of this Trade information.

The typical columns for a Trade view include the following:

Review Date

Indicates the date when the trade has been reviewed. If the trade has not been reviewed then the field is blank.

Create Date

Indicates the date the trade was created.

Review User

Indicates the name of the user who reviewed the trade. If the trade has not been reviewed then this field is blank.

Trade Type ID

The ID string of this trade. This string is used internally by the RCM and can be used when referring to a trade.

Trade Type

Name of the specific type of trade.
