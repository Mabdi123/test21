Ad-hoc Blotters Personalization

You can save a grid layout as your default layout, as well as a saved filter as your default using the Personalization link.

The following actions are available:

  • Save Layout: Click to personalize your layout after making changes to the grid.
    More Actions:
  • Return to Saved Layout: Click to return the view to the saved layout.
  • Return to Default Layout: Click to return the view to the layout designated as the default.
  • Set as Default Filter: Click to set the current filter as the default (opens the default filter fields.

If you saved a specific saved filter as a default and you want to change it to the out of the box filter fields, you must select New Filter from the filter list and then click Set as Default Filter in Personalization.
