Import Items

You can import selected items into the Platform list that you are currently viewing. You can import APF, CSV TXT or XLS files.
Note that these instructions are for importing CSV TXT or XLS files. For instructions on importing APF files see Import Items from an APF File.

You can import a file that contains up to 500,000 items.

To import items:

  1. Click the arrow next to the drop-down menu, and click Import action link. The Import Items window opens.
  1. Click the Browse icon and select the APF or CSV file that you want to import. If you would like to indicate all new and modified items as unapproved then select the checkbox Mark all modified items as unapproved. This checkbox appears only in lists that require four eye review and to users that have Approve permissions. See also Import Items Effect on Existing Items.


  1. The mapping window appears. This mapping window enables you to use the provided mapping sample to map each column from the imported file. Note that the mapping sample displays a maximum of 10 lines from the imported file.

  2. Select the initial line for import.

    3. The import window automatically maps recognizable fields to the appropriate columns. To map a field, click the arrow next to the Not Mapped setting and select the relevant field from the drop-down list that opens. Unmapped fields are not imported to the Platform List.

  1. Click the Import button in order to finish the import procedure.

You cannot map the same field to more than one column.

Mandatory fields must be mapped.
