Create a New Work Item

You can create a work item from an ad-.hoc Blotter.

To create a new work item:

  1. Click the New Item button in the Ad-hoc Blotters toolbar, the New Item wizard appears. This enables you to manually generate a new work item from the ad-hoc Blotter. The Data entry page of the New Item wizard includes the following fields and options.

 

Option

Description

Item Type

Use the drop-down list to select the Item Type that you want to assign to this Item. Only Item Types that support generation of manual work item appear within the drop-down list.

Item Date

Select the date that you want to appear as the Item Date within the main Work Items list. Click the icon to select a specific date.

Item Description

Enter a description for the new work item.

Step

By default all new work items are automatically assigned the step Ready.

Score

The score that is currently assigned to the account. The score is a numeric value that enables you to evaluate the severity of this work item (such as, the risk currently associated with the account at the time that the High Focus list was last updated).

The score can be based on a number of factors, including previous work items that were triggered for this entity and the threshold defined for the work item type of the triggered work items.

It is possible to manually change the score for a selected account.

Owner

The user who is responsible for investigating or viewing this work item.

Business Unit

Assign a business unit to this work item. Note that only users with permission to view work items assigned to this business unit will see this work item within their Work Items List.

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select the Business Unit that you require for this work item see Business Unit Explorer).
  1. Once you have completed the General Info fields, you can proceed to the Type Specific Info page of the Wizard. Click Next to proceed to this page.

The fields in this page change according to the Work Item Type that you selected. Note that fields indicated by an asterisk (*) are mandatory and must be filled in. If this page does not appear then there are no specific information fields required for the Item Type.

  1. After you add specific information that is required for this Item Type, you have the option of adding a note by clicking Next. You then proceed to the Add Note page of the Wizard.
  1. Click Finish to save the new work item to the Work Items list.
