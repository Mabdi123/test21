Duplicate User

Click the Duplicate User link within the User Management activity bar to open the Create User window that enables you to quickly create a user with the same reference data (roles and groups) as the original user. The Create User Window indicates the name of the user that this duplicated user is based on. A login name, identifier, and full name must be filled in for this duplicated user.

The duplicated user may be linked to a different business unit than that of the original user.

A duplicated user is always an internal user even when the original user is an external user.
