Assign a New RCM User to Groups

Once all required user details are entered, click Next to add the user details to the list of users and move to the next page in the wizard.

Selecting Next within the first page of the wizard saves the new user details to the database. If you do not want to save the details, you must select Cancel within the first status.

The second page in the wizard enables you to assign this user to one or more groups. All groups that are defined for use within your organization appear within the list box on the left side of the window. (LDAP groups do not appear when you assign users to groups). Assigning a user to groups enables an administrator to set up and edit permissions for an entire group rather than dealing with each individual user.

You can assign one or more group(s) to each user by clicking on the group name(s) and clicking on the appropriate arrow to move this group into the list box on the right side. You can change the group assignment later by clicking on the Change Group link within the Manage User activity bar.

Once you have completed assigning groups to this user, click Save to accept all group assignments for this user.
