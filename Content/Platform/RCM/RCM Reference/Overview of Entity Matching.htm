Overview of Entity Matching

The license for this feature is purchased separately.

The entity matching feature enables ingestion of work items from multiple systems and solutions in order to detect potential duplicate items during the process. The match engine compares the new entity with all existing ones. Each potential match goes through a decision making process to determine the status of the incoming entity. The system acts as follows.

  • Match: The incoming entity matches an existing one based on a specific score. RCM Action: The system performs an Automatic Merge of the entities.
  • No Match: The incoming entity does not match any existing entities. RCM Action: The system creates a New Entity.
  • In Doubt: There is doubt whether the incoming entity matches an existing one and the incoming entity is placed in doubt. RCM Action: The system then enables you to perform Doubt Resolution.
