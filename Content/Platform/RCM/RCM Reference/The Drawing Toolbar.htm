The Drawing Toolbar

The icons available within in the Drawing toolbar are as follows:

Enables selecting a single or multiple elements of the drawing

Enables moving the drawing across the canvas

Enables a zooming into a defined area of the drawing. You can define an area by clicking and dragging the mouse while it is on the drawing.

Zooms in on a built in scale

Zooms out on a built in scale

Auto fits the drawing to the canvas size

Defines the zoom level to a specific percentage of the default size

Undo the last action.

Redo the last action.

Emphasizes clusters inherent in the drawing topology.

Displays a hierarchical layout that emphasizes dependency relationships

Displays an orthogonal layout that produces compact drawings, using only horizontal and vertical link routing.

Displays a symmetric layout that emphasizes symmetries inherent in the drawing topology, while ensuring that adjacent nodes are close to each other.

This incremental layout icon reformats the Drawing, considering existing Drawing element positions.

Routes all links in a drawing. Routing avoids overlaps with Nodes and attempts to maintain Node positioning.

Routes a specified set of links.

Adjusts label positions while respecting the format of other drawing elements.

A print setup icon that defines the printing properties. Opens a window with printing options.

After defining your printing options, click the Print Preview button in order to preview and optionally print the drawing.

Enables previewing the print setup and printing the drawing that you have previously defined via the Print Setup button. To print the drawing select File from the menu and click the Print option.
