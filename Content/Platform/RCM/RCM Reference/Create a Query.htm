Create a Query

Each Records Widget may include predefined conditions. There are two types of predefined conditions:

  • Predefined conditions that are in a grey background. These conditions cannot be removed from the query.

  • Predefined conditions that are in a white background. These conditions can be removed from the query

The adding and removing conditions task is done in the same fashion as in the DART feature. For details, see Add Conditions.

The following DART features are currently not available for the Records Widget: The Or operator, the multi-field condition, creating a sub query, exporting query results, creating sequence based queries, creating a DART work item, creating scheduled queries and saving a query.
