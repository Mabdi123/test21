View and Select Individual Report Data

The Dashboard includes the following different types of reports

Icon

Description

Totals for Open Work Items

Displays the number of open work items per specific category.
See Totals for Open Work Items.

Open Work Items

Displays the number open work items according to a user-defined filter. See Open Work Items .

Open Work Item Aging

Displays counts of work items in time groupings. See Open Work Items Aging.

Open Work Items by Step and Item Type

Displays the number of steps for the selected work item. See Open Work Items by Step and Item Type.

Open Work Items by Score Ranges

Displays the score range distribution for the open working items. See Open Work Items by Step and Item Type.

Average Days for Resolution

Displays the average amount of time it took to close the item according to a user-defined filter. See Average Days for Resolution.

Open Work Items Over Time

Displays a graph of the open work items.
See Open Work Items Over Time.

All the data in the reports is based on the filtering definitions that were made in the filtering toolbar.

Due to the limited size of the screen, some of the data in the x axis of a report may not be visible. To obtain the full information, hover over the report to view its related tool tips.

Click the icon to print the resulting display.
