Consolidated Work Item

The Consolidated Work Item (CWI) merges incoming work items that have common criteria, which were defined by your organization. An example of a predefined criterion can be an identical account number: A CWI is configured to consolidate (merge) all work items that have the same account number. When the CWI feature has been enabled and defined by your organization, RCM inspects the incoming work items and consolidates data to the relevant fields of the CWI. The fields of the CWI are consolidated via aggregations that were predefined by the implementer. Example of aggregations for sample fields:

  • Sum: A CWI may be configured to add the amount values of all incoming work items that have the same account number and display the value in its Total Alerted Amount field.
  • Last: Populates the field with the value of the last item that is consolidated. For example, if an address field in a work item is configured via the Last aggregation then the Updated address field of the CWI is always updated with the last work item value of address that was received and consolidated.
