New Item

Click the New Item button on the Case Items toolbar to open the New Item wizard that enables you to manually generate a new item. Select the Case Items category.

The New Item Type page of the New Item wizard enables you to select the New Item Type that you want to assign to this new item. Only New Item Types that support generation of Manual Work Items appear. Click Next to move to the Data Entry window.

The fields in this page change according to the work item type that you selected. Note that fields indicated by an asterisk (*) are mandatory and must be filled in.

The following fields usually appear in this window:

Option

Description

Display Fields (not editable)
Item Category

The category of the work item.
Item Type

The type of case item.
Step

By default all new case items are automatically assigned the step Ready. After generating the item use the Edit Item button to change the step, when required.
Owner

The item owner.
New Item Fields

Item Name

Enter a name for the case item that is displayed in the Case Items list.

Item Date

Select the date to display for the Item Date within the main Case Items list. Click the icon to select a specific date.

Description

Enter a description for the new item. The Rich Text feature may be enabled. See also Rich Text Feature.

Priority

Select a priority level for the case item.

Business Unit

Assign a business unit to this case item. Note that only users with permission to view work items assigned to this business unit see this work item within their Work Item List.

Click the icon beside the Business Unit field to open the Business Unit Explorer, to select the business unit for the work item. (See Business Unit Explorer.)

Enter Data in Entity Related Fields

These types of fields appear only for work items that are defined by your organization as entities. An entity is a work item that can be related to another work item. For example, a Person entity can be related to a Suspect entity in a specific investigation.

Click the icon beside a Related Items Picker field to open the Items Picker window, and select the related Items that you require for this work item.

Example:

To enter data in the entity related fields:

  1. Click the icon of the Suspect Person field.

  1. The Items Picker window opens and the relevant suspects are selected. All selected Entities are displayed in the Selected Items field. You also have an option to create a new Item (Suspect) by clicking the New Item link. This link opens the New Work Item wizard for the specific type of Entity that you want to add (in this example it is Suspect). After the new item definition is finished it is added to the Items Picker window where it can then also be selected.

  1. Click OK and the suspect names appear in the Suspect Persons field.

  1. Once you have completed the Data Entry fields, you can proceed to the Add Note window of the Wizard. Click Next, and the Add Note page appears. Select a predefined note or enter free text
  1. Click Finish to save the new Manual Item to the Case Item List. Edit any manually generated work item using the Edit Item link.
