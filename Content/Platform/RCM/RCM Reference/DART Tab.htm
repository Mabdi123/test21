DART Tab

The following is an example of the DART tab in the New Role window.

Select a data source or All sources in the DART drop-down menu.

To add a permitted action, select one of the following actions from the Actions drop-down menu and click the Add button:

  • All - enables users with this role to have all the available permissions for the selected DART source(s).
  • Create DART Alert - enables users with this role to create a work item for the selected DART source(s).
  • Create Scheduled Query - enables users with this role to create a scheduled query for the selected DART source(s).
  • Delete Public Query - enables users with this role to remove a public query for the selected DART source(s).
  • Delete Scheduled Query - enables users with this role to remove a scheduled query for the selected DART source(s).
  • Export - enables users with this role to export a DART result as a CSV file for the selected DART source(s).
  • Manage Queries - enables users with this role to open the Manage Queries window for the selected DART source(s).
  • Publish Private Query - enables users with this role to change a private query into a public query for the selected DART source(s).
  • Rename Public Query - enables users with this role to change the name of a public query.
  • Save Private Query - enables users with this role to save the query as private.
  • Save Public Query - enables users with this role to save the query as public.
  • View - enables users with this role to view the selected DART source(s).
  • View History - enables users with this role to view the history of actions for the selected DART source(s).

To add another permitted action, select it and click Add. You also have the option to select another DART source and set the desired permitted actions for it.
