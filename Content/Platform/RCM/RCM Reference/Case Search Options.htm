Case Search Options

The RCM includes the following search options and filters.

Standard Search Options

The following standard search field enables you to search for cases according to criteria.

Enter any string in the field and press the Enter key to search through all the cases within the current view. This filters the display and shows only those cases that contain the specified information within the Case List summary. If the current view includes multiple pages, the search is conducted on all pages of the list.

When a filter is in effect an icon and information about the filter appears within the title bar. Following is an example of a filtered Cases view:

  • Click to open the filter box for more options.

  • The Field: drop-down list enables you to filter according to specific fields (for example, Case ID or Case Name).

  • Select the Exact match only checkbox to filter according to your exact entry.
  • Click x to remove the search filter and update the display to show all cases within the list, with no filter applied.
  • Click Reset to refresh the Field parameter.
  • Click Close to close the filter box.
  • Click to update the data on display, without changing any filter selection that may be in effect. For example: if you search for all cases whose owner is Admin, clicking Refresh updates the display to include the latest cases, while maintaining your search request.

In order to filter cases columns by date you must use the advanced filter feature. For details see Advanced Filter.

Advanced Filter

Click the Advanced Filter button to open the Advanced Filter window to search for specified information within the current view.

The Advanced Filter window enables you to filter the current view and to show only cases that correspond to the search factors that you are interested in.

The list of filter criteria in the Advanced Filter window depends on the current Case View that is in effect and on the configuration that is setup for your organization.

The list contains all fields that are commonly used for filtering plus any specific fields, within the current view, which are supported for filtering. Some fields may be grouped according to a business category. In this case, the business category appears as a search option with a + beside it, showing that there are individual fields that are included within this category. Clicking the + expands the category and displays the specific fields:

You can choose to search for the entire category of fields by selecting the category name when this section is closed, or you can expand the section and select only those fields, within the category, that you require.

Additional search options are available within the More Options tab.

You can check on one or more of the available criteria (in one or both tabs) in order to search through the list of cases and filter the case view according to the filter items that are selected.

The filter option may require a range, Enter from and to values for the required range.

If the item does not require a range, you can enter multiple values by separating them with a semi-colon (;).

Following are examples of search criteria that may be available within the Advanced Filter window:

Option

Description

Case ID

Select the case Identifier that you want the RCM to search for.

Case Name

View case(s) according to their name.

Case Type

Select the Case Type that you want the RCM to search for (for example, SAR, unusual wiring activity, etc.).

Step

Enter or select the step items that you want to be included in the search. For example you can search for all cases that are In Process. You can enter more than one Step item by separating the items with semicolons, as shown in the example below:

You can use the icon to open the Select window that enables you to select one or more steps from the list of available steps.

Priority

View only cases that have a priority score within the range that is specified for this field. (If your organization does not support this feature, this option may not appear.)

Description

Select cases according to their description.

Note

View only cases that contain a note with the specified text.

Case Date

Select cases according to the dates when they were created.

Closed Date

Select cases according to the dates when they were closed.

Owner

View only cases that are associated with the specified owner. You can use the icon to open the Select User window that enables you to select the business unit that the user belongs to, and then to enter the name or initial letters for the user of interest.

To view all users within a specified business unit, enter % within the Available users field and click Go.

Business Unit

Select case(s) according to the business unit associated with them.

Containing Alert ID

Select case(s) according to Alert ID(s) that are attached to them.

If cases are not displaying as expected within a selected view, verify that the options that are checked on within the Advanced Filter window are the options that you want to view.

Exact Match

Select the checkbox to search for only exact matches.
