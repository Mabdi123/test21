Review

Select one or more trades and click the Review button within the toolbar to perform the following:

  • When there are no open work items associated with any of the selected trades, the status of all selected trades is changed to Reviewed. If the current filter shows only Un-reviewed Trades, the newly reviewed trades are removed from this view and you can access them by selecting a different filter (e.g. all trades).
  • When at least one selected trade includes an open work item the Review window opens.
