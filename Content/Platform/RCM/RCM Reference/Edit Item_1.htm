Edit Item

You can edit any case item that was generated. Manual items can be generated either from the Case Items list (by clicking New Item) or from any report that supports generation of manual work items.

Selecting a case item and clicking the Edit Item button in the Case Items toolbar opens the Edit Item wizard that enables you to edit details for the case item. Some fields, within the wizard, are read only and cannot be edited. Specifically, the Description field can only be edited for manual work items.

This action may appear in the toolbar More menu. Buttons and action links are customizable depending on your organization implementation.

You can also edit work items that are generated and distributed by the AIS Models.
