Assign

Select an item and click the Assign button in the Case Items toolbar to open the Change Item User/BU window. This enables you to manually assign the selected case item to:

  • A different business unit
  • A different user than the one who is currently the owner of this work item.

You can assign a user to a case item either from the Case Items List or when you are viewing the Case Items Details.

When you assign a new user you must make sure that this user has permissions to deal with this case item.

The Change Item User/BU window includes the following information and options:

Option

Description

Item(s) current BU

Shows the business unit for which this case item is currently assigned.

Business Unit:

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select or type the business unit that you require for this case item.

Search for a case item according to its business unit.

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select or type the business unit that you are searching for.

To select a Business unit:
  1. Use the Business Unit Hierarchy drop-down list to select the hierarchy that you currently require.
  1. Once you select a hierarchy, the Hierarchy Tree shows the highest level of business units within this hierarchy.
  1. Click in the tree to open a node and display of the business units that are sub ordinates of the selected node.

Click the Business Unit name to select it and click OK to select it for assignment. Once selected, this business unit appears within the Assign to Business Unit / User window.

The Assign Item to User/BU window includes an auto-complete feature that enables you to enter part of any name in the Assign to User /BU fields and then receive a drop-down list of all names that contain this name

Assign to me

Allows you to assign the selected item to yourself, indicating that you are the user who is responsible for this item.

Assign to user

Click this option to open the Select User window, select a user and assign this case item to that user.

To select a different user for a case item:
  1. Select the appropriate business unit for the user that you want. Note that the user can belong to a different business unit than the one for which the item is assigned.
  1. Type in the first letters of a name and click Go to view all users whose names start with those letters.
    • To view the full list of users for a selected business unit, enter the percent sign (%) and click Go.
    • If you assign the case item to a new user, make sure that the selected user has permission to view this item.
    • The Assign Item to User/BU window includes an auto-complete feature that enables you to enter part of any name in The Assign To User /BU fields and then receive a drop-down list of all names that contain this name.

Unassigned

When Unassigned is selected, no user is assigned to this case item and the item displays when you select the Unassigned Items view (or the All Case Items view).
