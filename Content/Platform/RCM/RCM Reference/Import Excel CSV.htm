Import Excel/CSV

The Import/Excel/CSV button only appears on the screen if the currently selected setting supports this action. Click this button to open the Import Data from Excel wizard, to select an .xls or .csv file and import it into the setting table.

The first status in the wizard includes an option for selecting (or typing in) the required .xls or .csv file.

Once you have selected the file that you want to import, click Next to move to the next status in the wizard.

The second status in the wizard enables you to map the columns within the imported file to the columns defined for the table that is currently selected. Use the drop-down list boxes to select the associated column from the imported file.

Once you have mapped the columns, you can click Next to move to the next status, which enables you to preview the table according to the selected mapping.

If the mapping does not display correctly in the preview, you can return to the Map Columns status and re-map the columns.

The Preview page of this wizard provides the following options:

Option

Description

Add Row

Adds an empty row to the table. You can enter data by clicking on each field and typing into the fields. Each type of setting requires different actions and therefore a specific setting may or may not support the addition or deletion of rows to the table. It is important to check the logic of the setting before you edit an imported table.

Delete Row

Removes the selected row. It is important to check the logic of the selected setting before you delete rows from an imported table.

Once you are satisfied that the file is mapped correctly to the table columns, you can add the imported data to the setting table. There are two possible options that may appear at the bottom of the wizard screen (depending on the definition of the setting):

Option

Description

Merge Data

Merge adds new data to the table, updates the data that has the same key and leaves old data untouched within the table.

Replace Data

Replaces all existing data with the newly added file data.
