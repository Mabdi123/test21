Manage Delegations

The Manage Delegations actions links enable to assign delegations between other users.

This feature is available only to users that have been provided with the relevant permissions by the administrator.

To access managing delegations:

  1. Click the bar to display the activity bar with the associated actions.

The Manage Delegations table enables you to view all the current delegations. See also Delegate to Another User.

The Manage Delegations table includes the following columns:

Column

Description

Checkbox

Enables you to select a specific delegation in order to edit or remove it.


Active indication

A delegation with an active indication specifies that this delegation is enabled.
A delegation without an active indication specifies that this delegation is currently disabled.

Delegation Type

There are two types of delegations:

  • Managed: provides a delegation relationship between two other users via the Manage Delegations action links.
  • Personal: provides someone else permissions to act as yourself via the Delegate to another user link of the My Account application link. See also Delegate to Another User.

There can be multiple Managed Delegations but only one personal delegation.

From User

The user whose permissions are delegated.

To User

The user who receives the delegated permissions.

From Date

The date when the delegation period begins.

To Date

The date when the delegation period ends.
