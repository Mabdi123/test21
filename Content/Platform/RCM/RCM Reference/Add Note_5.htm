Add Note

Select one or more cases and click the Add Note button in the cases toolbar to open the Add Note window. This enables users to select a note that is available within a list of predefined notes, or to enter a note that is relevant for describing the progress of the case(s).

The type of note that you can add depends on the case(s) that you selected. Different case types support different note options (for example, some case types do not support adding free text, or do not support all of the predefined note options).

When you add a note to more than one case at the same time, you only see those options that are supported by all selected case types. If you require an option that is not supported by one of the selected cases, try selecting the cases one at a time.

To select a predefined note:

  • Click the arrow at the end of the available field to view the complete list of predefined notes that are configured for your RCM.

Select any existing note and then add or edit text according to your needs, or directly enter information into the available field.

The table of cases may include a note column that has the following icon: All cases that include this icon are with notes. Clicking on a note icon of a specific case displays its related notes.

Rich Text Format

This feature provides you with advanced styling and editing capabilities for the entered text. For more information see Rich Text Feature.

The rich text format feature may not be enabled by your organization.
