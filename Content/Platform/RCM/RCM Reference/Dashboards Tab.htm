Dashboards Tab

The following is an example of the Dashboards tab in the Update Role window.

Select the Dashboard Type or all Dashboards in the Dashboard drop-down menu.

To add a permitted action, select one of the following actions from the Actions drop-down menu and click the Add button:

  • All - enables users with this role to have all the available permissions for the selected Dashboards.
  • View - enables users with this role to view the selected Dashboards.
