View RFI

The RFI State column indicates the status of specific RFI. A closed envelope icon indicates that there are messages regarding a specific RFI that have not been read. An open envelope icon indicates that all messages regarding a specific RFI have been read. The following displays a work item with an RFI whose messages have all been read.

To view RFI(s) regarding a specific work item select a work item and click the View RFIs button in the Work Items toolbar. A window opens:

An RFI can be in one of the following steps (statuses).

  • Sent: The RFI was successfully sent.
  • Received – unread: The RFI reply was received but has not yet been read.
  • Read: The RFI reply was received and has been read.
  • Only when the user that created the RFI reads the received message, the message status automatically changes to Read.
  • Each time you forward or reply to a message, the step of the RFI is changed back to Sent.
  • If another reply is received later, the step is changed to Received – unread and then to Read.

To view a specific RFI thread click on its identifier (You cannot select more than one RFI). The window displays the contents of the thread and enables you to reply or forward a selected RFI message.

You also have an option to click on a specific heading of a message in order to view all of its contents.

To go back to the list of RFIs, click on the Back to list of messages link.
