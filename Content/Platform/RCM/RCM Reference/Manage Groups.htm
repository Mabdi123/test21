Manage Groups

This feature enables the administrator to manage groups for users.

To access managing groups:

  1. Click the bar to display the activity bar with the associated actions.

The main window displays the currently defined list of groups and associated information for each group.

The possible default columns for the Group list are: 

Option

Description


(checkboxes)

Click on one (or more) checkbox to perform an operation on the selected group(s) (e.g., Assign users). Clicking on the checkbox in the title bar enables you to select all groups in the list.

Identifier

A unique identifier assigned to each group that is used by external applications (e.g., Web Services).

Group name

The group name that is entered by the system administrator.

Members in group

Lists all other groups that contain this group (i.e., groups in which this group is a member).

Group roles

Displays all roles assigned to this group. If a group is a member of another group, the group also acquires the roles of the parent group.

The Manage Groups activity bar includes the following options:

  • Add Group
  • Add External Group
  • Update Group
  • View History
  • Assign Users to Group
  • Assign Groups to Group
  • Assign to Other Groups
  • Assign Roles
  • Delete Group
