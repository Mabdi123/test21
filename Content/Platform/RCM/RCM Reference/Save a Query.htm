Save a Query

To save the query, click the button. The Save Query As window opens, enabling you to enter the query’s name and description.

A query that has an asterisk next to it indicates that the query has been modified and not saved yet.

By default all saved queries are private and can only be viewed by the user that saved it. Your organization may provide you with permissions to save a public query that can be viewed and used by other users.

To save a public query, select the Save Query as Public checkbox.

The Save Query as Public checkbox appears only to those users that have been provided with this option by their organization.
