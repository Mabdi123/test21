Run a Query

After creating the conditions of the query, run the query by clicking the Run Filter icon. The results of the query appear on the screen. The records of the results may include a + icon. This icon indicates that additional layouts have been added to the results by your organization. In order to view them you should press the + icon that is in the heading of the results table. Click on the + icon that is next to a specific record in order to view its layout.

Change the Sort Order and Adding Conditions via the Query Results

To change the sort order of a column click the right mouse on the column heading and select between the available options. The displayed window also provides you the option to add a condition that is based on the column heading.

Clicking the right mouse when in a specific cell provides you with the option to add conditions that are based on the data in the cell. In order to add a selected condition, click the left mouse.

Click on any column heading within the table to cancel the current sorting order and set the new sorting definition.
