Assign to User

Selecting a report and clicking on the Assign to User button on the Report toolbar to open the Assign Report window. This enables an authorized user to assign this report to a different user.

The Assign Report window enables you to do one of the following:

Option

Description

Assign to User

Clicking on this option opens the Owner Lookup window that enables you to select a user and assign this report to that user. You should make sure that the user has permission to view this report.

You can type in the first letters of a name and click Go to view all users whose names start with those letters.

Unassigned

When Unassigned is selected, no user is assigned to this report and the report displays when you select the Unassigned Reports view (or All Reports view).

You cannot change the business unit assignment for this report, from this window.
