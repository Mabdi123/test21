Drop-Down Menus

Drop-down menus are indicated via a down arrow image that is displayed on the icon - . Hovering over an icon with a down arrow displays its menu items.

A menu item can include additional submenus. A menu item that has additional items is displayed via an arrow that points to the right - . The following is an example of a menu that has a sub menu.
