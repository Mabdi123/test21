Overview of the Operations Dashboard

The Operations dashboard is a managerial tool for tracking the performance of the organization. In contrast to management reports which show one view of data at a time, the Operations Dashboard shows many views of the repository simultaneously, all unified by global user-selected parameter values. For managers with permission to use this tool, the interactive dashboard provides volumes of work items grouped according to time and business dimensions. Managers see a default view, which can be filtered repeatedly to produce different views of data in the repository.

The Dashboard enables you to:

  • Select a time period and/or business units for all reports.
  • Select one or more business units for all reports.
  • View different summary reports and, when relevant, select data retrieval.
  • Drill down to the related work items/cases module to the data on which the trend report was based.
  • Print the Dashboard reports.
