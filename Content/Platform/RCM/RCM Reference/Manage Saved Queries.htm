Manage Saved Queries

Click the Manage Queries button to open the Manage Queries window.

You can perform the following actions for a saved query:

  • Delete: Select a query and click Delete.

A query that is used in other queries as a sub query, cannot be deleted. You must delete all queries that reference a specific query in order to be able to delete it

  • Publish: Select a query and click Publish.
  • History: Select a query and click History.
  • Rename: Select the specific query and enter its new name in the text field of the window and click the Rename button.
