Report Views

The following reports views are available under the Reports menu. Click the next to the Report view title to open the menu.

The Report views menu enables you to select the view that you currently require. The view options are:

Option

Description

On-Demand Reports

Displays a tree, within the right side of the window, which enables you to view all Report Types that support generating reports on demand.

Click a Report Type within the tree to open the Report Generation Wizard enabling you to generate reports based on the templates that are defined for the selected type.

For a detailed description of this feature, see On-Demand Reports.

All Reports

Displays all reports within the right side of the window, regardless of the user to whom the reports were assigned.

Unassigned Reports

Displays all reports that were not yet assigned to any specific user. This view option shows all New, Viewed and Signed-Off reports.

Awaiting Sign-Off Reports

Displays all reports that require Sign-Off by any specified user and have not yet been signed off.

Signed-Off Reports

Displays all reports, regardless of the user to which the report was assigned, which have already been signed off.

Archived Reports

Displays reports that have been archived. Once it has been signed off and archived the report retains the data of the user who performed the sign off.
