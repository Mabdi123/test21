Advanced Sort for Cases

The Advanced Sort window enables you to define multiple sort requirements (sort by Case ID, then by Case Type, etc.).

To define multiple sorting conditions:

  1. Click the Advanced Sort button in the Cases toolbar to open the Advanced Sort window.
  2. Use the Column selection drop-down list to select the column (within the current Cases View) that you want to sort by.

  1. Use the available selection drop-down list to select either Ascending or Descending for the current sort criteria. For example, if you select the criteria Priority and you indicate that you want to sort in Ascending order, the current Case View will be presented according to the score of the work item, with lowest score at the top of the window.
  1. Click the icon to add the specified column to the list of columns that you want to sort by.
  1. You can specify more than one sort condition by adding multiple Columns to the list. The list is sorted first according to the column that appears at the top of this list, then according to the next column etc.

To remove a sorting condition from a multiple sorting definition:

You can remove any sorting condition by using the Sort window.

  1. Click to access the Advanced Sort window.
  1. Select the column that you want to remove within the Sort By: list and use the icon.
  1. The selected condition is not used for sorting.

To change the sort order:

  • Select the column that you want to move up or down within the Sort By: list and use the up or down icons to change its position within the list. The Case list is sorted first according to the column name at the top of the list.

To change the direction for a sort criteria

  • Select the column that you want to change within the Sort By: list and use the icon to switch the direction (for example, change from ascending to descending). 

Once you have defined the sorting conditions that you require, click to sort the pages of the current view.

To view the current sorting definition:

When multiple sorting is in effect arrow indicators appear, in the Case List window, within the column headings that are used as sort conditions. The arrow indicates whether the column is sorted in ascending (arrow points up) or descending order (arrow points down).

You can view the current sorting order by hovering your mouse over the column headings within the Case List. A tool tip provides information about the full sorting order that is in effect.

Clicking any column heading within the Case List cancels the current sorting order and sets the new sorting definition to single sorting based only on the newly selected column heading.
