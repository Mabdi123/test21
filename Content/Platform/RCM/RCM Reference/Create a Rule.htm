Create a Rule

Initially, you must select a Policy Type that is a Query Rules type. After selecting a specific Query Rule a table displays the existing rules for this Policy Type and their current state.

Click the New button and then click the Add Template link in order to view the templates for the Policy Type. Each Query Rule Policy Type consists of one or more templates. A template is composed of a name and the actual template. The text in blue represents field that is filled in order to create the rule. Select a specific template by clicking on it.

In order to create a rule, select a template and fill in the fields of the template. You have the option to use Add Not in order to invert the rule.

You can create a rule by using multiple templates. Once you apply the rule from the first template, click the Add Template link to use another template for the rule.

After you have finished entering the templates do the following:

  1. In the General tab, provide the rule with a name. You also have the option to modify the end of the rule's identifier and add a description for the rule.

  1. The Output Fields subtab may include additional fields that require an input. Each of the fields in this subtab may have a default value that can be modified. For example, the following screenshot the Output Fields tab includes a Score attribute with a default value of 40. This value can be modified before validating and running the rule.

  1. The Tags subtab provides the option to add one or more tags for the rule. Multiple rules can be filtered and viewed according to their tag(s). Your organization may require you to enter a specific tag name for a rule. If you are entering more than one tag then you must enter a semicolon between the tag names.

  1. Click Save in order to save the rule.
