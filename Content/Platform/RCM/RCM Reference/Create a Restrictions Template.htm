Create a Restrictions Template

Click the Work Items tab to define a work items restrictions template or click the Cases tab to define a cases restrictions template. Afterwards, click the Create Restrictions Template. The Workflow Restrictions Template opens.

Permissions from all of a user's role actions are aggregated together. Workflow Restrictions take precedence over regular permissions. Therefore, Workflow Restrictions from one role can cause an action to be restricted even if the same action is not restricted in another role.

  1. Select a workflow from the Workflow drop-down menu and then enter a name and identifier for the template.

  1. Select a step from the Current Step drop-down menu.

    2. Select the Restricted checkboxes that are next to the actions that you want to restrict.

  1. In the Restricted Transitions section select the forward and backward options that you want to restrict for this step.
  1. Repeat the previous instruction to all relevant steps and then click Save. The newly created template appears in the Workflow Restrictions Templates table.
