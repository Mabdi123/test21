View History

Select a user and click the View History link within the Manage Users activity bar to open the View History window.

The View History feature allows you to view all actions and updates that were performed for the selected user. For example, if permissions of a user are modified, the type of action appears within the Action column and the details about the action appear within the History Line column.

The View History window includes the following columns, by default:

Option

Description

Date

Shows the date when the action occurred.

Action

Shows all actions that were performed on this role.

Performed by

Shows which user performed the action.

History Line

Shows more detailed information about the action. Further details on the highlighted action also appear in the scrollable window of the window.
