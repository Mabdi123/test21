Add Multiple Conditions to a Query

You can create complex queries by adding multiple conditions to a query separated by the AND and OR operators. By default every new condition has the AND operator.

  • To change a condition to OR, click on the AND link.
  • To change an OR back to AND click on its link.

You can combine multiple conditions in a single query using the AND and OR operators.

However, only the following variations are supported:

  • Using AND operators for all conditions.
  • Using OR operators to divide the conditions into a few sets when all conditions in each set use the AND operator.
