Duplicate a Rule

A duplicated rule enables you to quickly create a rule that is similar to the original rule.

To duplicate a rule:

  1. Select the rule that you want to duplicate.
  2. Click the Duplicate icon.
  3. The Rule Criteria window is displayed. Make the necessary changes and click the Save icon. For more information on editing a rule, see Edit a Rule.
  4. Click the up arrow of the Rule Criteria to move back to the Draft Policy tab.
  5. The duplicated rule appears in the Rules table. The initial name of the rule is Duplicate of <original Rule name> (for example, Duplicate of high risk).
