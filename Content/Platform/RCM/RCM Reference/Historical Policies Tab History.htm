Historical Policies Tab - History of Activated Policies

To view the history of previously activated policies:

  • Click the icon. The Activation Report History window appears.

The Activation Report History window displays the policy activations in chronological order and includes the following columns:

Option

Description

Radio button for selecting the historical policy that you would like to view.

Request User

The name of the RCM user that activated the policy.

Requested Date

The date when the request for activation was made.

Deployed Rules

Displays the rules of the policy.

AIS Statuses

Indicates the following:

  • The name of the AIS server that processed the request
  • The date and time when the AIS processed the policy’s rules.
  • Whether the policy was deployed successfully or failed.

After selecting a specific historical policy, click the Validate Details icon. The Historical Policies tab displays the policy that you have selected.
