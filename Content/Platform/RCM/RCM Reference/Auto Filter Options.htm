Auto Filter Options

DART enables you to create conditions quickly by right-clicking a specific column heading or column field. Options are based on the specific data type.

You can:

  • Display all records that include the selected value in this column.
  • Display all records that do not include the selected value in this column.
  • Display all records that have a null value in this column.
  • Display all records that do not have a null value in this column.

The different column types may provide differing options.
