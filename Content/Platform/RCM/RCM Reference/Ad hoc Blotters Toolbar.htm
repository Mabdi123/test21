Ad-hoc Blotters Toolbar

The toolbar buttons that appear at the top of the Blotter Details list depend on the needs of your organization and the permissions that are assigned to your role.

The following buttons can be displayed.

Option

Description

Enables you to sort trades according to specific requirements. See Sort Options.

Opens the View Notes window where you have the option to view the notes of the selected trade (this action does not allow selecting multiple trades). See also View Notes.

Enables the users to manually generate a new work item for a specific trade. See Create a New Work Item.

Select a Trade Blotter and click the View History button to view the audit history pertaining to the Blotter.
