Filter Options for Risk Based Trades

Click the Trade Details link to access the Risk Based Trade filter. The filtering options are the same as those for regular trades. For details see Filtering Options.

Trades are filtered according to specific criteria. Risk based trades can be automatically recalculated when the page is refreshed. (according to your organizational implementation)

If a filter has been applied to the list the following appears: Filter is applied.

If a filter has not been applied to the list the following appears: No filter is applied. Click to expand.
