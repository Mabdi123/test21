Rich Text Feature

This feature provides you with advanced styling and editing capabilities for the entered text in notes, descriptions, and comments.

The rich text format feature may not be enabled by your organization.

The toolbars of rich text include these options:

Option

Description

Choose a font for the selected text

Choose a font size  for the selected text

Makes the selected text bold

Makes the selected text italic

Underlines the selected text

Strikethrough the selected text

Aligns text to the left

Centers text

Aligns text to the right

Aligns text to both left and right. Added space may be added between words

Changes font color for selected text. Click the arrow next to the font color button to select a font color

Changes background color for selected text. Click the arrow next to the background color button to select a background color
Decreases indentation

Increases indentation

Adds bullets to selected text

Adds numbering to selected text

Cuts the selected text

Copies the selected text

Pastes the text that is in the clipboard

Pastes the text that is in the clipboard as unformatted text

Opens a window that enables you to paste text that has been copied from a Microsoft® Word® document.

Opens a window that enables you to define and insert a table.

Opens a window that enables you to format the selected row.

Opens a window that enables you to format the selected cell.

Inserts a row above the selected row

Inserts a row below the selected row

Deletes the selected row

Inserts column to the left of the selected column

Inserts column to the right of the selected column

Deletes the selected column

Splits selected cell that has been previously merged from two or more cells

Merges the selected cells into one cell.

Removes all formatting definitions from selected text

Left to right text direction

Right to left text direction
