The Historical Policies Table

The Historical Policies table includes the following options:

Option

Description

Is test

Indicates whether or not the rule is a test rule.

#

Indicates that the rule is prioritized. When the rule is prioritized a number indicates its priority in relation to the prioritized rules. See Set/Remove the Rule Priority.

Rule Name

Displays the name of the rule.

State

Displays one of the following states for the rule:

  • New - a newly created rule.
  • Changed - a rule that has been changed for this policy.
  • Unchanged - a rule that has not been changed for this policy.
  • Deactivated - a rule that has been deactivated for this policy.
