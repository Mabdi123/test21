Open Workspace

Select a work item and click the Open Workspace button on the Work Items toolbar to open the work item workspace. The workspace is configurable according to your organization’s requirements.

Users with the Add free text Note permission can edit the workspace. Users with the View Notes permission can view the workspace only.

The Workspace enables you to enter and store open information that pertains to the work item. The Workspace can be edited at any time and the information is saved automatically.

The Edit Workspace window includes the following information and options:

Option

Description

Close

Closes the window and saves the workspace.

Create as Note

Click to create a note containing the current workspace content. The Edit Workspace window closes and the Add Note window opens displaying the workspace information. The original workspace data remains and the note enables you to further edit the content.

Autosave feature

The workspace is automatically saved intermittently.

The following messages are displayed in the workspace:

Saving…: During editing.

Workspace saved: Once the user stops editing for three seconds:
If you exit the window without clicking Close, before three seconds has passed since you edited the workspace, your latest information will not be saved.

The Add Note and Change Step windows have added workspace capabilities. For details, see Change Step and Add Note.
