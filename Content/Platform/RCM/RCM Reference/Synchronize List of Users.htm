Synchronize List of Users with LDAP Users List

Once you have added all relevant LDAP groups to the list of groups, you can continue by synchronizing the users list with the LDAP users list.

The system only synchronizes the list to include those users who are associated with the LDAP groups that you added to the .

Circular references (for example, A is a member of group B and B is a member of group A) are not allowed in LDAP.

To synchronize the list of RCM users with the LDAP users:

  • Click Synchronize Users in order to bring all LDAP users, who are associated with the LDAP groups that you have added, into the database.

    • This process may take some time. A message window is displayed when the process finishes, indicating success or failure.

An external user is added to the only if that user is linked to a group and the group is linked to a role. If an external user attempts to login to the via Single Sign On and they are not linked to a group that has a role then they will not be able to login to the .
