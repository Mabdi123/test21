The Draft Policy Table

The following displays the Draft Policy list.

The Draft Policy list includes the following columns:

Option

Description

Enables you to select a rule in order to perform a specific task via one of the rule icons.

Planned State

Indicates whether the rule will be activated during the next time the Activate Policy button is clicked.

  • Active: The rule will be activated.
  • Inactive: The rule will not be activated.

For details, see Activate a Draft Policy - The Policy Activation Tab.

Current State

Indicates whether a rule is currently running in an Active policy.

  • Active: The rule is currently activated.
  • Inactive: The rule has not been activated.

For details, see The Active Policy Tab.

#

Indicates that the rule is prioritized. When the rule is prioritized a number indicates its priority in relation to the prioritized rules. For details, see Set/Remove the Rule Priority.

Rule Name

Displays the name of the rule. Click on the rule name in order to move to the Rule Criteria window. An (M) next to the rule name indicates that the rule has been modified. For details, see Create a New Rule.

Additional Rules

Additional columns include customized options that are displayed according to your organization’s requirements.

Sort the Table

You have the option to sort the rows in the table in descending or ascending order according to any of the column headings excluding Criteria.

To sort the table:

  1. Click a specific column heading (e.g., Order Number).
  1. A down arrow appears next to the column indicating that the table is now sorted in descending order via the column heading.
  2. Click the same column heading again in order to change the order to ascending order. Note that an icon of an up arrow appears.
  1. Click the Reset icon in order to remove the sorting filter from the table.
