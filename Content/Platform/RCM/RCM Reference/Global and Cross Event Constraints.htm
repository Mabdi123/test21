Global and Cross Event Constraints

Global Constraint

This feature enables you to define global constraints for all the events in the sequence. After you have defined the global sequence timeframe you have the option of adding a global constraint.

To add a global constraint:

  1. Hover over the sequence timeframe line and click on the Add constraint link that appears.
  1. A drop-down menu with the default selection For the same is selected. Select a field in the table that must be the same for all events. For example, in this sequence the amount must be the same for all events in the sequence.
  1. Click the Apply button to save the global constraint.

    2. To add another constraint hover over the sequence timeframe line and click on the Add constraint link that appears.

Cross Event Constraint

This feature enables you to define constraints between events that are not sequential. After you have defined the global sequence timeframe you have the option of adding a Cross Event Constraint.

To add a Cross Event Constraint:

  1. Hover over the sequence timeframe line and click on The Add constraint link that appears
  1. A drop-down menu with the default selection For the same is selected. Change the selection to Cross Event Constraint.
  1. Select a field of the Data Source. The Cross Event Constraint displays additional drop-down menus.
  1. Select the values from the drop-down menus to define the constraint.
  1. Click the Apply icon to save the Cross Event Constraint.

    2. To add another Cross Event Constraint hover over the sequence timeframe line and click on the Add constraint link that appears.
