Form Sets

Form sets are group of fields that can appear more than once in the form. For example, last name, first name, and region. Select one or more Form Types and fill in one or more Set form fields in order to filter the list. For example, in order to search for all forms of type 8 that are from the NY region the user fills in NY in the Region/Name field and clicks Go. Alternatively, the icon can be used to open a window that enables you to select a specific region.

  • To view all form sets search fields click the Expand All icon.
  • To view only headings of form sets search fields click the Collapse All icon.
  • To view only form sets search fields click the Show Used icon.
