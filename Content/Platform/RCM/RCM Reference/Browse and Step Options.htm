Browse and Step Options

A navigation and step bar, at the top of the window, includes the following options:

Option

Description

Enables you to return to the page from which you accessed the Work Item Details. For example, if you accessed the Work Items Details from the general Work Items List, this takes you back to this list. If, however you accessed the details from a link within a different work item, this takes you back to that work item page.

Clicking the arrow opens a drop-down menu with the name of the previous page that you viewed. Click this name in order to move to that page.

Provides information about the step of this work item. The two icons show whether the work item is open or closed, and whether it is an issue, non-issue or undetermined. (For details about work item steps, see Change Step.

Click to open the View Notes window and view the notes associated with the work item. This icon is only displayed when the work item has notes attached to it and you have permission to view them, (for example, viewing confidential notes). For details see View Notes.

If a file is attached to this work item, a paperclip icon appears in the navigation and step bar. You can click the paperclip to view the links of the attached files.

Browse options enable you to move to the previous or next Work Items Details according to the list within the Work Items View that is currently active.

There is an upper limit to the number of Work Item Details pages that you can browse through at one time. If you reach the upper limit you are notified and you can return to the general Work Items List, select to view the details for the next work item and then start browsing again.
