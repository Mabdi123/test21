Tuning Tab

The following is an example of the Tuning tab in the New Role window.

Select All in the Tuning drop-down menu.

To add a permitted action, select one of the following actions from the Actions drop-down menu and click the Add button:

  • All - enables users with this role to have all the available permissions for this module.
  • Edit Test Parameters - enables users with this role to edit relevant Platform List test parameters.
  • Run Test - enables users with this role to configure the parameters of the Test Execution and run the test.
  • View Test - enables users with this role to view results of the Test Run that has been conducted via the Test Execution tab.
