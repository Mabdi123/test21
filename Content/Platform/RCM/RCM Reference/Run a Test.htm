Run a Test

The next step after selecting and/or defining the test values is to configure the parameters of the Test Execution. Enter values for all the parameters of the Test Execution as all are mandatory.

You have the option to save the entered values of the Test Execution by clicking Save Values. All values remain saved after logging out of RCM.

Click Run Test in order to conduct a test run. The right portion of the tuning bar displays the status of the test run.

While the test is running, you cannot run the test again and the test items of the list instance are not editable. If another user attempts to run the same test during this status then a displayed message indicates that the test is locked.

Click the Refresh button in order to view an updated status of the test run.

You have the option to click the Abort button that appears during this stage in order to abort this test. In this scenario the status of the test changes to Test is aborting and finally to Test was aborted.

Once the test is finished the status changes to Test finished successfully.

If the test is not successful then the status of the test is Test failed.
