View History

Click in the More menu in the Cases toolbar to view the history for any selected case

This action may appear directly in the toolbar. Buttons and action links are customizable depending on your organization implementation.

This feature allows you to view all actions and updates that were performed for the selected case. For example, if a user changes the step of a case, opens a case or adds a note to the Case, the type of action appears within the Action column and the details about the action appear within the History Line column.

The View History window includes the following columns, by default:

Option

Description

  

Date

Shows the date when the action occurred.

Action

Shows all actions that were performed on this case. For example, if you use the Change Step action, this information appears here.

Performed by

Shows which user performed the action.

History Line

Shows more detailed information about the action. For example, if someone assigns this case to a different user, the details (previous user and new user names) appear here.
