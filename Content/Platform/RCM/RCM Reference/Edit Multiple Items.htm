Edit Multiple Items

You have the option to edit more than one item at the same time. Select the items that you want to filter and click the Edit Selected Items button.

The Edit Multiple Platform List Items window opens enabling you to edit all the fields excluding the primary field(s).

Select the checkbox to edit a value.

All fields that have not been modified by you display the Unchanged text.

You cannot edit a value which was defined as part of the list’s key.

Click Save after you finish editing the fields. The selected fields are displayed with their updated values.

If some of the items in the list are approved and cannot be edited then the RCM displays a message that asks if you would like to edit only the items that still enable editing. If you click the No button then the entire multiple editing action is canceled. If you accept this action, then only the editable items are edited. See also Approve/Disapprove an Item.

When you are saving a relatively large amount of edited items a progress bar indicates that the items are being saved. This save operation of a large amount of edited items may affect performance. The saving will proceed even if the browser window is closed.

If the procedure of saving multiple items is not successful, an error message appears.
