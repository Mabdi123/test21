The Network Types Filter

The Network Types filter displays a list of checkboxes for various graph types names. The selected graphs are displayed on the drawing.

Select the relevant Network Types and then click Go. Only the selected Network Types appear on the drawing.

Click the All/None links to select/clear all the Network Types.

Click the icon at the top of the Network Types Filter window to collapse the display of this window. Clicking the icon of the collapsed window expands it back to its default size.
