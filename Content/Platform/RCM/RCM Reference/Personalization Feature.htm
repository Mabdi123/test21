Personalization Feature

You can permanently personalize a view so that it automatically displays preferences according to user. Personalization is available for most modules:

You can personalize the layout as follows:

  • Sort order
  • Column width
  • Column order.
  • Column double-click extension
  • Minimum width – 24 pixels
  • Maximum width – 1000 pixels
  • Default view

For details on these customization features, see Items List Area.

The Personalization link enables you to perform layout personalization actions. Click the arrow to view options and More Actions to view the personalization actions.

The Personalization option displays your current view and whether it is the default view.

When you click Personalization in other views, the following opens:

You can change the layout but you cannot save your changes.

When substituting for a user, any saved changes done by the substituting user on the relevant substituted user are saved on the substituted (original) user.
When a user acts on behalf of an absent user, personalization views and changes are loaded based on the acting user and not the absent user. Similarly, saving personalization changes are saved to the acting user. The default view is also displayed according to acting user.
