View Attachments

Click the Attachments button in the Work Items toolbar to open the View Attachments window that enables users to view attachment details.

The View Attachments window provides a detailed list of all the attachments for the work item. It also enables you to add/remove attachments.

Add Attachments

You can add an attachment as follows.

The default for the maximum size of single attachment is dependent on your organization’s configuration.

To add an attachment:

  1. Click the Add Attachment button. The add Attachments window opens.

  1. Select Add File or Add Link.

To add a file:

  1. Click the Add File button and enter the path and the file name that you want to upload in the window that appears.
  1. You can add more than one file at a time. If you would like to add another file click Add File button again and repeat the previous steps.
  1. Optionally, add a description and a note to the attachment. For more information on adding notes see Add Note.
  1. Click the Upload button.

To add a URL:

  1. Click the Add Link button and enter the following in the window that appears:
  2. Link name – provide a name for the link. It is recommended to use the ”http://" format for links.
  1. Enter the URL of the link (for example, http://www.actimize.com). Note that you cannot add a link to your local machine, but only a Web or a server link.

  1. Click the Add button.
  1. You can add more than one link at a time. If you would like to add another link click Add link button again and repeat the previous steps.
  1. Optionally, add a description and a note to the URL. For more information on adding notes see Add Note.
  1. Click the Upload button.
You can add multiple links and files and click the Upload button once.

Remove Attachments

You can remove an attachment from a work item as follows.

To remove attachment(s):

  • Select the checkbox next to the attachment(s) and click the Remove Attachments button.

Add a Confidential Attachment

Users that have the confidential level feature enabled, can create a confidential attachment that cannot be viewed by users that do not have this feature enabled. To add a confidential attachment, select the Confidential Attachment checkbox from the Add Attachment window.

This checkbox may not be available for all users.
  • The icon is displayed next to rows that have a confidential attachment.
Users that do not have the confidential feature enabled by the administrator cannot view confidential attachments.
