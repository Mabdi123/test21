New Delegation

The New Delegation action link enables you to create a Managed Delegation.

Clicking the New Delegation action link opens the New Delegation window. It includes the following fields.

  1. Initially select either the Active or Not active checkbox. An active delegation specifies that this delegation is enabled. An inactive delegation specifies that this delegation is currently disabled.
  1. Enter the user whose permissions are delegated in the From User field. Clicking the icon opens the User picker window, which enables you to select or type a specific user.
  1. Enter the user who receives the delegation in the To User field. Clicking the icon opens the User picker window, which enables you to select or type a specific user.
  1. Enter the date when the delegation period begins in the From Date field. You also have an option to define a specific time for the beginning of this period.
  1. Enter the date when the delegation period ends in the To Date field. You also have an option to define a specific time for the end of this period
  1. Click Save. The new delegation appears in the Delegations table.
