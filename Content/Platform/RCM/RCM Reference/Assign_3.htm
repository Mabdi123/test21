Assign

Click the Assign button in the Cases toolbar to open the Change Case User/BU window, which enables users to manually assign the selected case(s) to: 

  • A different business unit.
  • A different user than the one that is currently the owner of this case.

You can assign a user to a case either from the Case List or when you are viewing the Case Details.

When you assign a new user you must make sure that this user has permissions to deal with this case.

The Change Case User/BU window includes the following information and options:

Option

Description

Case(s) current BU

Shows the business unit for which this case is currently assigned.

Assign to another Business Unit

New Business Unit

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select the business unit that you require for this case. (See Business Unit) Or enter the name of the Business Unit. A drop-down menu appears enabling you to auto-complete the Business Unit Name.

Assign to another user

Assign to me

Allows you to assign the selected case to yourself, indicating that you are the user who is responsible for dealing with this case.

Assign to user

Allows you to assign the selected case to a different user. You can either type the name of the user or click the icon to use the Select User window.

The Assign Case to User/BU window includes an auto-complete feature that enables you to enter part of any name in the Assign to User /BU fields and then receive a drop-down list of all names that contain this name.

Unassigned

When Unassigned is selected, no user is assigned to this case and the case displays when you select the Unassigned Case view (or the All Cases view).

Add Note

Select to add a note to the assigned case.
