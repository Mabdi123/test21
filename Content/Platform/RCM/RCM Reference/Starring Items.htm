Starred Items

From the work items list, specific items can be starred. This is useful for easily finding the desired items and performing batch actions on them.

Starring is personal, that is, only you see them on the items you starred. Starring is persistent, even across sessions, and will be displayed to you in the work items list until you remove them.

Particularly if the starred items span multiple pages of the view, it is convenient to use the Advanced Filter icon () and select Show Starred Items from the menu to display only the starred items.

It is then easy to select the relevant rows from among the starred items and perform an action on them as a batch. When viewing a filtered page that shows only the starred items, you can return to the full list of work items by using the Advanced Filter icon again and selecting Show All Items.

The Advanced Filter window also has a Starred Item drop-down menu. This can be used to combine the advanced search with the fact that the result must include or exclude starred items, if necessary.

When working with an individual work item, the Work Item Details page has a star in the header that can be selected to mark the item as a starred item, or remove the star if the item was already starred.

Starred items are saved, even after logging out. As there is a maximum number of allowed starred items, you need to deselect the stars from items that are no longer required. If you exceed the maximum number of allowed stars, the system will remove deleted and archived items from the starred items list to free up more stars but, if there are none, you will not be able to star more items until you deselect some starred items.
