Jump to another Data Source

You may have the option to jump to another data source when there is linked data of two or more data source fields (the same business type). This linkage is defined by the administrator and may not be available in all data sources. You can view this link when the icon next to the column heading appears. When you click on a specific cell of the query results table, opens a menu where you have two options to jump to the other data source:

To jump to another data source:

  1. Jump to the other data source according to the mutual field data of the focused cell (trade type is A). In this scenario the match is done according to the mutual field (according to the business type) of the data source

  1. Jump to the other data source according to similar records. This option provides one or more mutual fields of the other data source according to the record in focus.

When performing this jump, mutual conditions from the original data sources are also moved to the other data source query. For example, the period of the original data source is moved to be part of the conditions for the linked data source.
