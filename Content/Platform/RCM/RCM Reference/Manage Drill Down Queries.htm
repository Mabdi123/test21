Manage Drill Down Queries

This feature provides you with the option to change the default values of DDQs without using the Designer and while the is in production mode.

To access managing Drill Down Queries:

  1. Click the bar to display the activity bar with the associated actions.

  1. Expand the Manage Drill Down Queries bar by clicking on its + icon. Select a DDQ and click the Edit Drill Down Query action link. A window with the DDQ's default values appears.

  1. Fill in the new default values and click Save.

This feature is available only to users that have been provided with the relevant permission by your administrator.
