Admin

Select the checkbox for providing all the GUI Items Admin permissions.

Click on the Admin link to select specific GUI Items permissions.

  • Manage LDAP – select this checkbox to provide all the LDAP management permissions for users with this role.
  • Configure LDAP connection - select this checkbox to provide the permissions to configure LDAP connections for users with this role.
  • Synchronize Users - select this checkbox to provide the permissions to synchronize users for users with this role.
