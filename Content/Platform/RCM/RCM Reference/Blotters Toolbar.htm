Blotters Toolbar

The toolbar buttons that appear at the top of the Blotter window depend on the needs of your organization and the permissions that are assigned to your role.

The following are typical actions that are available from the Blotter window.

Option

Description

This filters the display and shows only those Trade Blotters that contain the specified information within the Trade Blotters List. If the current view includes multiple pages, the search is conducted on all pages of the list.

Select a Blotter and click the Change to under review button to change the blotter step to Under Review. Only enabled for Blotters with the relevant workflow. See Under Review Action.

 

Select a Blotter and click the Assign button to open the Change Trade Blotter User/BU window that enables users to manually assign the selected Trade Blotter to a different business unit or a different user than the one who is currently the owner of this Trade Blotter. See also Assign.

Select one or more Trade Blotters and click the Add Note button to open the Add Note window. This enables users to select a note that is available within a list of predefined notes, or to type in a note that is relevant for describing the progress of the Trade Blotters. See also Add Note.

Select a Trade Blotter and click the View Notes button to open the View Notes window where you have the option to view the notes of the selected Trade Blotter. See also View Notes.

Select one or more Trade Blotters and click the Sign-Off button to open the Sign-Off Trade Blotter window. See also Sign-Off.

Select a Blotter and click the Print all Trades button to print all of the trades for the Blotter.

Select a Blotter and click the Export button to export all transactions to a .csv file.

Select a Trade Blotter and click the View History button to view the audit history pertaining to the Blotter.

Opens the Ad-hoc Blotter page. Enables you to perform an Ad-hoc review of the overall trading activity. It also provides you with the option to save the results of the investigation as a work item. Results of the investigation also display the specific branch managers work items and notes on trades. See also Ad-hoc Blotters.
