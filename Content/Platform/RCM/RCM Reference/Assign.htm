Assign

Select a work item and click the Assign button in the Work Items toolbar to open the Change Item User/BU window. This enables you to manually assign the selected work item to:

  • A different business unit.
  • A different user than the one who is currently the owner of this work item.
You can assign a user to a work item either from the Work Items List or when you are viewing the Work Items Details.

When you assign a new user you must make sure that this user has permissions to deal with this work item.

The Change Item User/BU window includes the following information and options:

Option

Description

Item(s) Current BU

Shows the business unit for which this work item is currently assigned.

Business Unit:

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select or type the business unit that you require for this work item. (See Business Unit Explorer).

Note: The Assign Item to User/BU window includes an auto-complete feature that enables you to enter part of any name in the Assign to User /BU fields and then receive a drop-down list of all names that contain this name

Assign to me

Allows you to assign the selected work item to yourself, indicating that you are the user who is responsible for this work item.

Assign to user

Click this option to open the Select User window, select a user and assign this work item to that user.

To select a different user for a work item:

  1. Select the appropriate business unit for the user that you want. Note that the user can belong to a different business unit than the one for which the work item is assigned.
  1. Type in the first letters of a name and click Go to view all users whose names start with those letters.
  • To view the full list of users for a selected business unit, enter the percent sign (%) and click Go.
  • The Assign Item to User/BU window includes an auto-complete feature that enables you to enter part of any name in The Assign To User /BU fields and then receive a drop-down list of all names that contain this name.
  • If you assign the work item to a new user, make sure that the selected user has permission to view this work item.

Unassigned

When Unassigned is selected, no user is assigned to this work item and the work item displays when you select the Unassigned Items view (or the All Work Items view).
