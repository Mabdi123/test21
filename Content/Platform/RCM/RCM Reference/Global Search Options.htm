Global Search Options

The Global Search feature provides quick and robust search capabilities for users.

It is currently available only for work items and is based on an external indexing engine. In order to view the Global Search feature, you must have the Alerts - Open Alerts by Owner/Type permissions.

Clicking the cog wheel icon next to the search box opens the global search configuration options.

The options in this menu allow you to filter the results even before doing the initial search. The search options menu is divided into 3 sections:

  • Modules: Select the modules that are included in the search. Currently the Work Items module is supported including notes, RFI and workflow steps.
  • Time Range: Select the time range for the search – Always, Past Week, Past Month, Past Year. Once you select a time range option, it is saved in for all future searches it is changed.
  • Extended Info: Select whether or not to include attachments and history of actions in your search. The global search feature supports searching for attachments in text files, CSV file, PDF files, and Microsoft® Office® files. Once you select an Extended Info option, it is saved in for all future searches until it is changed.
You may not be able to view all the options in the configuration menu since this depends on your user permissions and specific organizational configuration.

By default all the modules are selected, the time range is set to Always, and Include attachments is selected. Clicking the Restore Defaults link resets the search options to the default settings.
