Export Items

You have the option to export items to APF and CSV files.

Export an APF File

  • To export specific items, select the items that you would like to export
  • To export all the items, click the arrow next to the drop-down menu, and click the Export All Items link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported APF file.

Export Filtered Items to an APF File

  • To export filtered items, filter the list by clicking the Filter () button. Click the arrow next to the drop-down menu, and click the Export Filtered Items link.
  • To export all the items, click the arrow next to the drop-down menu, and click the Export All Items link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported APF file.
