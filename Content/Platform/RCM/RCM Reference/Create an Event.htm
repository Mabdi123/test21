Create an Event

To create an event, initially create one or more conditions. Once you have finished entering the conditions, click the And then link.

This action defines the conditions as an event. An additional event then opens, enabling you to create its conditions.

Name the Event

The default name is event<number of event in the sequence> (for example, Event3). You are provided with the option to change the event’s name.

To name an event:

  1. Click the event link.

While conditions are in editing mode the event name link is disabled.

  1. A window appears. Click the Create a name link.
  2. Fill in the name of the event in the textbox and press the Enter key.
  3. The new name of the event appears.

Test the Event

Testing the event enables you to run a specific event and view its results.

Before testing an event you must define a time frame for the sequence.

To test an event:

  1. Click the event link.
  2. A window appears. Click the Test link.
  1. The results of the test are displayed.

Remove an Event

To remove an event:

  1. Click the event link. A window opens.
  1. Click the Delete link. The event is removed from the screen.
