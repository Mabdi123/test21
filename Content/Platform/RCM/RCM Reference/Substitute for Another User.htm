Substitute for Another User

This feature allows you to substitute for another user without initially defining a delegation (For more information on delegations, see Delegate to Another User and Manage Delegations). Click the Substitute for Another User link in the User Info window.

The Substitute for Another User window opens. Enter the name of the user that you want to impersonate in the User field and click OK. Clicking the icon opens the User picker window, which enables you to select or type a specific user.

This feature is available only to users that have been provided with the relevant permissions by the administrator. For details see Admin Tab .

.

The User Info window indicates the name of the user that is currently being substituted and the tooltip of the substitute icon provides details on the substitution.

To go back to using your own credentials click the Change User Context application link and click the Switch Back link.
