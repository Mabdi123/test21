Export/Mail/Print

The Export/Mail/Print links are found in the menu. It allows you to print cases, print case details, and send case details via email.

Print List

Click to open a window with a printable version of all cases that are currently selected within the Case List. When no cases are selected, this option prepares a printable list of all cases in the current view.

To send the displayed information to a printer, click the link in the upper right corner above the list in the new window.

Print Details

Select one or more cases within the Case List and click to print the Case Details for the selected case(s). Depending on the supported options for the selected case, this link may open a window that lets you select additional information that you may want to include within the printed version of the details. By default, you will have an option to include:

  • Notes
  • Attachments (See Attachments)
  • History (See History Subtab)

The Print Details action item presents a printable version of the Case Details for the selected cases, including all selected additional information.

Export

Select one or more work items and select the action to open a standard Windows Download window enabling you to save the selected work items to a CSV or XLS file format that can be opened by any application that supports these formats (e.g. Excel).

Send Mail

Click to send a link to email recipients to enable them to access Case Details for selected cases.

When you send case details for more than one case, the recipient clicks the link within the email and the first page of Case Details opens within the RCM. The recipient can use the standard browse options, available within the Case Details page, to view the next cases.

The recipient must be an authorized RCM user and must have permission to view these case details.
