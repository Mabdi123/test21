Manage Users

The Managing Users feature enables the administrator to manage users and groups and assign roles to users.

To access managing users:

  • Click the bar, the activity bar opens to show the associated actions.

The main window displays the currently defined list of users. The possible columns for the User list are:

Option

Description


(checkboxes)

Click a checkbox (several checkboxes) to perform an action on the selected user(s) (e.g., Delete selected users). Click the checkbox in the title bar to select all users in the list. Some actions can only be performed for a single user.

The second column may show a icon which indicates that there is an expanded list of roles associated with this user. Click the icon to open the expanded view of information, as follows:

The expanded rows shows all roles that were assigned directly to this user (See Assign Roles per Business Units).

Identifier

A unique identifier assigned to each user for external systems (e.g. Web Services).

Login name

The user’s login name that is entered by the System Administrator.

Depending on the configuration of your system, external user login names may be presented as <domain name>/<user name>.

User roles

All roles to which this user is directly assigned. Roles determine the actions and views that a user is permitted to perform (and view) for a given business unit. (See Assign Roles per Business Units).

Roles that are assigned to a user as a result of membership within groups do not appear within this list.

Member in groups

All groups of which the user is currently a member. Groups are defined using the Manage Groups activities. (See Manage Groups).
