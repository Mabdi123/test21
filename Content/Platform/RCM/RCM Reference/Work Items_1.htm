Work Items

Select the checkbox for providing all the GUI Items Work Items permissions. Click on the Work Items link to select specific GUI Items permissions.

 

Select one of the following to display options:

  • Tools
  • Actions

Tools

  • Mark All – select this checkbox to provide all the permissions that are listed in the Tools window.
  • New Case - enables users with this role to create a new case from the Work Items module.
  • Add to Existing Case - enables users with this role to add a work item to an existing case from the Work Items module.

Actions

Actions - select this checkbox to view all the permissions that are listed in the Actions window.

  • Quick Change Step – users with this permission are able to change a step of a selected case based on predefined configuration bypassing the change step permission.
