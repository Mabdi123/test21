Other GUI Items Tab

The Other GUI Item tab includes Blotters, Work Items, Reports, and Admin, that can be included within the permission for the role that you are defining.

When the checkbox within a bar is selected, this ensures that all listed items under that bar are permitted within the corresponding module for all users who are assigned this role.

If you select specific items, for example specific Views for Reports, then users (who are assigned this role) will only see the selected items listed within the corresponding module.

Clicking on the checkbox beside each of the buttons at the top ensures that all options contained within these categories are permitted for the current Role.

To view the items within a category you must clear the checkbox beside this category and then click on the category name.

The Other GUI Items includes the following categories:

Option

Description

Blotters

Enables you to assign permissions for ad-hoc Blotters.

Work Items

Enables you to assign permission for all Work Items views (My Work Items, Unassigned etc.) and for use of miscellaneous actions.

Reports

Includes permissions for the Reports Views (My Reports, Unassigned Reports, Signed-off Reports).

Admin

Includes permissions for accessing the modules that are available within the Administrators’ Customization Application (Config Parameters, GUI,Connections, Settings, Users/Groups). For more information about the Administrators application, refer to your RCM Configuration Guide.
