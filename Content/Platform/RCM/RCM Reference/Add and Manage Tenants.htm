Add and Manage Tenants

The license for this feature is purchased separately

In order to gain access to the Tenants Management tab you must assign the user the relevant role.

To activate Tenant Management:

  1. Click the Manage Roles bar and click Add Role.
  1. In the Modules tab select the Tenants Management checkbox to allow the user to access the Tenant Management tab.
  1. Click the Tenants Management tab. The Tenants table appears. A built in Tenant by the name of Host is already in the table. This is the built in Tenant that cannot be removed/updated. Its identifier is always –1.

Initially, as the administrator you are not linked to a Tenant and therefore you cannot view any segmented data that is linked to a Tenant. Segmented data is all the data that is attached to a specific Business Unit (Work Items, Cases, Blotters, Reports, DDQ, etc.). The administrator without a Tenant can only view the common data, which is the data in the Admin tab, References, Admin site (Admin.jsp), and all the data in the Designer.
