More Actions

The More Actions drop-down menu provides you with the following options:

Option

Description

Activate

Enables you to activate a rule. An activated rule is included during the next policy activation. For more information, refer to Activate/Deactivate a Rule.

Don’t activate

Enables you to deactivate a rule. A deactivated rule is not included during the next policy activation. For more information, refer to Activate/Deactivate a Rule.

Test

Enables you to create a Test rule. When a test rule is activated its results are stored only for statistical purposes. A test rule does not affect the final policy result (i.e., the final decision on what to do with the transaction such as block, delay, etc.)

Production

Enables you to change the status of a rule from a test rule to a production rule. Note that a rule that was not defined as a test rule is by default a production rule.

Priority

Enables you to define when the rule is executed in relation to the other rules (a prioritized rule). For more information, refer to Set/Remove the Rule Priority.

Non Priority

Enables you to remove the priority setting from a rule. For more information, refer to Set/Remove the Rule Priority.

Set all to  be activated

Enables you to activate/deactivate a rule. An activated rule is included during the next policy activation. For more information, refer to Activate/Deactivate a Rule.

Export all Rules

Enables to export all the rules from the policy type. For more information, refer to Export Rules.
