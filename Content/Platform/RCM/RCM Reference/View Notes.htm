View Notes

Select a work item, click the View Notes button in the Work Items toolbar to open the View Notes window, and view the history of all notes associated with the work item. You can also view notes by selecting a work item that has notes and clicking in the Work Item details page.

This action can only be performed on one work item at a time.

The View Notes window includes the following information and options:

Option

Description
Item ID

The ID of the selected work item appears at the top of the window.

Enables you to add a note to the current list of notes.

Note Date

The date and time that the note was added.

Entered By

The name of the user who added the note.

Related Action

The action that the user was performing when the note was added.

Notes can be added to a work item while performing different actions.

You can add notes from the general Work Items List by selecting any work item and then performing one of the following:

Add Note – you can add a note directly from the Work Items List or while viewing the history of notes.

Change Step – you can add a relevant note that relates to the current action from the Change Step window.

Assign to User – you can add a relevant note about this action from the Assign to User window.

You can also add notes from the Work Items Details window for any work item by selecting Add Note or Change Step at the top of the Details window.

Note

The text of the note.

View Notes that Include Tags

The View Notes window provides information on all the relevant tag values and enables you to use the tag values as links to more information about a specific value.

For example, when a user clicks on the John Smith Broker value, all the relevant notes about John Smith are displayed in all related work items and transactions.
