Configure the LDAP Connection

If you are using LDAP, you must configure the LDAP connection parameters.

To configure the LDAP parameters:

  1. Click the Configure LDAP connection action item. The Test LDAP Connection window appears, enter the appropriate values for your system.

  1. According to your system requirements, set the values for the following parameters that are maintained within the Actimize Admin Site Configuration Parameters:

Administrators can also assign values to these parameters from the RCM Admin Site application using the options under the Config Parameters tab.

Host Settings:

  • Host name or IP - sets the value for the RCM Configuration parameter actimize.ldap.computerName
  • Port - sets the value for the cm_config parameter actimize.ldap.port

Authorization Details

  • User DN - sets the parameter for the configuration parameter actimze.ldap.userDN
  • Password - sets the value for the configuration parameter actimize.ldap.userPassword

Base DN - This parameter only appears for systems that use SunOne. This parameter sets the value for the configuration parameter actimize.ldap.domainDN.

  • For Active Directory, set this parameter to null
  • For SunOne, set this to the appropriate domainDN
