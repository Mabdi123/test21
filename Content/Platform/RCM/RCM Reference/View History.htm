View History

Selecting the View History button in the Work Items toolbar switches to the Work Item History view for any selected work item. This feature allows you to view all actions and updates that were performed for the selected work item. For example, if a user changes the step of a work item or opens a work item, the type of action appears within the Action column and the details about the action appear within the History Line column.

This action may appear in the toolbar More menu. Buttons and action links are customizable depending on your organization implementation.

The Work Item History includes the following columns, by default:

Option

Description

Date

Shows the date when the action occurred.

Action

Shows all actions that were performed on this work item. For example, if you use the Change Step action, this information appears here.

Performed by

Shows which user performed the action.

History Line

Shows more detailed information about the action. For example, if someone assigns this work item to a different user, the details (previous user and new user names) appear here.
