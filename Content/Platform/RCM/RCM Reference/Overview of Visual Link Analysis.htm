Overview of Visual Link Analysis

This module assists in revealing dependencies, structures and patterns in a work item that enable you to investigate various scenarios. VLA displays relations between multiple logical/physical entities that facilitate in detecting suspicious issues or reach various conclusions.

The module consists of the following elements:

  • Drawing Canvas: Area that displays the main Drawing and enables various interactions.
  • Drawing Toolbar : Provides tools for visually working with the drawing.
  • Filters –
  • Entity Types Filter: Enables you to define the type of nodes that are displayed in the drawing.
  • The Network Types Filter : Displays a list of checkboxes for various graph type names. The selected graph(s) are displayed in the drawing.
  • Network Highlighting: Enables you to highlight a specific graph in the drawing.
  • Overview Window: High level view of the drawing that can be used to pan and zoom to a defined area of the drawing.
  • Properties: Displays the properties and attributes of selected drawing elements.

To view the Visual Link Analysis (VLA) tab you must initially select a work item that includes a link to VLA. Your organization has initially defined specific work item types that include a link for VLA.

If you click on the VLA item of the Research tab without initially selecting a specific work item then you will receive an error message.

Your organization may have not defined any work items that include VLA. In this scenario, the VLA feature is not used.
