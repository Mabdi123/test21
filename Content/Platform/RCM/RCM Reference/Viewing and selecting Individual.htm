View and Select Individual Report Data

The Dashboard includes the following different types of reports

Icon

Description

Totals for Open Cases

Displays the number of cases per specific category. See Totals for Open Cases.

Open Cases

Displays the number open cases according to a user-defined filter. See Open Cases

Open Cases Aging

Displays counts of cases in time groupings. See Open Cases Aging

Open Cases by Step and Case Type

Displays the number of steps for the selected case. See Open Cases by Step and Case Type

Open Cases by Priority

Displays the priority distribution for the open cases. See Open Cases by Priority

Average Days for Resolution

Displays the average amount of time it took to close the cases according to a user-defined filter. See Average Days for Resolution

Open Case Over Time

Displays a graph of the open cases.
See Open Cases Over Time

All the data in the reports is based on the filtering definitions that were made in the filtering toolbar.
