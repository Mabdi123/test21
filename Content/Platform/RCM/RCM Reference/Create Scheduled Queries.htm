Create Scheduled Queries

Scheduled queries enable you to define queries that can be processed in the future once or at a recurring interval.

You cannot create scheduled queries to data sources that do not belong to a specific scheduled query group. In this scenario, the Create Scheduled Query icon is not displayed. Only the implementer has the option to associate data sources to specific batch query groups via the RCM Designer.

To define a scheduled query:

  • Click the Create Scheduled Query icon. The Scheduled Queries window opens. All fields of the window must be filled in. The window includes the following fields:

Option

Description
Name:

Enables you to enter a name for the scheduled query.
Data source:

Enables you to select a data source table for the query. The default data source table that is displayed is the data source that was displayed when the Create Scheduled Query icon was clicked.
Period:

Enables you to define the period for the query. Clicking on the area next to this field opens a drop-down menu that enables you to select a period. Click the Apply icon after the selection is made.
Start Date:

Enables you to enter the initial date when the query runs. Click the Calendar icon to select a date.
End Date:

Select the last date when the query runs from the following options:

  • No end date – Query has no end date
  • End by – Select a date for the end date. Click the Calendar icon to select a date.
Query Scheduling

Enables you to define the number of times in which the query runs from the defined start date to the end date. Select from the following options.

  • Once – Query has no end date
  • Specified interval – Select daily, weekly, or monthly. For example, if you select weekly then the query run once a week between the defined start and end date.
Alert Type:

Select the type of alert for the query results. Note that only Work item types that have been selected by the RCM implementer appear in this list.
Step:

Displays the initial default step of the work item type.

Score:

Enables you to enter a score for the generated work item.
Business Unit:

Select a business unit to associate with the generated work item.

After you complete the Scheduled Queries window, click Go. A confirmation message is displayed.
