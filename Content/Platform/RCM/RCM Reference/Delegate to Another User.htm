Delegate to Another User

Delegation to another user enables you to assign your delegations to another user. Click the Delegate To Another User link in the User Info window.

The New Delegation window opens. It includes the following fields that must be filled in.

This feature is available only to users that have been provided with the relevant permissions by the administrator.

  1. Initially select either the Active or Not active checkbox. An active delegation specifies that this delegation is enabled. An inactive delegation specifies that this delegation is currently disabled.
  1. Enter the user whose permissions are delegated in the To User field. Clicking the icon opens the User picker window, which enables you to select or type a specific user.
  1. Enter the date when the delegation period begins in the From Date field. You also have an option to define a specific time for the beginning of this period.
  1. Enter the date when the delegation period ends in the To Date field. You also have an option to define a specific time for the ending of this period.
  1. Click Save. The new delegation appears in the Delegations table.

During the period that the delegation is active a delegation icon appears with the tooltip Other users may act as you.

See also Manage Delegations.

When the delegation period is over, the delegated user must log out in order to stop the delegation.
