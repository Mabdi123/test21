Trade Toolbar

The toolbar buttons that appear at the top of the Trade Details list depend on the needs of your organization and the permissions that are assigned to your role.

The following are typical buttons that are available.

Option

Description

Enables you to sort trades according to specific requirements. See Sort Options, page 1.

When there are no open work items associated with the selected trade, clicking the Review button changes the status of the trade to Reviewed.

When there is at least one open work item associated with the selected trade, click the Review button to open the Review window that enables you to:

  • Change the status of the selected trades to Reviewed
  • Add a note to the selected trades. You can choose to add a note only to those trades that have associated open work items.
  • Change the status of all associated open work items.
  • Add a note to work items when you change the status of the work items.

See also Review.

Opens the Un-Review window where you have the option to add a note and click OK to change the Status of the selected reviewed trades to Un-Reviewed. See also Un-Review.

Opens the Add Note window where you have the option to add a note to the selected trades. See also Add Note.

Opens the View Notes window where you have the option to view the notes of the selected trade (this action does not allow selecting multiple trades). See also View Notes.

Enables the users to manually generate a new work item for a specific trade. See also Create a New Work Item.

Select a Trade Blotter and click the View History button to view the audit history pertaining to the Blotter. See View History .
