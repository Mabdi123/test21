Generate a DHTML Report

When you generate a report to a DHTML format, the report instance opens within the JReport DHTML viewer. The layout of the report can vary, depending on the requirements of the organization. The following is an example of a Report Generation Wizard.

When you generate a DHTML report only, the report is not saved to the repository once this report instance is closed, and does not appear within the Report Views. You can, at any time, generate the report again to obtain a new instance of the report.
