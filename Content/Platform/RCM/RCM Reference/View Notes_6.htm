View Notes

Select a case and click the View Notes button in the Cases toolbar to open the View Notes window. This enables users to view the history of all notes associated with the case.

This action can only be performed on one case at a time.

The View Notes window includes the following information:

Option

Description

Case ID

The ID of the selected case appears at the top of the window.

Enables you to add a note to the current list of notes.

Note Date

The date and time that the note was added.

Entered by

The name of the user who added the note.

Related Action

The action that the user was performing when the note was added.

Notes can be added to a case while performing different actions.

You can add notes from the general Case List by selecting any case and then performing one of the following:

  • Add Note: You can add a note directly from the Case List or while viewing the history of notes.
  • Change Step: You can add a relevant note that relates to the current action from the Change Step window.
  • Assign: You can add a relevant note about this action from the Assign to User window.

You can also add notes from the Case Details window for any case by selecting Add Note or Change Step at the top of the Details window.

Note

The text of the note.
