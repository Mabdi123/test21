Guidelines for Local Admin Users

User/Group login names must be unique. Therefore, local admin users cannot create the same user/group although they cannot view the user/group on their list.

When a local admin creates a new user, the BU field must not be left blank. The BU picker will show all the BUs in the repository but after the selection a validation message will be displayed in a scenario where the user selected a BU that is not in his/her department.

Local admin users can only remove users that are in their department.
