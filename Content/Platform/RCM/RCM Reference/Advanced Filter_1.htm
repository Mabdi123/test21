Advanced Filter for Outgoing Forms

The Advanced Filter window enables you to filter the current view.

Click to open the Advanced Filter window to search for specified information within the current view.

The Advanced Filter window enables you to filter the current view and to show only Outgoing Forms that correspond to the search factors that you are interested in.

The list of filter criteria in the Advanced Filter window depends on the current Outgoing Forms View that is in effect and on the configuration that is setup for your organization.

The list contains all fields that are commonly used for filtering plus any specific fields, within the current view, which are supported for filtering. Some fields may be grouped according to a business category. When a business category is used, the business category appears as a search option with a beside it, showing that there are individual fields that are included within this category. Click the icon to expand the category and display the specific fields.

Search for the entire category of fields by selecting the category name when this section is closed, or expand the section and select only those fields, within the category, that you require.

Additional search options are available within the More Options tab.

You can check on one or more of the available criteria (in one or both tabs) in order to search through the list of outgoing forms and filter the Outgoing Forms view according to the filter items that are selected.

The filter option may require a range, enter from and to values for this range.

If the item does not require a range, you can enter multiple values by separating them with a semi-colon (;).

The expected values are not found in the following scenario: Searching for text that begins or ends with the ‘--' characters when then exact match option is enabled.
