Add a Tenant

You can add a tenant to as follows.

To add a Tenant,

  1. Click the Add Tenant link and enter the fields of the opening window:
  1. Tenant Name – Enter a name of the Tenant
  1. Tenant Identifier – Enter a unique identifier for the Tenant. The tenant ID cannot start with a number.
  1. Prefix – Enter a prefix for the Tenant. The prefix should not include an underscore. The prefix enables you to easily define BUs with the same meaning to different Tenants. For example, bank1_all and Bank2_all where Bank1 and Bank3 are prefixes for all. The prefix cannot be changed.
  1. Description – Enter a description for the Tenant. This field may be left blank.
  1. Click the Activate Tenant checkbox to make this tenant active immediately.
