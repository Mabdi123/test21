Edit Item

You can edit any work item that was generated. Manual items can be generated either from the Work Items List (by clicking New Item) or from any report that supports generation of manual work items.

Selecting a work item and clicking the Edit Item button in the Work Items toolbar opens the Edit Item wizard that enables you to edit details for the work item. Some fields, within the wizard, are read only and cannot be edited. Specifically, the Description field can only be edited for manual work items.

This action may appear in the toolbar More menu. Buttons and action links are customizable depending on your organization implementation.

You can also edit work items that are generated and distributed by the AIS Models.
