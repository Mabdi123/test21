General Tab

Following are examples of search criteria that may be available within the Advanced Filter window:

Option

Description

Case ID

Type the Case ID that includes forms.

Form ID

Type the Form ID that you want the RCM to search for.

Form Name

Type the Form Name that you want the RCM to search for.

Form Type

Type the Form Type that you want the RCM to search for.

Status

Type in or select the Status to be included in the search. For example you can search for all Forms that are In Process. You can enter more than one Status by separating them with semi-colons.

Create Date

View only Outgoing Forms that are generated on the specified date. Use to select a date range.

Last Update Date

View only Outgoing Forms that were updated in specific date range. Use to select a date.

Business Unit

Search for an Outgoing Form according to its business unit.

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select or type the Business unit that you are searching for.

To select a Business unit:
  1. Use the Business Unit Hierarchy drop-down list to select the hierarchy that you currently require.
  2. Once you select a hierarchy, the Hierarchy Tree shows the highest level of business units within this hierarchy.
  3. Click in the tree to open a node and display of the business units that are subordinates of the selected node.
  4. Click the Business Unit name to select it and click to select it for assignment. Once selected, this business unit appears within the Assign to Business Unit / User window.

Owner Name

View only outgoing forms that are associated with the specified owner of a case. Enter the name of the user or use the icon to open the Select User window that enables you to select the business unit that the user belongs to, and then to enter the name or initial letters for the user of interest.

To view all users within a specified business unit, enter % in the Available Users field and click Go.

If outgoing forms are not displaying as expected within a selected view, verify that the options that are checked on within the Advanced Filter window are the options that you want to view.
