Overview of On-Demand Reports

On Demand Reports are generated according to predefined templates that are assigned to each report type. Click the Research tab and select On-Demand Reports from the drop-down menu. You can select the type of report that you currently require from the directory tree in order to open the Report Generation Wizard.

All parameters that are required for the current report are included within the report template, and are presented within On-Demand Reports page of the Report Generation Wizard. The report details may require values such as dates, branch numbers, business units, etc. You can select or enter the appropriate value for each parameter, according to the available options. It is possible to filter reports via multiple business units.

Once you have entered the required values for the parameters, you have the option to:

  • Generate - Generates a report to a DHTML format (see Generate a DHTML Report).
  • Save... - Opens the second step of the Report Generation Wizard enabling you to specify the required report output (see Save On-Demand Reports to the RCM Repository).
