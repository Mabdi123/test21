Work Items Details Toolbar

The work items buttons that appear in the toolbar depend on the needs of your organization and the permissions that are assigned to your role. The following are typical actions that are available from the Work Details window.

Option

Description

Opens the Change Step window that lets you change the step of the current work item. For details, see Change Step.

 

Opens the Change User/BU window that lets you change either the Business unit that this work item is assigned to or the user that is responsible for this work item. For details, see Change Step.

Opens the Add Note window that enables you add a note to a work item. For details, see Add Note.

Opens the View Notes window that enables you to view the history of all notes associated with the work item. For details, see View Notes.

Opens the Edit Workspace window enabling you to add relevant information pertaining to the work item. For details, see Open Workspace.

Enables you to manage work item attachments. For details, see:

  • View Attachments
  • Manage Attachments

The Request for Information (RFI) feature allows you to collaborate with external and organizational users via e-mail. For details, see Create RFI.

Enables you to view specific RFI indicators. For details, View RFI.

Show as PDF - Opens a window that lets you select additional information to be included with the work items Details. Click OK within this window to display the details of the current Work Items Details in a PDF format. You can send the PDF version to a selected printer. For details, see Show as PDF.

A PDF viewer plug-in (e.g., Adobe Reader) should be installed in order to properly view any item that has been exported as a PDF file.
