Edit RFI Templates

You can edit RFI templates for use when you create and send RFI emails relating to work items. Select an existing template from the list, then click the Edit RFI Template link within the Manage RFI Templates activity bar or click the link itself to open the Edit RFI Template window.

The Edit RFI Template window contains the fields and parameters for a template and item type selection. For details, see Create RFI Templates .
