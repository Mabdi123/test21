Modify the Encryption Key

provides you with the option to modify the encryption key of the passwords in order to provide you with superior control of the secured environment. It enables you to replace the out of the box encryption key with a new encryption key stored in a standard keystore that is located in the repository.

In addition, this action enhances the out of the box encryption algorithm from DES to 3DES.To modify the encryption key go in the Admin tab of the and click the encryption properties link.

Enter a new keystore password and new alias for the new encryption key. A warning message is then displayed indicating that this action is irreversible. From this point on the uses the new key store’s key. All the passwords of the are then decrypted and encrypted again.

Enter the current password for the new key store and its respective alias (these are the alias and password that you defined in the previous screen). If you also decided to modify the external key store password then enter a new password and alias. The new encryption uses triple DES encryption and enables a secure and robust environment.
