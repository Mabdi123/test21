List Options

The options that appear depend on the permissions assigned to the currently logged on user and on the requirements of each type of list. For example, some lists may support the addition of a new item to the list, other lists may only support editing information within a selected row, and other lists may not support addition or editing.

You can select (click the checkbox) a row within the list in order to perform any of the standard operations that appear in the upper right corner of the window (below the tabs). These standard options are described in the following.

Add Item

Edit List Information

Delete List Entity

List History

Print List
