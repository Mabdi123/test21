Open Work Items

This report provides data about open items for any selected filtering option.

You can configure this report to retrieve the data that you want by using the By: drop-down menu, which has the following options:

  • View is based on the category.
  • View is based on the item type.
  • View is based on the step.
  • View is based on the owner of the work item.
  • View is based on the assigned business unit.

To choose a view, select the sorting option from the drop-down menu and click Go. The top ten values are then displayed.
