Include Host Items in the List

In a Multi-Tenant environment you are allowed access to items created or modified by you and/or other users from your tenant. When you click the Include Host Values action link then you can also items created by the users that are associated with the host. All items created by the host are indicated by the icon and can only be viewed (cannot be edited).

All bulk action items (e.g., edit all filtered items, edit all selected items, remove all items) only apply to tenant’s items. Host users can only view and edit their own items.

You can view list items that have been created by the host only when your organization provides you with the required permissions.
