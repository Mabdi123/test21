Print Menu

The Print links allow you to print trades and export information about trades to external files.

The links that appear are described in the following.

Print

Select a Trade and click Print to print the trade

The Print window includes the following options:

  • Include Trade Notes: Trade Notes are included in the printed details.
  • Include Trade History: Trade History is included in the printed details.

The printed details appear in a separate browser window. Click the Print link in order to print a hard copy of the details.

Print All Trades

Select a Trade Blotter within the Trade Blotter List and click the Print All Trades button to print the Trades for the selected Trade Blotter. Select additional information to include within the printed version of the details.

The Print All Trades window includes the following options:

  • Include Trade Notes: Trade Notes are included in the printed details.
  • Include Trade History: Trade History is included in the printed details.

The printed details appear in a separate browser window. Click the Print link in order to print a hard copy of the details.

Export

The Export links allow you to export the Trade Blotter information as a comma delimited file (CSV) that can be opened by popular spreadsheet programs such as Microsoft® Excel.

Selecting one Trade Blotter within the Trade Blotter List and clicking the Export link opens a standard Windows Download window that enables you to save the selected Trade Blotter to a CSV file format
