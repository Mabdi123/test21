The Items List Pane

The Items Lists pane displays the list of items and enables you to edit one or more test items, as shown in the following example:

If the list supports four-eye review, then you will only see approved items. Non-approved items cannot have test values.

Multi-Tenant values of the host are displayed in the list. However, you will not be able to create test values for them.

The Items List Toolbar

The Items list enables you to perform various tuning functions.

  • Your organization may use customized buttons in the toolbar and may add/remove toolbar buttons.
  • Selecting one or more lists or list items may cause toolbar buttons to be enabled or disabled depending on your organization’s configuration.

The following table describes the toolbar buttons:

Button

Name

Description

Filter

Filters the tuning list. For more information see Advanced Filter.

New Item

Add New Platform List Test Item. Enables you to manually add a platform list item to the list. Clicking this button opens the New Platform List Test Item window that enables you to add the values for the new test item.

Duplicate Item

Enables you to duplicate a platform list item. Select an item and click the button to open the New Platform List Test Item window with the selected item details displayed. You must change at least the key for the duplicated test item to be saved in the system.

The duplicated item is created as a testing item and not a production item.

Edit Selected Items

Enables you to edit the items that were selected. This button opens a window that enables you to edit all the values for the test items that are not key values.

Edit Filtered Items

Enables you to edit all the items that are filtered. This button opens the Edit All the Items window that enables you to edit all the values for the test items that are not key values.

It is not possible to delete a production value. When a test value is provided instead of a production value the test value is in bold and the production value is crossed out. For example, in this screenshot the test value is 5 and the production value is 111.

Remove Item

Enables you to remove a selected item. When the item is a production item then it is defined in the test run as a removed production item. The production item cannot be removed from the list while Test items can be removed from the list.

Remove Filtered Items

Removes all filtered items. Enables you to remove all the Platform List Items that have been filtered

Apply Selected Items

Enables you to apply the test values of the selected items. This task may be done only after running a test. See also Apply Values for Production.

Revert Selected Items

The Revert Selected Items button. Enables you to cancel the test values of the selected items. This task may be done only after applying test value(s) and running a test. See also Apply Values for Production.

Apply Filtered Items

Enables you to apply the test values of all the filtered items. This task may be done only after running a test. See also Apply Values for Production. Note that if no filter is applied then all the items are applied.

Revert Filtered Items

Enables you to cancel the applied test values of the filtered items. This task may be done only after running a test and applying test values. See also Apply Values for Production. Note that if no filter is applied then all the items are reverted.

Import/
Export Menu

Opens a menu with the following links.

Import: Enables you to import an APF or CSV file that includes a list of Platform List Items.

Export Selected Items to APF: Enables you to export a selected Platform List item to an APF file.

Export Current List to APF: Enables you to export the current list to an APF file.

Export Selected Items to CSV: Enables you to export a selected Platform List item to a CSV file.

Export Current List to CSV: Enables you to export the current list to an APF file.

View History

Enables you to view the history of actions for the Platform List.

For information on import and export see Export Items and Import Items in Platform Lists.

Four-eye Review is not available for the Tuning feature.
