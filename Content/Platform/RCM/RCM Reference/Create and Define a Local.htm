Create and Define a Local Admin

You can create a local administrator as follows.

  1. Click the Administration link in the main page and then click on the Manage Roles bar. Select to either update an existing role or to create a new role. The role definitions window appears:

  1. Select the checkbox Apply this Role to all descending Business Units in the following Hierarchy and select a BU hierarchy from the drop-down list. This local administrator is responsible for the selected BU hierarchy. Note that you do not have to select this checkbox (leaving this checkbox cleared indicates that is without a BU hierarchy relation).
  1. Select the checkbox Used for local administration.
  1. Click the Admin bar. It is recommended to select the following actions for the local Admin:
  1. Assign Local Role/BU to Group – This action allows the local Admin to assign only roles that are local to groups.
  1. Assign Local Role/BU to User – This action allows the local Admin to assign only roles that are local to users.
    • View Users
    • View Groups
    • Add Users
    • Add Groups
    • Delete Users Group
    • Assign User to Group
    • Assign Group to Group

Any other general admin actions assigned to the local admin role are not filtered according to the BUs and their respective hierarchy (department)
