The Work Items Widget Bar

The Work Items widget bar is available as follows.

According to your organization's settings the widget bar may include some or all of the following options:

  • Standard search options: The Filter by: feature enables you to filter the current list to show only those items that contain the information of interest. The Reset button removes all the filtering and sorting definitions that you have added. You can enter any string (letters or numbers) in the available field and click Go to search the current list for that string. All rows that contain the entered string appear within the list area. Clicking Refresh removes the search filter and updates the display to show all items within the list, with no filter applied.
  • Predefined actions may have been defined by your organization. To use a specific action, select it from the drop-down menu and click the Execute button.
