Adjust the Order of Prioritized Rules

This option enables you to quickly redefine a specific rule’s priority number. It is relevant only to rules that have an order number.
For example, a rule with the priority number 4 should be redefined to be a rule with priority number 2.

To adjust the order of prioritized rules:

  1. Select the rule with Priority number 2.
  1. On the Rules toolbar, click the icon that is next to Order. The rule’s order number has now been changed to 3.
  2. Select the same rule that now has priority number 3 and click the icon that is next to Order. The rule’s order number has now been changed to 4.
