New Case

Select one or more work items within the Work Items List and click the New Case link in the Cases menu of the Work Items toolbar to open the Add Items To Case wizard, and add the selected work items to a new case.

This action may appear in the toolbar. Buttons and action links are customizable depending on your organization implementation.

A case enables you to group work items, notes, and various file attachments under a single business case for further handling. Typically, a work item that requires further examination is attached to a case. Similar work items, notes, and attachments can then be added to a case. The General Info page of the New Case wizard includes the following fields and options:

Option

Description

Case Type

Select the appropriate Case Type from the drop-down list. The description of the Case type appears on the right side of the drop-down menu.

Case Name

Enter a name for this case. 

Case Date

The date and time when this case was created. Click the icon to select a specific date.

Case Description

Enter a description of the case. The Rich Text feature may be enabled. See also Rich Text Feature.

Step

The current step of the case. Steps are defined according to your organization’s needs. For example, typical step definitions may be Ready or In process.

Owner

The name of the user responsible for investigation of this type of case.

Priority

Enables you to select the priority for this case. Following are the priority options:

  – red represents high priority

  – yellow represents medium priority

  – green represents lowest priority

Business Unit

Assign a business unit for this case.

  1. Click the icon beside the Business Unit field to open the Business Unit Explorer, and select or enter the Business Unit that you require for this case.
  1. Click Close to return to the main window

Once you have completed the general information fields, proceed to the Type Specific Info page of the wizard. Click Next to proceed to this page.

The fields in this page change according to the type of case that you selected.

Note that fields indicated by an asterisk (*) are mandatory and must be completed. In some cases this page does not appear since there are no specific information fields for the Case Type.

After you add specific information that is required for this Case Type, you have the option of adding a note

To add a note:

  1. From the Type Specific Info page, click Next. The Add Note window appears. The Rich Text feature may be enabled. See also Rich Text Feature.
  1. Select a note that is available within a list of predefined notes,

    2. or

    Enter a note that describes the progress of the case.

The type of note that you can add depends on the case(s) that you selected. Different case types support different note options (for example, some case types do not support adding free text, or do not support all of the predefined note options).

To select a predefined note:

  1. Click the arrow at the end of the available field to view the complete list of predefined notes that are configured for your .

  2. You can select any existing note and then add or edit text according to your needs, or you can choose to directly type into the available field.

  3. You will be provided with the option to close the selected work items at the last stage of the Add To New Case window by clicking Next.

To close the related work item(s):

  1. Select a step that closes the work items in the New Step drop-down menu. For more information on changing a step of the work items see Change Step.

  2. Optional, add a note on the Close Work Item action.

  3. Once you click the Finish button the case appears in the case list page with its updated data. For more information on cases, see Cases.
