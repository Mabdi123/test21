Add Group

Click the Add Group link within the Manage Groups activity bar to open the Add Group wizard that enables the administrator to enter group details for the new group. The first status of the Add Group wizard enables you to enter a unique name for the group.

Once the Group Name is entered, click Next to proceed to the next status in the wizard that enables you to assign users, or other groups, to this group.

Filter BU is an attribute on a user group which was added to increase Admin level security. When a local admin has permission to view only groups on they hierarchy then this is the field that defines which group is related to the organization’s BU. See also Define Local Admin Users.

Users must first be registered within the system before you can assign them to this group.

You add internal users using the Add New User link within the User Management activity bar.

If a user or group is already assigned to the selected group, the name appears in the list box on the right side of the screen.

All users and groups (other than those that already belong to this group) appear within the left side list box. Note that the word (group) appears beside all groups within the list.

To assign users and groups to the currently selected group:

  1. Click on the required name(s) within the left list box. (You can use <ctrl> or <shift> for multiple selection.)
  1. Click the icon to bring this name into the list box on the right side.
  1. Click Save.
