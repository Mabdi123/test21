Export Items

You have the option to export items to APF and CSV files.

The APF file export options may not be displayed. Your organization has the option to enable/disable these options.

Export an APF File

To export specific items, select the items that you would like to export Click the arrow next to the drop-down menu, and click the Export Selected Items to APF link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported APF file.

Export Current List Items to APF File

To export filtered items, filter the list by clicking the Filter () button. Click the arrow next to the drop-down menu, and click the Export Current List Items to APF link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported APF file.

Export a CSV File

To export specific items, select the items that you would like to export. Click the arrow next to the drop-down menu, and click the Export Selected Items to CSV link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported CSV file.

Export Current List Items to CSV File

To export filtered items, filter the list by clicking the Filter () button. Click the arrow next to the drop-down menu, and click the Export Current List Items to CSV link. After you confirm the action via a window confirmation window, a Save as window appears. This window enables you to provide a name and a path for the exported CSV file.
