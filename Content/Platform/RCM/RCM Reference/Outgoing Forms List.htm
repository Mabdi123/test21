Outgoing Forms List

The following displays the outgoing forms list.

Click a column title in order to sort the list according to the selected column (for example, click on Create Date).

Typically, there is one column within the outgoing forms that is a link that lets you access the Outgoing Forms Details for the corresponding outgoing forms.

Select an outgoing form, by selecting the corresponding checkbox, the row within the list changes color.

The columns that appear within this list are configured according to the needs of your organization and the permissions that are currently in effect. As a result, the options that you see may not be the same as the standard list presented below.

Some of the typical columns that appear within the Outgoing Forms List are:

Option

Description


(checkboxes)
  • Click a checkbox or several checkboxes to perform an operation on the selected outgoing forms.
  • Click the checkbox in the title bar to select all outgoing forms in the current page.

An icon indicates that the corresponding case is assigned to me. When no icon appears, the case is assigned to a different user or is unassigned.

Case ID

The ID of the case where the specific form is in. Click on the Case ID in order to view it.

Form ID

The ID number of this form. This number is used internally by the RCM and can be used when referring to a form.

This Form ID appears within the list as a link and you can click the link to view the form in PDF format.
A PDF viewer plug-in (for example, Adobe Reader) should be installed in order to properly view any item that has been exported as a PDF file.

Form Name

The name of the form.

Form Type

The type of form.

Status

The status of filing the form (for example, in process).

Create Date

The date when the form was triggered.

Last Update

The when the form was last updated.

Last Update User

The last user that updated the form.

Reference

The document control number (for example, FinCEN number).

Amendment of

When the form is an amendment form then this column displays the original form ID number.
