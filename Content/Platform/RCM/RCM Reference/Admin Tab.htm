Admin Tab

Select one of the following actions for the Administration module of the :

  • All - enables users with this role to have all the available permissions for the Administration module.
  • Add Group - enables users with this role to add a group.
  • Add Role - enables users with this role to add a role.
  • Add Tenant - enables users with this role to add a tenant (relevant only for the multi tenant Multi Tenant mode of ).
  • Add User - enables users with this role to add a user.
  • Assign Group to Other Groups - enables users with this role to add a group to other groups.
  • Assign Groups to Groups - enables users with this role to assign groups to other groups.
  • Assign Local Role/BU to Group - enables users with this role to assign Local Role/business unit to a group (relevant when there are local administrators).
  • Assign Local Role/BU to User - enables users with this role to assign local role/business unit to a user (relevant when there are local administrators).
  • Assign Role/BU to Group - enables users with this role to assign role/business unit to a group.
  • Assign Role/BU to User - enables users with this role to assign role/business unit to a user.
  • Assign Users to Group - enables users with this role to assign users to a group.
  • Change Tenant - enables users with this role to change a tenant (relevant only for the multi tenant Multi Tenant mode).
  • Change User’s Groups - enables users with this role to change the groups that are linked to a user.
  • Delete Group - enables users with this role to remove a group.
  • Delete Role- enables users with this role to remove a role.
  • Delete Tenant - enables users with this role to remove a tenant (relevant only for the multi tenant Multi Tenant mode).
  • Delete User - enables users with this role to remove a user.
  • Duplicate User - enables users with this role to duplicate a user.
  • Edit Tenant - enables users with this role to edit a Tenant’s settings (relevant only for the multi tenant mode).
  • Edit User - enables users with this role to edit a user’s settings (relevant only for the multi tenant host of ).

  • Substitute for Others - enables users with this role to substitute for other users.
  • Execute Web Services – Delete Items - enables users with this role to use the Web Service of the that deletes work items.
  • Manage Dictionary - enables users with this role to manage the Terms Dictionary of the .
  • Manage Templates - enables users with this role to manage the Workflow Restrictions Template of the .
  • Personal Delegation - enables users with this role to assign delegations to other users.
  • Reset All Passwords – enables users with this role to reset all the passwords of the at once.
  • Reset Password – enables users with this role to reset a password of the .
  • Update Group – enables users with this role to update a group of the .
  • Unlock User - enables users with this role to unlock users that have been locked for an unlimited amount of time (Lock Duration has been set to -1 in the Manage Password window).
  • Update Password Policies – enables users with this role to update password policies of the .
  • Update Role – enables users with this role to update a role of the .

  • View all Users - enables users with this role to view all users of the .
