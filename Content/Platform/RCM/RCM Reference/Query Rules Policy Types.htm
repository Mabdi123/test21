Query Rules Policy Types

The Query Rules feature enables running analytical rules on relatively large sets of data by automatically translating rule conditions to SQL. A Query Rule is similar to a one-by-one Policy Manager rule since it basically states the following:

Create an issue or work item to be investigated for each and every Account/Card/Customer/Other entity that individually satisfies the rule.
