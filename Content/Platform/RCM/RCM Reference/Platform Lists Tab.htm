Platform Lists Tab

The following is an example of the Platform List tab in the Update Role window.

Select the Platform List or all Platform Lists in the Platform Lists drop-down menu.

To add a permitted action, select one of the following actions from the Actions drop-down menu and click the Add button:

  • All - enables users with this role to have all the available permissions for the selected Platform Lists.
  • Add List Item - enables users with this role to add or duplicate a list item for the selected Platform Lists.
  • Approve List Item - enables users with this role to approve a list item for the selected Platform Lists.
  • Edit List Item - enables users with this role to edit a list item for the selected Platform Lists.
  • Edit Multiple List Items - enables users with this role to edit multiple list items for the selected Platform Lists.
  • Enable Use of Host Values - Relevant to a Multi-Tenant environment. Enables the users to view and use the following related action links for a Platform List:
    • Include Host Values
    • Use Only Tenant Values
    • Export List Data - enables users with this role to export a Platform List as an APF file for the selected Platform Lists.
    • Import List Data - enables users with this role to import a Platform List APF file for the selected Platform List(s).
    • Remove All List Items - enables users with this role to remove all Platform List items via the Remove all Items link for the selected Platform Lists.
    • Remove List Item - enables users with this role to remove all Platform List items via the Remove all Items link for the selected Platform Lists.
    • View List - enables users with this role to view the Platform List(s) name(s) in the Platform Lists drop-down menu for the selected Platform Lists.
    • View List History - enables users with this role to view the Platform List history of actions for the selected Platform Lists.
    • View List Items - enables users with this role to view the Platform List items for the selected Platform Lists.

To add another permitted action, select it and click Add. You also have the option to select another Platform Type and set the desired permitted actions for it.
