Manual Work Item Entity Matching

Entity matching is also performed on a manually created work item. This includes creating a new item from the Related Items picker. The process takes place after the item is created with one of the following results:

  • No Match: Added as a new entity. The following message is displayed:

  • Doubt: Entity is created in doubt. The following message is displayed:

For details, see Doubt Resolution.

If you do not have view permissions on all existing items, when the entity matching result is match or doubt, the new item is placed in doubt and there is no UI indication displayed.
