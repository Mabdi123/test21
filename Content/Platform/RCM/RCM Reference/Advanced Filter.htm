Advanced Filter

The Advanced Filter window enables you to filter the current view and to show only work items that correspond to the search factors that you are interested in.

Click to open the Advanced Filter window to search for specified information within the current view.

The list of filter criteria in the Advanced Filter window depends on the current Work Item view that is in effect and on the configuration that is setup for your organization.

The list contains all fields that are commonly used for filtering plus any specific fields, within the current view, which are supported for filtering. Some fields may be grouped according to a business category. When a business category is used, the business category appears as a search option with a beside it, showing that there are individual fields that are included within this category. Click the icon to expand the category and display the specific fields.

Search for the entire category of fields by selecting the category name when this section is closed, or expand the section and select only those fields, within the category, that you require.

Additional search options are available within the More Options and Related Items tabs.

You can check on one or more of the available criteria (in one or more tabs) in order to search through the list of work items and filter the work item view according to the filter items that are selected.

The filter option may require a range, enter from and to values for this range.

If the item does not require a range, you can enter multiple values by separating them with a semi-colon (;).

Following are examples of search criteria that may be available within the Advanced Filter window:

Option

Description
Item ID

Enter the Item ID that you want to search for.

Item Type

Select the Item Type that you want to search for.

Item Category

Select the Item Category that you want to search for

Step

Enter or select the step to be included in the search. For example you can search for all work items that are In Process. You can enter more than one step by separating the items with semi-colons, as shown in the example below:

You can use the icon to open the Select window, which enables you to select one or more steps from the available steps in the list.

Score (optional)

View only work items that have a score within the range that is specified for this field. (If your organization does not support this feature, this option may not appear.)

Description

Search for a work item according to its description.

Item Date

View only work items that are generated on the specified item date. Use to select a date.

Closed Date

View only work items that were closed on the specified work items date. Use to select a date.

Owner

View only work items that are associated with the specified owner. Enter the name of the user or use the icon to open the Select User window that enables you to select the business unit that the user belongs to, and then to enter the name or initial letters for the user of interest.

To view all users within a specified business unit, enter % within the Available Users field and click Go.

Business Unit

Search for a work item according to its business unit.

Click the icon beside the Business Unit field to open the Business Unit Explorer, which enables you to select or type the business unit that you are searching for.

To select a Business unit:
  1. Use the Business Unit Hierarchy drop-down list to select the hierarchy that you currently require.
  1. Once you select a hierarchy, the Hierarchy Tree shows the highest level of business units within this hierarchy.
  1. Click in the tree to open a node and display of the business units that are sub ordinates of the selected node.
  1. Click the Business Unit name to select it and click OK to select it for assignment. Once selected, this business unit appears within the Assign to Business Unit / User window.

Exact Match

Select this checkbox to search only for Items that match the exact filter criteria exclusively.
If work items are not displaying as expected within a selected view, verify that the options that are checked on within the Advanced Filter window are the options that you want to view.
