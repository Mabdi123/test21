Set the Time Period

The first step in creating a query is setting the time period.

  1. Under the data source heading click the Period drop-down list and select the relevant option. Some options may open further options.

  1. Click Apply to confirm the condition.
  2. Click Go to update the query results.

The Always condition is the default setting and indicates that there is no date or time limit on the data and all of it should be included in the query.
