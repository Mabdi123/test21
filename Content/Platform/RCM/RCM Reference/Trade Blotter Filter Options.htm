Trade Blotter Filter Options

The Trade Blotters filter enables you to search through all the Trade Blotters within the current view according to the available parameters. The filtering option enables you to display blotters information according to specific criteria. Blotters are automatically recalculated when the page is refreshed.

If a filter has been applied to the list the following appears: A filter is set. Expand to modify.

If a filter has not been applied to the list the following appears: Expand to set a filter.

Click anywhere in the message bar to open the filter.

The following filter parameters are available.

Option

Description

Blotter Date

Enables you to select a starting and ending date and time in order to define a specific time frame.

Select the relevant start and end dates for the filter range and click OK.

Blotter Type

Enables you to filter the blotters via the Blotter Type.

Select or enter the relevant blotter type for the filter and click OK. Select Exclude to exclude the type selected.

Business Unit

Enables you to filter the blotters via the business unit.

Select or enter the relevant business unit for the filter and click OK. Select Exclude to exclude the BU selected.

Owner

Enables you to filter the blotters via the blotter owner.

Select or enter the relevant owner for the filter and click OK. Select Exclude to exclude the owner selected.

Trades with no owner will not be displayed when excluding a specific owner.

Unassigned: Select to include trades that are unassigned.

Assigned: Select to include trades that are assigned.

Blotter Step

Appears only if the Blotters workflow steps are enabled in your organization. Enables you to filter the Blotters via the Blotter Step.

Select the Blotter step for the filter.

Select Exclude to exclude the selected step.

All Conditions

 

Display only Blotters that match all of the conditions in the filter. This option is the same as inserting the and operator between the filtering fields.

Any Condition

Display only Blotters that match any of the conditions in the filter. This option is the same as inserting the or operator between the filtering fields.
  • Click to apply your selected filter. This filters the display and shows only those Trade Blotters that contain the specified information within the Trade Blotters List of the current view.
  • Click to remove all filters that were requested and displays the full list of the current Trade Blotters View again.
  • Click to refresh the data on display, without changing any filter selection that may be in effect. In other words, if you search for all Trade Blotters whose owner is Admin, click refresh updates the display to include the latest Trade Blotters, while maintaining your search request.
