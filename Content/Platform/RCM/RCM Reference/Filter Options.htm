Filter Options

Search options appear at the top left of the window for any task or activity that supports this feature.

The Filter by: feature enables you to filter the current list to show only those items that contain the information of interest.

You can enter any string (letters or numbers) in the available field and press Enter to search the current list for that string. All rows that contain the entered string appear within the list area.

Click to open the filter options.

  • Click x to remove the search filter and update the display to show all items within the list, with no filter applied.
  • Click Reset to refresh the Field parameter.
  • Click Close to close the filter box.

A Filter option is available, within many of the modules, which enables you to select advanced filter options that are available for the corresponding module (see the Advanced Filter Options within each of the modules for further information about the specific options that are available).
