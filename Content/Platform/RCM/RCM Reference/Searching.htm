Searching

Enter the search string in the Search… box and press the Enter key or click . A search indication is displayed while the search is loading and afterwards the search results are displayed. If there are no results then a ‘no results found’ message is displayed.

The results window is divided into 2 panes.

  • Facets tree (left pane): This pane is used for narrowing down the results found by the search. It includes all the relevant item types, item states, and time ranges. Click on a specific facet in order to add it as a filter to the results.

  • Once a facet is added, the results list is reloaded with the added filter and the facets tree displays the filtering status. You can remove a filtering facet by clicking on the x button that is next to it.

  • Results list (right pane): Displays the results of the search. The results list has a heading that indicates the amount of results displayed out of the total number of results. At the left of this pane there is an ‘expand’ button that collapses/expands the facets tree pane.

  • Each result is composed of the following details:
  • A header that displays the module and item type of the result, an identifier of the Item that is also a link that moves to the actual Item, and the business date of the result.

  • A snippet of the section where a match was found. The searched string is highlighted in yellow and is displayed in italic characters. The specific section in the Item where the match was found is also displayed.

  • Hits: A hit contains the field and the value of the item where the searched string was found. If the value is long then the hit includes only the initial portion of the value (for example, the first 100 characters of a 1000 character note are displayed). There may be occasions where many fields and values are displayed for the same item and in such scenarios a More link is displayed in order to display additional hits.

  • A Hide link may also appear in order to view fewer related fields and values of the item:

Your organization defines the amount of hits that are displayed for an item.
New items that have been added to are not immediately indexed into the global search feature and it may take a while before they are added to the search results.
You can only search for items that you have permissions to view.
If notes are encrypted using 's internal note encryption feature, their content is not searchable in Global Search. For more information, contact your system administrator for details on your organization’s implementation.
