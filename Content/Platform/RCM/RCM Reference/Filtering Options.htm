Filtering Options

The following describes the filtering options for Trade views.

Trades are filtered according to specific criteria.

If a filter has been applied to the list the following appears: A filter is set. Expand to modify.

If a filter has not been applied to the list the following appears: Expand to set a filter.

Click anywhere in the message bar to open the filter.

The filter now displays six default fields and enables you to remove and add fields.

Click beside a filter field to remove it from the display. The Trade Date and Review state fields cannot be removed.

The following is displayed automatically:

Option

Description

Filter Box

Enables you to select:

  • A specific filter to perform an action.
  • New Filter: Clears the current filter and returns to the default fields state.

The Manage Filters option only appears when you have saved filters and the permissions to either update or delete any of them

  • Save Filter: Click to save a newly created filter or update the currently selected filter.
    More Actions
  • Save as..: Click to save a newly created filter or save an existing filter with a different name.
  • Reset Filter: Click to revert the currently selected filter to its saved state and apply it to the list. 
  • Filter Description: When you select a saved filter, the description you previously entered is displayed. For example: This is a test filter

Trade Date

Enables you to select a starting and ending date and time in order to define a specific time frame. Only trades that were made during the defined time frame are displayed.

Select the relevant start and end dates for the filter range and click Close.

  • time: Set the hour, minutes and seconds of the start and end of the range.
  • Not in Range: Include trades that are not in the given range.

Review State

Enables you to select one of the following options:

  • Reviewed: Displays only the trades that have been reviewed
  • Not Reviewed: Displays only the trades that have not been reviewed

Items

Enables you to select one of the following options:

  • Has Alerts: Displays only the trades that have associated alerts.
  • Without Alerts: Displays only the trades that do not have associated alerts.

Note

Displays the trades that have notes with the entered text in this field.

Notes

Enables you to select one of the following options:

  • Has Notes: Displays only the Trades that have attached notes.
  • Without Notes: Displays only the Trades that do not have attached notes.

Review User

Enter or select a name of a user in order to view only Trades that were assigned to that user and click Close.

  • Exclude: Select to exclude the user from the filter.
  • No Value: Select to include trades with no user value.
  • Any Value: Select to include trades with any user value.

Review Date

Enables you to select a starting and ending date and time for the review time frame. Only trades that were reviewed during the defined time frame are displayed.

Select the relevant start and end dates for the filter range and click Close.

  • time: Set the hour, minutes and seconds of the start and end of the range.
  • Not in Range: Include trades that are not in the given range.
  • No Value: Select to include trades with no date value.
  • Any Value: Select to include trades with any value.

Additional Fields

Click to open the Additional Fields window, enabling you to select and display more filter fields.

  • Select one or more field checkboxes and click Add to add the fields to the displayed filter. If you select one field only, the field opens directly in the filter parameters.
  • You can type the field name directly in the search box to select and add a specific field.

All Conditions

Display only trades that match all of the conditions in the filter. This option is the same as inserting the and operator between the filtering fields.

Any Condition

Display only trades that match any of the conditions in the filter. This option is the same as inserting the or operator between the filtering fields.
