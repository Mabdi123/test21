Multiple Selection of Work Items (Bulk Items)

You can select more than one work item and click Change Step to display the Change Step window. When you select a step for the Next Step field the Reasons tab displays only the common findings reasons for the selected work items. If you de-select a checkbox the reason is removed from the steps of all of the relevant work items.

Findings Reasons checkboxes can appear as follows:

Common Reasons : All of the selected work items have previously selected specific reasons.

Indeterminate: In this state some of the selected work items have common existing reasons (but not all have). When you select the next step, the system identifies that the common reasons that were selected for only some of the work items that are now common to all for the next target step. This reason is now found on the reasons list for the next step selection and displayed with a partially selected checkbox. The clock icon indicates which step the reason was selected for or gives an explanation for an indeterminate checkbox indication.

Reasons that have been previously selected but are not common to all of the work items in the bulk selection, are not displayed and the following message appears:

<Number of reasons> Reasons that were previously selected are not listed because they are not common to all work items.

The Reasons tab may not appear if there is no platform reasons list defined.
