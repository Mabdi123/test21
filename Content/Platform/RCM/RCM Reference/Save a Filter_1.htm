Save a Filter

Click Save Filter next to the Filter box. The Save Filter window opens:

Enter a name and description for the saved filter. The description is displayed beside the filter when selected.

Select the Public Filter checkbox in order for all other users to be able to view and use the filter. If the checkbox is left unchecked then filter can be viewed and used only by the user who defined it.

The Public Filter checkbox appears only to those users that have been provided with this option by their organization.

Click Save to save this filter.
