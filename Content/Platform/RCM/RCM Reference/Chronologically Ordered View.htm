Chronologically Ordered View

To select this view, choose Chronologically Ordered View from the subtab’s menu.

This view enables you to view all the actions that were made on the case in a chorological order.

The following action buttons are available for each type of document.

Option

Description

Opens a window that lets you add a note. See Add Note.
