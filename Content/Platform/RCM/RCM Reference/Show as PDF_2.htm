Show as PDF

The Show as PDF feature supports most western character sets. For other character sets, such as Japanese and Hebrew, PDF Language Extensions must be licensed separately.

Select one or more case items within the Case Items List and click the Show as PDF button to convert the work item details for the selected item to Adobe Acrobat© PDF format.

A PDF viewer plug-in (e.g., Adobe Reader) should be installed in order to properly view any item that has been exported as a PDF file.

Depending on the supported options for the selected work item, this link may open a window that lets you select additional information that you want to include within the PDF version of the details.

The Show as PDF action link generates a version of the case item details that is converted to PDF and can be viewed in the appropriate viewer, saved as a PDF file or sent to a printer. (See Case Item Details for information about this feature.
