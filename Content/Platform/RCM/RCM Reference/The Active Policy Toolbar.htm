The Active Policy Toolbar

The following options are available in the Active Policy toolbar:

Option

Description

Filter

Enables you to filter the rules that you want to view. For details, see Filter a Rule.

Reset

Enables you to reset the display of the Rules table and remove any sorting filter from the Rules table. For details, see Sort the Table.

Refresh

Refreshes the display of the rules.

Activation History

Enables you to view the previous policies that have been activated. For details, see Historical Policies Tab - History of Activated Policies.
