Edit Dictionary

The list of strings in the dictionary may be used as part of the passwords policy. For details, see Update Password Policies. Click the Edit Dictionary link to proceed to the Manage Dictionary window:

To add a string to the list:

  1. Click the Add icon in the Manage dictionary window

    2. The Add Term window appears.

  1. Enter the string in the Value field
  1. To remove a string from the list, select the string and click Delete.
