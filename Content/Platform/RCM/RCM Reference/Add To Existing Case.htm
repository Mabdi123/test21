Add To Existing Case

Select one or more work items within the Work Items List and click the Add to Existing Case link in the Cases menu of the Work Items toolbar. The Add Items to Case window enables you to add these items to an existing case. This window includes all the open cases that you have permission to view.

This action may appear in the toolbar. Buttons and action links are customizable depending on your organization implementation.

You have the option of searching for additional cases. For example, you may want to find a closed case or a case that is assigned to another user.

To find additional cases:

  1. Click the Find Other Cases button. The Advanced Filter window appears. For more information on this filter, see Advanced Filter.
  2. Select the case that you want to add to the selected the work item(s).

You can select only one case when you are adding work items to an existing case.

At this stage you may complete the procedure by clicking Finish or alternatively, to add a note, click Next. For more information on adding a note to a case, see Add Note.

You may complete the procedure by clicking Finish or alternatively, to change the step of the work item(s), click Next to proceed to the Close Work Items window. You are also provided with the option to close the selected work items at the last stage of the Add To Case window. For example, you may close the work items that have been attached to a case and add a relevant note on this action, which will be attached to the work items. For more information on changing a step of a work item, see Change Step.

Certain steps may be disabled because of review rules.

Once you click Finish the case appears in the Case List page with its updated data. For more information on cases, see Cases.
