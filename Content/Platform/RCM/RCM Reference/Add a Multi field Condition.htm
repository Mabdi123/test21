Add a Multi-field Condition

The Multi-field conditions enable you to search for two or more values, for example, Account Key and Product Key.

The Add Multi-field Condition link should be used only when one data source is connected to another data source by more than one Business Type.

This feature allows the user to select a saved query from another data source and use it as a sub query within the parent query.

To add a multi-field condition link

  1. Click the Add Multi-field Condition link.

  1. Click the icon to view a list of available data source fields.

  1. Select the mutual fields for the two data sources. Click the <ctrl> or <shift> keys in order to select multiple data source fields.

  1. All sub queries for data sources that have the selected mutual fields are displayed in the drop-down menu next to In Sub Query. If no sub queries are displayed then there are no data sources that have the selected mutual fields.

  1. Click Apply to confirm the condition.
  2. Click Go to update the query results
