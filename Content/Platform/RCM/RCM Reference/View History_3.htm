View History

Select the View History action link to switch to the Trades History view for any selected trade. This feature allows you to view all actions and updates that were performed for the selected Trade. For example, if a user reviewed a trade, the type of action appears within the Action column and the details about the action appear within the History Line column.

The View History includes the following columns, by default:

Option

Description

Date

Shows the date when the action occurred.

Action

Shows all actions that were performed on this trade. For example, if this trade has been reviewed then this information appears here.

Performed by

Shows which user performed the action.

History Line

Shows more detailed information about the action. For example, if someone added a note while switching a reviewed trade review to Un-Review then this information appears here.
