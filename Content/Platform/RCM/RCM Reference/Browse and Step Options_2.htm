Browse and Step Options

A navigation and Step bar, at the top of the window, is displayed and includes the following options:

Option

Description

Enables you to return to the main page that lists the cases according to the last view. For example, if the last view was My Cases then this will be the view when you return to the main page.

Clicking the arrow opens a window with the name of the previous page that you viewed. Click this name in order to move to that page.

or

Provides information about the step of this case. The icons show whether the case is open or closed (For more information about case step, see Change Step).

The current Case Name and Identifier.

Browse options enable you to move to the previous or next case details according to the list within the Case View that is currently active.

There is an upper limit to the number of Case Details pages that you can browse through at one time. If you reach the upper limit you are notified and you can return to the general Case list, select to view the details for the next case and then start browsing again.

The Deadline message indicates the date that you have before the deadline for a current step occurs. This is only relevant if a workflow was defined for cases. See the Workflow subtab for more details.

Indicates the current case that you are viewing in relation to the total number of cases that are within the current Case View.
