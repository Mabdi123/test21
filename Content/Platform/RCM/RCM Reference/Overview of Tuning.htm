Overview of Tuning

The tuning feature enables you to manage and edit test values.

The tuning feature is accessed from the RCM application à Settings à Tuning tab.

This feature currently supports testing values only in Platform Lists.

This feature enables you to do the following:

  • Select a tuning context
  • View Platform List items Access settings via the tree navigation.
  • Edit test items’ values.
  • Run tests that utilize the values.
  • View results of the test run.
  • Apply the selected test values to the lists as production values.

Your organization has the option to restrict actions in the tuning feature.

The following terms are used for this feature:

Term

Definition

Tuning Context

An instance of the module. For example, Account Security Blanket Sandbox or Monthly Thresholds Sandbox.

Test Value

A value for a Platform List item which is different than the production value of that item.

Test Item

A Platform List item with one or more test values.

Test (Run)

The results of a single run with a set of test values.
