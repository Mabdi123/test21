Content Tab

The Content tab contains a list of all related work items, notes, RFI’s, attachments, and forms that have been added to the case item.

The Contents tab contains the following fields:

Option

Description

Category

The content type icons, including:

Notes

RFI

Attachments

Related work items

Forms

State

The state that the item is in.

Date

The date the content was created or the work item business date.

Content ID

Click the relevant content link to view the content details.

Content Type

The type of content.

User

The owner of the content.

Description

The description of the content.
