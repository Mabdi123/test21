Import Items from an APF File

To import items:

  1. Click the arrow next to the drop-down menu, and click Import action link. The Import Items window opens.
  1. Click the Browse icon and select the APF or CSV file that you want to import. If you would like to indicate all new and modified items as unapproved then select the Mark all modified items as unapproved checkbox. This checkbox appears only in lists that require four eye review and to users that have Approve permissions. See also Import Items Effect on Existing Items.
