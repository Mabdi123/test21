Overview of the Settings Tree

The Settings Tree enables you to quickly find a specific setting or list item and view it. Selecting a specific leaf displays its contents.

The displayed options and parameters are based on your specific solution, and therefore may appear entirely different from the examples in this guide. For solution specific information, refer to your solution documentation.

Key features include the following:

  • Quickly navigate to specific leaves via the Settings Tree.
  • Find leaves in the Settings Tree via its filter utility.
  • Add setting rows or list items.
  • Edit values in a leaf.
  • View the history of changes that were made to a leaf.
  • Export the data to a CSV file.
  • Print the details.
