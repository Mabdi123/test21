Overview of DART

The license for this module is purchased separately.

The following are common use cases for the DART research tool.

  • Research transactional data related to solutions
  • Analyze historical data and alerts
  • Automate research and alert generation

DART Key features include:

  • Creating a new query
  • Adding multiple conditions to a query
  • Saving, renaming and deleting queries
  • Referencing one query from another
  • Exporting query results to spreadsheet programs
  • Running scheduling queries that require heavy processing to a later time
  • Creating queries that are based on sequences
  • The DART module includes a Data Source drop-down menu of the available Data Source files.
