Assign to User/Business Unit

Select a report and click on the Assign to User/Business Unit button on the Report toolbar to enable an authorized user to select a different business unit than the one that this report is currently assigned to. You can use the Business Unit Lookup option to find the business unit that you want to assign to this report. (See Selecting a Business Unit).

Authorized users can also assign the report to other users from this window. Note that, if you assign this report to a new business unit, you must make sure that the selected user has permission to view this report. (For more information about permissions, see About RCM Permissions.
