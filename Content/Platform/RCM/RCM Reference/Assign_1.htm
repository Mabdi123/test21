Assign

Select a Blotter and click the Assign button within the Blotter toolbar to open the Change Blotter User/BU window that enables users to manually assign the selected Blotter to:

  • A different business unit
  • A different user than the one who is currently the owner of this Trade Blotter.

When you assign a new user you must make sure that this user has permissions to deal with this Trade Blotter.

The Change Item User/BU window includes the following information and options:

Option

Description

Blotter (s) current BU

Shows the Business unit for which this Trade Blotter is currently assigned.

New Business Unit

Click the   icon beside the New Business Unit field to open the Business Unit Explorer, which enables you to select or type the Business Unit that you require for this Trade Blotter.

This window includes an auto-complete feature that enables you to enter part of any name in the BU fields and then receive a drop-down list of all names that contain this name, as shown below.

Assign to me

Allows you to assign the selected Trade Blotter to yourself, indicating that you are the user who is responsible for dealing with this Trade Blotter.

Assign to user

Click the    icon beside the Assign to user field to open the User Explorer, which enables you to select or type the user that you require for this Trade Blotter.

This window includes an auto-complete feature that enables you to enter part of any name in the and then receive a drop-down list of all names that contain this name.

If you assign the Trade Blotter to a new user, you should make sure that the selected user has permission to view this Trade Blotter.

Unassigned

When Unassigned is selected, no user is assigned to this Trade Blotter.
