Apply Test Items of the Selected List

The items list toolbar includes the following options:

Button

Name

Description

Apply Selected Items

Applies the test values of the selected items as production values.

Revert Selected Items

Reverts the test values of the selected items to production settings.

Apply Filtered Items

Applies the test values of filtered items to production values. If no filter is used then all of the list’s items are applied.

Revert Filtered Items

Reverts the applied filtered test values of the items. If no filter is used then all of the list’s items are reverted.
