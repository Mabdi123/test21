Example 2-User-Defined Values for Thresholds

A setting entry may be defined that requires the user to set a value, for example the score that should generate a work item.

In the above example, the user can change the minimum score for generating an work item from a specific rule. The default value remains the same (and continues to appear in the Default Value column) but the system uses the defined User Value instead of the default value. You can remove the user-defined value at any time in order to return to the default value.
