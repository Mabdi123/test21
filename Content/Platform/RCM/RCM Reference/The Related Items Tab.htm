The Related Items Tab

The Related Items tab enables you to search for work items that are related to one another. For more information on work items that are defined as entities, see Enter Data in Entity Related Fields. The following illustrates an example of this filtering mechanism.

Your organization may not use these types of entities and/or may not have any defined entities

To search for related items:

  1. Click the Related Items tab. This tab displays all of the ’s defined entities.

  1. Click the icon of a relevant field. Select the Related Items that you would like to filter. The selected items appear in the Selected Items display area. You also have an option to create a new Item (in this example it is Suspect) by clicking the New Item link. This link opens the New Work Item wizard for the specific type of work item that you want to add. After the new item definition is finished it is added to the Items Picker window where it can then also be selected.

  1. Click OK. The Filtered results appear on the screen.
