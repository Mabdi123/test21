DART Query Types

The following are the types of queries you can create and manage using DART.

  • Simple queries

    • In a simple query, the data of a selected data source is filtered by indicating criteria of interest based on a specified time period and at least one of the available columns (fields) within the data source.

  • Complex queries

    • In a complex query, multiple query conditions are defined, separated by the AND or OR operators. By default every new condition includes the AND operator.

    Click the AND/OR link to change an OR back to an AND. You can combine multiple conditions in a single query, however only the following variations are supported:

  • Using AND operators for all conditions.
  • Using OR operators to divide the conditions into a few sets when all conditions in each set use the AND operator.

Multi-field queries that contain sub-queries

In a multi-field condition, you can add a sub-query to the current query.
