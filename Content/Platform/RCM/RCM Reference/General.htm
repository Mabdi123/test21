General

The following is an example of the General tab of the Settings permission:

Select the following actions for the following methods:

  • Setting with no work item type.
  • Settings for a specific Work Item Type or for all Work Item types:
  • All - enables users with this role to use all the available permissions for General Settings.
  • Add - enables users with this role to add a record.
  • Delete - enables users with this role to delete a record.
  • Update - enables users with this role to update a record.

To add another permitted action, select it and click the Add button.
