Ad-Hoc Blotters

Select the following actions for Ad-Hoc Blotters:

  • All – enables users with this role to have all the available permissions for Ad-Hoc Blotters.
  • Delete Private Filter - enables users with this role to remove a private Ad-Hoc Blotter filter.
  • Delete Public Filter - enables users with this role to remove a public Ad-Hoc Blotter filter.
  • Save Private Filter - enables users with this role to save a private Ad-Hoc Blotter filter.
  • Save Public Filter - enables users with this role to save a public Ad-Hoc Blotter filter.
  • Update Private Filter - enables users with this role to update a private Ad-Hoc Blotter filter.
  • Update Public Filter - enables users with this role to update a public Ad-Hoc Blotter filter.
  • View - enables users with this role to view Ad-Hoc Blotters.

To add another permitted action, select it and click the Add button.
