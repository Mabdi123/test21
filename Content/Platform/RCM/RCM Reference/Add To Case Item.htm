Add To Case Item

Select one work item within the Work Items List and click the Add to Case Item in the Work Items toolbar. The Add to Case Item window enables you to add a work item to a new or existing case item.

This action may appear in the toolbar. Buttons and action links are customizable depending on your organization implementation.

You have the option of searching for additional cases. For example, you may want to find a closed case or a case that is assigned to another user.

To add to a new case:

  1. In the Add to Case Item window select the Add to a new case checkbox. The New Case Item wizard opens.

  1. Enter the relevant details in the Data Entry window and click Next. Proceed through the New Item wizard. For details see, New Item.
  1. At this stage you may complete the procedure by clicking Finish or alternatively, to add a note, click Next. For more information on adding a note to a work item, see Add Note.

You can also change the step of the work item, and then click Next to proceed to the Close Work Items window. You are also provided with the option to close the selected work items at the last stage of the Add To Case window. For example, you may close the work items that have been attached to a case item and add a relevant note on this action, which will be attached to the work items. For more information on changing a step of a work item, see Change Step.

Certain steps may be disabled because of review rules.

To add to an existing case:

  1. In the Add to Case Item window select the Add to an existing case checkbox. Select a case item from the list and click Add to Case.

  2. At this stage you may complete the procedure by clicking Finish or alternatively, to add a note, click Next. For more information on adding a note to a case item, see Add Note.
  3. You can also change the step of the work item, and then click Next to proceed to the Close Work Items window.
