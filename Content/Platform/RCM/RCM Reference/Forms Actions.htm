Forms Actions

You can perform the following with forms and form attachments.

Button

Description

Click to open the Select form type window.

Select a form and click Remove to delete the form.

Select a form and click to open the Add Attachments window. Click Add File to add a file to the selected form. You can add a description to the attachment.

Select a form and click to open the View Attachments window. You can view the attachments that were added to the form.

You can also add and remove attachments from this window.

Select a form and click to convert the form to PDF format. A new window opens and displays the file in PDF format. Note that a PDF viewer (e.g., Adobe Reader®) must be installed.

Performing form actions is dependent on your user permissions, the form or workflow status and the form type definition.
