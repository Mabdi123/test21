New Case

Once suspicious activity is determined, a new case can be created for all the relevant work items, notes, and files. Click the New Case button in the cases toolbar to open the General Info page of the New Case window.

The New Case window includes the following options:

Option

Description

Case Type

Select a category for this case that is appropriate for this task from the drop-down menu.

Case Name

Enter a name for this case. 

Case Date

Select the date and time when this case was created. Click the icon to select a specific date.

Description

Enter a description of the case.

Step

The current step of the case. Case steps are defined according to your organization’s needs. For example, typical Step definitions may be Ready, In process.

Priority

Enables you to select the priority for this case. Following are the priority options:

  – red represents high priority

  – yellow represents medium priority

  – green represents lowest priority

Owner

The name of the user responsible for investigation of this type of case. When you select to view only My Cases, the user name is always your name.

Business Unit

Assign a business unit for this case.

Clicking on the icon beside the Business Unit field opens the Business Unit Explorer, which enables you to select or type the business unit that you require for this case. See Business Unit Explorer.

Click Close to return to the main window.

Once you have completed the General Information fields, click Next to proceed to the Type Specific Info window.

The fields in this window change according to the type of case that you selected. Note that fields indicated by an asterisk (*) are mandatory and must be filled in. In some cases this window does not appear since there are no specific information fields for the case type.

In this stage, you have the option of adding a note by clicking Next. The Add Note window appears.

Select a note that is available within a list of predefined notes by clicking the arrow and selecting one from the drop-down list, or select Free Text and type in a note that is relevant for describing the progress of the case.

The type of note that you can add depends on the case(s) that you selected. Different case Types support different note options (for example, some Case Types do not support adding free text, or do not support all of the predefined note options).

To select a predefined note:

  1. Click the arrow at the end of the available field to view the complete list of predefined notes that are configured for your RCM.
  1. Select any existing note and then add or edit text according to your needs, or enter information directly into the available field.
  1. Click Finish. The new case appears in the list of all cases.
