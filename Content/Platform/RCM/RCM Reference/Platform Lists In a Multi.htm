Platform Lists In a Multi-Tenant Environment

The license for this feature is purchased separately

The following is relevant only to Tenant users that use an configured for a Multi-Tenant environment. The following icons may be displayed:

  • The icon indicates a Multi-Tenant Enabled list.
  • The icon indicates Multi-Tenant enabled lists that include items created by the Host tenant.

If you are using the business calendar feature and the support host values are enabled, then the host values will be counted for the relevant tenant. For details see Business Calendar Lists.

By default, when you select a Platform List you are able to view items that have been created by the host and by you or other users that are linked to your tenant. You can only view items that have been created by the host (these items cannot be edited). You can edit items that have been created by you or other users that are linked to your tenant. In the items table, all items created by the host are also indicated by the icon.
