Actions

Click on the Actions link to view and select all the available permissions. You also have the option to select the Actions checkbox to enable all Admin site permissions

Connections Module

Includes permissions for the Connection module of Admin site.

  • Conn. Actions – select this checkbox to provide all the connection permissions for users with this role.
  • New Connection – select this checkbox to provide the permission to create a new connection for users with this role.
  • Edit connection – select this checkbox to provide the permission to edit a connection for users with this role.
  • Delete Connection – select this checkbox to provide the permission to edit a connection for users with this role.
  • Diagnostic - select this checkbox to provide the permission to use the diagnostic tool for users with this role or alternatively select the Diagnostic checkbox below it.

Conditional Settings Module

Includes permissions for the Conditional Settings module of Admin site:

  • New - select this checkbox to provide the permission to create new conditional settings for users with this role.
  • Edit - select this checkbox to provide the permission to edit conditional settings for users with this role.
  • Delete - select this checkbox to provide the permission to delete conditional settings for users with this role.

Parameters Module

Includes permissions for the Config Parameters tab of Admin site.

  • Config Actions - select this checkbox to provide all the Config Parameters tab permissions for users with this role.
  • Edit Parameter - select this checkbox to provide permissions to edit parameters in the Config Parameters tab for users with this role.
  • Reload Data - select this checkbox to provide permissions to edit parameters in the Config Parameters tab for users with this role

GUI Items Module

Includes permissions for the GUI Items module of Admin site:

  • GUI Actions - select this checkbox to provide all the GUI Item tab permissions for users with this role.
  • Create GUI Item - select this checkbox to provide the permission to create a GUI item for users with this role.
  • Change to Action - select this checkbox to provide the permission to use the Change to Action action link for users with this role.
  • Change to Tool - select this checkbox to provide the permission to use the Change to Tool action link for users with this role.
  • Edit GUI Item - select this checkbox to provide the permissions to edit a GUI item for users with this role.
  • Delete GUI Item - select this checkbox to provide the permissions to delete a GUI item for users with this role.
  • Apply Changes - select this checkbox to provide the permissions to apply changes for users with this role.

Thresholds Module

Includes permissions for the Thresholds module of Admin site:

  • Thresholds Actions - select this checkbox to provide all the thresholds permissions for users with this role.
  • Settings Actions - select this checkbox to provide the Settings Actions permissions for users with this role.
  • New Setting - select this checkbox to provide the permission to create a new threshold setting for users with this role.
  • Edit Setting - select this checkbox to provide the permission to edit a threshold setting for users with this role.
  • Delete Setting - select this checkbox to provide the permission to delete a threshold setting for users with this role.
  • Conditional Settings - select this checkbox to provide the thresholds’ Conditional Settings permissions for users with this role.
  • New Conditional Setting - select this checkbox to provide the permission to create a new Thresholds Conditional Setting for users with this role.
  • Edit Conditional Setting - select this checkbox to provide the permission to edit a new thresholds Conditional Setting for users with this role.
  • Delete Conditional Setting - select this checkbox to provide the permission to delete a new thresholds Conditional Setting for users with this role.

AISConnections Module

Includes permissions for the AIS Connections module of Admin site:

  • AIS Connections Actions - select this checkbox to provide all the AIS Connections permissions for users with this role.
  • Add AIS Connection - select this checkbox to provide the permission to add an AIS connection for users with this role.
  • Edit AIS Connection - select this checkbox to provide the permission to edit an AIS connection for users with this role.
  • Remove AIS Connection - select this checkbox to provide the permission to remove an AIS connection for users with this role.
