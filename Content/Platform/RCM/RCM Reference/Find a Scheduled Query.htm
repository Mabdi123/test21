Find a Scheduled Query

The Filter by: feature enables you to filter the current list to show only those items that contain the information of interest.

You can enter any string (letters or numbers) in the available field and press Enter to search the current list for that string. All rows that contain the entered string appear within the list area. Click x to remove the search filter and updates the display to show all items within the list, with no filter applied.

Click to update the data on display, without changing any filter selection that may be in effect.

Filter Options

  1. Click the icon next to Filter to display the filtering options.

  1. Enter information in one or more of the filtering fields and click the Go icon to view the results.
  1. Click the Reset icon to clear all the advanced filtering fields.
