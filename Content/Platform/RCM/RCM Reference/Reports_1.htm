Reports

Select the checkbox for providing all the GUI Items reports permissions.

Click on the Reports link to select specific GUI Items permissions.

  • Views - select this checkbox to provide all the view permissions for all the reports that are listed in the window.
  • On-Demand Reports - select this checkbox to provide the view permissions for the On-Demand reports.
  • My Active Reports - select this checkbox to provide the view permissions for the My Active Reports
  • My Reports - select this checkbox to provide the view permissions for the My Reports reports.
  • All Active Reports - select this checkbox to provide the view permissions for the All Active Reports.
  • All Reports - select this checkbox to provide the view permissions for the All Reports.
  • Unassigned Reports - select this checkbox to provide the view permissions for the unassigned reports.
  • Awaiting Sign-Off Reports - select this checkbox to provide the view permissions for the Awaiting Sign-Off reports.
  • Signed Off Reports - select this checkbox to provide the view permissions for the Signed-Off reports.
  • Archived Reports - select this checkbox to provide the view permissions for the Archived reports.
