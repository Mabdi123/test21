Create RFI

The Request for information (RFI) feature allows you to collaborate with external and organizational users via e-mail. It provides you with the ability to select a case item and send an email for more information on the item. Once the email is answered it is attached to the correct item.

You can also select a pre-defined template to create your RFI. For details on creating an RFI template, see Manage RFI Templates.

This action may not be enabled by your organization and therefore may not appear in the toolbar.

To create an RFI request:

  1. Select a case item and click on the Create RFI button in the RFI menu.
  2. The Create RFI window opens. Complete the fields and click Send.

It includes the following fields:

Option

Description

Templates

Select a template from the drop-down list. The template details populate the relevant fields. If no template was defined for the work item type, then this field is not displayed. Selecting Blank Template clears all RFI fields.

RFI Name

Enter a name for the RFI. 

To, CC, Bcc

Enter the email addresses of the recipients in one or more of these fields. These fields are identical to email message fields. Use a ';‘ to separate between addresses.

Subject

Enter a subject for the RFI. Identical to the subject of an email message.

Attach item data as PDF

Select this checkbox to convert the case item details to an attached PDF file that is sent to the recipients. Note that this may not be available for specific work items.

Message

A text box for entering data to the recipients on the RFI. Identical to the body of an email message.

RFI supports plain text only.

The current attachments are displayed beside the Manage Attachments button when you have selected a template that automatically adds all item attachments.

Click the Manage Attachments button to view and add specific work item attachments to this RFI. Note that you cannot add URL based attachments.

The View Attachment permission is required to use a template with automatic added attachments selected and to view the attachments.

Using two number signs (##) in a row of text in the RFI body, deletes the row.
