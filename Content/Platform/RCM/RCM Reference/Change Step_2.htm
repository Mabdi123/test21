Change Step

This option only appears when the logged on user has permission to change the step of cases. You can select one or more cases in order to change the step of the case(s). Click the Change Step button in the Cases toolbar to open the Change Step window.

The Change Step window lets you select the required step from a drop-down list of available items. The available step assignments are defined according to your organization’s requirements.

Each step definition includes three attributes, as follows:

Option Description

The name of the step that you want to assign to this case. For example, your organization may include a Step In Process that indicates that the case is still under consideration and has not yet been resolved.

Each step is defined as being in either an Open or Closed state. The state of the case is part of the case step definition, and is automatically displayed when you select a step. For example, if you select the No Violation Determined step, this is usually associated with the Closed state.

Each step is defined as one of the following:

Issue – this step requires further action

Non-issue – this step does not require further action

No-Determination – this case has not been evaluated.

Whether or not a step is considered an issue is set as part of the definition of the step. For example, an In Process step may be defined as associated with being an issue.

Filter the Step List within the Change Step Window

If the list of available Step items is very long you may want to filter this list so you can view only those Step names that are relevant. For example, you might want to view only those Step items that are considered Issues.

To filter the list of available step names:

  1. Select the Change to Step drop-down arrow.
  1. Click on the small arrow beside the State - icon or beside the Issue icon to select the appropriate State/Issue of interest.

    2. For example, if you want to view only steps that are defined as Issues, you would click the icon to open a selection box. Similarly, if you want to filter the list of steps to view only those that are open, you can click on the icon.

  1. Select the appropriate item within the resulting drop-down list (for example, ).

    2. Only the step names that are associated with the selected item appear within the drop down list of steps. For example, if you selected to filter the list by selecting Open, only those steps that are considered Open appear within the list of steps.

You can select All in order to see all steps again.

Once you have selected the step that is appropriate for the selected case(s), clicking changes the step and records the action in the Work Items History.
