Print/PDF/Email Options

You can perform the following actions from the Case Item Details window:

Option

Description

Print - Print Details is not supported for case items.

Show as PDF – Enables you to display the details of the current case item in a PDF format. You can send the PDF version to a selected printer. For details see Show as PDF.

A PDF viewer plug-in (e.g., Adobe Reader) should be installed in order to properly view any item that has been exported as a PDF file.

Send Mail - Opens the email exchange that enables you to send a link to a designated email recipient. The recipient can click on the link to open the Case Items Details. For details, see Send Mail.
