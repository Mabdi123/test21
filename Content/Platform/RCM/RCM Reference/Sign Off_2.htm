Sign-Off

Select one report (click the checkbox beside the required report) and click the Sign-Off button in the Report toolbar. This opens the Sign-Off Report Info window.

This window includes the following information and options:

  • User Name: The current user that is signing off the selected report (taken from the name of the currently logged on user),
  • Current Date: The current date and time indicating that this report is being signed off at this time.
  • Report Status: The report status can be New, Viewed, Signed-Off,  Archived or Purged.
  • Comment: Enables you to type in any comment that may be relevant for this report. You can view these comments again within the Description column of the History tab when you view Notes for this report (See View Notes).
  • Sign Off: Clicking the Sign Off button changes the status of the selected report to Signed off and closes this window.
