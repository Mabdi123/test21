Attachments

The following displays the attached files associated with the case.

Attachments are files or links that are manually uploaded by the user in order to be attached to the case. Note that you may not have permissions to add links. Depending on the configuration of your RCM, this action may only appear within the Cases toolbar when you access the details of a case. The default for the maximum size of a single attachment is 200 KB and 300 KB for multiple attachments (it can be increased up to 50 MB).

To add an attachment:

  • Select Add File or Add URL.

To add a file:

  1. Click the Add File button and enter the path and the file name that you want to upload in the window that appears.
  1. You can add more than one file at a time. If you would like to add another file click Add File button again and repeat the previous steps.
  1. Optionally, add a description and a note to the attachment. For more information on adding notes see Add Note.
  1. Click the Upload button.

To add a URL:

  1. Click the Add Link button and enter the following in the window that appears:
  1. Link name – provide a name for the link.
  2. Enter the URL of the link (for example, http://www.actimize.com). Note that you cannot add a link to your local machine, but only a Web or a server link.

  1. Click Add.
  1. You can add more than one link at a time. If you would like to add another link click Add link button again and repeat the previous steps.
  1. Optionally, add a description and a note to the URL. For more information on adding notes see Add Note.
  1. Click Upload.

You can add multiple links and files and click the Upload button once.
