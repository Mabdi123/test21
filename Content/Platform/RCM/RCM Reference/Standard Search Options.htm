Standard Search Options

The following standard search field enables you to search for reports according to criteria.

Enter any string in the field and press the Enter key to search through all the reports within the current view. This filters the display and shows only those Reports that contain the specified information within the Report List summary. If the current view includes multiple pages, the search is conducted on all pages of the list.

When a filter is in effect an icon and information about the filter appears within the title bar. Following is an example of a filtered Work Items view:

  • Click to open the filter box for more options.

  • The Field: drop-down list enables you to filter according to specific fields (for example, Case ID or Case Name).

  • Select the Exact match only checkbox to filter according to filter according to your exact entry.
  • Click x to remove the search filter and update the display to show all reports within the list, with no filter applied.
  • Click Reset to refresh the Field parameter.
  • Click Close to close the filter box.
  • Click to update the data on display, without changing any filter selection that may be in effect. For example: if you search for all reports whose owner is Admin, clicking Refresh updates the display to include the latest reports, while maintaining your search request
  • Click x to close and remove all filters that were applied (removes the string from the Filter field and removes any Advanced Filters that may be in effect) and displays the full list of the current Reports View again.
