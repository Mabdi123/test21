Tuning Tree

You access the tuning tree from the Test Values tab. The tuning tree enables you to quickly find a specific list and view its contents in the right pane.

The Test Values tuning tree navigation options are described in the following table:

Option

Description

Filter by:

Filters the Tuning Tree to show only the items in the tree that contain the entered text.

In the text field, enter text and click .

To display the tree in its original state (with no filter), click .

Collapse All

Collapses the branches in the tree

Expand All

Expands all of the branches in the tree

Previous

Displays the parameters of the previous branch in the right side of the screen

Next

Displays the parameters of the next branch in the right side of the screen
