Import Query Rules

You can import selected Query Rules into the Policy Type that you are currently viewing.

To import Query Rules:

  1. Click the Import button. The Import Rules window opens.

  1. Click the Browse button and select the APF file that you want to import. The imported APF file must include Query Rules that are relevant to the Policy Type you are viewing.

Query Rules in Multi-Tenant mode: If the Query Rules that you are importing already exist then the imported rules overwrite the Planned state and the host recommendation values of the existing rule.
