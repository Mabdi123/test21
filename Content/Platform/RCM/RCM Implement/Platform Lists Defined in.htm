Platform Lists Defined in the RCM

You can create lists that are defined in the RCM: These lists are completely defined via the RCM Designer as follows.

  1. From the Platform Lists tab, click the icon. A window opens.

  1. Select Define it in the RCM: The list is defined in the RCM Designer. If you decide to use this type of platform list, click Finish. The platform list General tab automatically opens.
