Access the Queries Options

To access the tools and options for defining queries:

  • Click the Queries bar and then click the Queries tab in the Entities pane.

The icons available for this task are as follows:

Icon

Description

Displays the following three tabs on the right side of the screen that enable you to define a new query:

  • General – Enables you to add a name and description for the new query
  • Query Definition – Enables you to enter the sql query and define parameters for this query
  • Result Display – Enables you to setup the display of the results of the query

Selecting a query within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected query.

Deletes a selected query. You must close any page associated with this query before you can delete it.

Duplicates the selected query. The copy is named “copy of <query name>.

Refreshes the query list.
