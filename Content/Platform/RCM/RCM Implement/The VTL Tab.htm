The VTL Tab

The VTL tab enables you to modify the RCM UI layout rendering.

Select a UDDT from the User Defined Data Types tab in the General bar. Click the VTL tab.

The VTL can be changed or edited according to relevant requirements in RCM.

It is not mandatory to configure the VTL.
