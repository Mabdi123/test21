v_acm_report_parameter_values

Description: Returns value about report parameters

Field Name

Description

report_instance_id

The report internal Join ID (v_acm_report_instances2)

parameter_name

The report parameter name

data_type

The report parameter data type

number_value

*Numeric value of the parameter value

float_value

*Float value of the parameter value

date_value

*Date value of the parameter value

string_value

*String value of the parameter value

bool_value

*Boolean value of the parameter value

business_type

The business type of the value (Alert_Type, Case_Type, Report_Type, Business_Unit, Report_Instance_BU, Drill_Down_Query, Users, None)

* Value is in only one of the value fields (depends on the attribute data type).
