v_acm_batch_jobs

Description: This view lists existing DART scheduled queries.

Field Name

Description

job_identifier

The batch job identifier

name

The batch name

where_clause

The DART batch where clause.

src_db_view_name

The Table or DB view that the query will be executed on.

target_db_view_name

The Table the query result will be inserted to.

period_field_column_path

The location of the period field in the source Table or DB view

batch_id_field_column_path

The location of the batch id in the target Table.

period_from

The period from which the query is scheduled.

period_to

The period to which the query is scheduled.

owner_identifier

The user who generated the batch job

create_date

The date when the batch job was created
