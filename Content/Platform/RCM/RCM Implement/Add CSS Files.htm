Add CSS Files

For every image that you add to RCM you need to add a CSS file to one of the following paths:

  • To the VFS:

<VFS>\app_gui_items\ RCM \<theme>\actions\css

  • Inside the .war file:

<Web root>\app_gui_items\ RCM \<theme>\actions\css

The CSS class name should be in format: .custom-<icon file name>. IIf the file name, for example is: add_new_alert.png, then CSS file should be named add_new_alert.css and its content should be in the following structure:

.custom-add_new_alert{

background: url(‘../icons_picker/add_new_alert.png’) no-repeat;

}
