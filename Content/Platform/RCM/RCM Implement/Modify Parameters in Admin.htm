Modify Parameters in Admin Site

The RCM_md_config_params table can be viewed by selecting the Config Parameters tab within the RCM Admin Site - Customization application.

Two different types of parameters are maintained within the RCM_md_config_params table:

  • Parameters that have default values assigned. It is recommended not to modify these parameters unless a recommendation is made by Technical Support. Changing default parameters can cause performance problems.
  • Parameters that have null values assigned. These parameters can be adjusted in order to customize the CM application functionality. Changing null values will change the way in which the application reacts but will not affect the connection between the RCM application and the AIS Server.

Parameters that include the in the “!” column indicate that when this parameter is edited, the server must be restarted.

To edit a parameter within the RCM_md_config_params table:

  1. Select the parameter that you want to edit within the table by clicking on the radio button beside the parameter name.

  1. Select Edit Parameter from the activity bar for this module.

    The Edit Configuration Parameters window opens with the currently defined value for the selected parameter

  1. Enter the value that you require in the Value field. Note that some of the parameters should only be edited by authorized personnel, according to instructions from support. For a full description of each of the parameters within this table, see Edit Guidelines.
