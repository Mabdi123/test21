Access the Alert Types Options

To access the tools and options for the alert types:

  • Click the Alerts bar and then click the Alert Types tab in the Entities pane.

The icons available within the Entities pane for this task are:

Icon

Description

Displays four tabs on the right side of the screen that enable you to define a new alert definition. The Alert Types tabs are:

  • General – Enables you to add a name, description, ID and alias for the new definition.
  • Alert Details – Enables you to indicate which XML file this definition requires and the location of the relevant XSL file.
  • Alert Type Fields – Enables you to define the attributes of this alert type.
  • Assigned Predefined Notes – Enables you to select possible notes for a specific alert type.
  • Steps Workflow – Enables you to select possible steps for a specific alert type or select a workflow for the alert type.
  • Associations and Views – Related to items relations.

Selecting an alert (item) type within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected alert type.

Deletes a selected alert (item) type. You must close any page associated with this alert type before you can delete it.
