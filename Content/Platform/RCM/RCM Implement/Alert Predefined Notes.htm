Alert Predefined Notes

Use this list to define the various types of predefined notes, which can later be associated with specific alert types.

The icons available within the Predefined Notes window are:

Icon

Description

Enables you to add a new predefined note.

Deletes a selected predefined note.

The options that are available within the Predefined Notes page are:

Option

Description

Identifier

Enables you to enter a reference ID for this predefined note.

Description

Predefined note description.

Scope

Enables you to select the alert types that will have this predefined note. Select between the following:

All Alert Types - A predefined note that has this scope will be associated with all current and future alert types.

Selected Alert Types - Add/remove a predefined note that has this scope to specific alert type(s).

Text

Contents of note.
