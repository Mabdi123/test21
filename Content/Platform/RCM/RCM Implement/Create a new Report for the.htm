Create a new Report for the Selected Catalog

Once you have successfully created a catalog, you can define a new report based on this catalog.

To define a new report:

  1. From the JReport Designer main toolbar, click to open the Open Report window.

  1. From the Open Report window, select the template that you want to use for this report and click Create.

The Report Form Creation Wizard opens for the selected template, enabling you to select the User Data Source for this report.

  1. Select the appropriate UserDataSource in the left pane and use the arrow icon to add it to this report. This adds the required query to the report.
  1. Click the Display tab to bring this page forward, and choose the columns that you want to display in this report to the end user (as shown in the example below).

  1. Click Finish to create this report.

A cls file is generated for this report. You can now use any of the JReport options to develop your report further (e.g. add bitmaps, etc.).

  1. Click in the JReport Designer toolbar to save your report.
