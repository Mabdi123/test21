v_acm_alert_types2

Description: This view describes Work Item Types of Category Alerts, Ad-hoc Blotters and DART. It filters all other Item Type categories and is used for backward compatibility only. Use v_acm_item_type for a list of all Item Types.

Field Name

Description

alert_type_join_id

The internal Join ID to the Alert Type.

alert_type_identifier

The identifier of the Alert Type.

short_name

The short name of the Alert Type.

Name

The name of the Alert Type.

view_join_id

The Item Type View Join ID (used for Entities)
