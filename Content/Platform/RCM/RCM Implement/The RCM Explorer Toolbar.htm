The RCM Explorer Toolbar

The toolbar has the following icons and options.

Icon

Description

Enables you to go up one level in the Virtual File Server

Enables you to create a new folder

Open the selected file for viewing (read only)

Enables you to modify the name of the selected file

Enables you to show/hide the folders structure

Cut, Copy and Paste icons.

Deletes the selected file

Enables you switch between File mode and Zip mode. In zip mode, the files are uploaded and download in the zip format

Enables you to download the selected files.

Enables you to upload files to the current path in the Virtual File System.

Click on the arrow next to this icon in order to select the type of view for the files.

Refreshes the screen view.
