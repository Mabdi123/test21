Create a New Alert View and Assigning General Information

You can create views in RCM as follows.

To create a new Alert View:

  1. From the Alert Views tab, click the icon.
  1. The screen on the right side changes to include four tabs: General, Fields, Default Sort, and Filter tabs.
  1. Using the options within the General tab, provide a name for this view.
  1. Enter the Alert View Identifier that you want to appear within the RCM. The Alert View Identifier is a string that represents the Alert View within the alert-view table of the RCM database.
  1. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field box.
  1. Select the checkbox to enable selecting this view in the User Roles Settings of the RCM.
  1. You are provided with the option to group views so that they will appear next to each other in the Item Views menu of the RCM UI. To create a new group for this View enter the name of the group and then press the enter key.

To add the view to an existing group

  1. Click Refresh.
  1. Click the arrow next to the group field and select the desired group from the Group field (The field does not display any groups when no groups have been previously defined).
