Maximum Nesting Level

When exporting nested widgets to a PDF, the actimize.widget.maximumNestingLevelForPdf parameter, set in admin.jsp, determines the maximum level of nesting allowed in a PDF. Its default value is 0, that is, no printing of nested widgets. The value must be set manually to enable exporting of nested widgets to the PDF.

If the results contain more rows or more levels of nesting than are allowed by the settings, this is indicated in the PDF.
