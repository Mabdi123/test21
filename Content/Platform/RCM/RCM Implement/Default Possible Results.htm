Default Possible Results

  • “ServerUp” – Both the RCM and the RCM repository are up and running.
  • “Error connecting to the database” – RCM is up and running, but RCM repository is either down or not connected properly.
  • Default HTTP error of console (e.g., “There might be a typing error in the address”) – Both the RCM and RCM repository are not up and running or not connected properly.
