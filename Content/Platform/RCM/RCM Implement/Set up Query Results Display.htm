Set up Query Results Display

You can set up the way that you want the results of the query to be displayed using the Result Display page.

To set up the display of the query results:

  1. Provide a title for the page that presents the results.
  2. Indicate the size of the page (default or custom size).
  3. Indicate if you want to allow sorting by checking on the available checkbox.
  4. If the result is a single row, you can choose to present the result in a list format (with each field presented below the previous) by checking the Show as Name/Value list checkbox.
  5. Click within the Column toolbar to add columns to the result display and then select the appropriate width for each column.
  6. Save the query.
