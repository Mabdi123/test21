Set up Alert Views

The RCM Designer enables implementers to set up the Alert (Work Item) Views that are used within the RCM application. Each view is usually composed of Built In Fields and Custom Fields.

You must logout and then login to RCM in order to view the changes that you made in a view.

There are two types of Alert Views:

  • System Views: Views that are readily available after the RCM Designer installation.
  • Defined Views: Defined views that can be added by the implementer.

Setting up defined alert views involves the following tasks:

  • Providing a name, description for this alert view
  • Defining the fields, which will be viewed, and the format of their display.
  • Defining the default sort of the alerts view, and/or defining a filter for it via an SQL query.
  • Saving the newly configured view.
