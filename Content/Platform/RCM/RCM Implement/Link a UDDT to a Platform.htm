Link a UDDT to a Platform List Field

You can link a UDDT that you configured in the User Defined Data Types tab to a platform list field. For details on setting up the UDDT, see Configuring User Defined Data Types.

To link a UDDT to a platform list field:

  1. In the RCM Designer, click the Platform Lists bar and open the relevant Platform List.
  1. Open the Fields tab and go to the relevant platform list field you want to link to a UDDT.

  1. In the Value Type field select UDDT from the drop-down list.
  1. In the Value field, select the UDDT name that you had given in the UDDT creation. See The UDDT General Tab.
  • If you connect a platform list field to a UDDT and you rename the UDDT, then the name in the platform list is also renamed.
  • If you change the platform list identifier, then when you create a new DDQ it is added to the available list.
  • You cannot delete a UDDT that has been linked to a platform list field.
