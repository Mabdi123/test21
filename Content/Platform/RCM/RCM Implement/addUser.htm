addUser

ACMResult addUsers(in Users[] users);

Description - This method accepts an array of User objects and creates them in the RCM repository.

Passing a non-unique user identifier will cause a failure.

IN

  • Users – An array of users.

NOTE:

The actimize.authentication.createUserWithExpiredPassword configuration parameter for users added via this web service is set according to the default defined for the configuration parameter in the Admin Site (admin.jsp). For details, see section Parameter Descriptions Parameter Descriptions page 1.
