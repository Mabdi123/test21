Fields Filtering

This feature enables you to define filters based on the specific built in and custom view fields.

To define a field filter:

  1. Click the Add a field to filter icon.
  1. A view field appears. Click the arrow on the right side of the column to select another field.

    The filter changes according to the data type of the field that was selected. The following field filters are available:

    • String data type: Click the three dots button and then click the Add new string item icon in order to add a string to the list. Repeat this action to add another string to the list. Finally, click Accept.

    • Numeric Data type: Click the three dots button and select from the following options:
    • Select the Exact Value radio button and enter a specific value.
    • Select the Range radio button and then select a comparison operand from the drop-down menu. Enter a value in the field that is next to the selected comparison operand.
  2. In order to define a multiple clause range, click the Click here to add another clause link and select And or Or from the drop-down menu that appears:

  3. Enter the value of the new clause in the same fashion as the initial clause was filled in. When you are done defining all the clauses of the range, click Accept.

For RFI State filter, you must use one of three possible values:

0 – No RFIs

1 – Has RFIs but all messages have been read

2 – Has RFIs but there are unread messages

    • Select the Exact Value radio button and enter a specific value.
    • Select the Range radio button and then select a comparison operand from the drop-down menu.
