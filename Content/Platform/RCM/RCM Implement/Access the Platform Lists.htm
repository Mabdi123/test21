Access the Platform Lists Options

To access the tools and options for defining platform lists:

  • Click the General bar and then click the Platform Lists tab.

The icons available within the Entities pane for this task are as follows:

Icon

Description

Displays the following tabs on the right side of the screen that enable you to define a new platform list:

  • General – Enables you to add a name and description for the new platform list.
  • Fields – Enables you to add and define fields for the platform list.
  • View – Enables you to define the formatting for the platform list.

Selecting a platform list within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected platform list.

Deletes a selected platform list. You must close any page associated with this platform list before you can delete it.

Duplicates the selected platform list.
The copy is named “copy of < platform list >.

Refreshes the platform lists.

Enables you to display the Identifiers of the platform list.
