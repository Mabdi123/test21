Overview of Setting up Work Items

In the Designer, work items are referred to as alerts. When setting up alerts you perform the following tasks:

  • Define alert steps and predefined notes
  • Customize the display title of the built in fields
  • Create custom fields
  • Create value lists
  • Define the score ranges of the alert (work item)
  • Define the order of the alert (work item) views in the alert pop up window

You cannot use two or more Designers simultaneously for designing the same RCM.
