About the Risk Case Manager Designer

The RCM Designer enables implementers to set up and design the following modules:

  • Alert Types
  • Alert Views
  • Blotters
  • Business Unit Hierarchies
  • Case Types
  • Case Views
  • DART Views
  • Dashboards
  • Drilldown Queries
  • Languages
  • Navigation Trees
  • Outgoing forms
  • Policy Manager
  • Report types
  • Tuning Modules
  • Visual Link Analysis
  • Additional settings and features that are described in this guide

Refer to the RCM Installation Guide for information on installing the RCM Designer and connecting to an RCM server.
