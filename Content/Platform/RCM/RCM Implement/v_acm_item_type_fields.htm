v_acm_item_type_fields

Description: Returns all item type fields.

Field Name

Description

item_type_join_id

The Item type internal Join ID

fl_mandatory

An indication whether this is a mandatory field

default_value

Field default value

customized_field_join_id

The field internal Join ID

display_order

The field display order as defined in its item type

e_updatable

The field update mode (i.e. Updatable, Updatable on Creation …)

fl_cacheable

An indication whether this field will be included in cache data of parent xml

fl_display_field

An indication whether this field will be used in the relations box control
