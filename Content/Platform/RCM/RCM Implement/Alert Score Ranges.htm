Alert Score Ranges

The Score Ranges page enables you to define the ranges for the three available scores that reflect the severity of alerts.

You cannot add or remove score ranges but you can modify their ranges.

The options that are available within the Score Ranges page are:

Option

Description

Image File Name

A view only field that displays the file name of the image for the relevant score.

From

The initiating number for score range. This field can be modified.

To

The ending number for score range. This field can be modified.

Verify that ranges do not overlap and that there no gaps between the score ranges.
