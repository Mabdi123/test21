Maintain Predefined Notes within the RCM Designer

All notes that are available for assignment to a Case Type are written to the Notes Table within the RCM database. Notes within this table may be:

  • Common Notes – These are automatically assigned to all Case Types.
  • Available Notes – These are notes which can, optionally, be assigned to any Case Type using the RCM Designer Case Types à Assigned Predefined Notes page.

These notes are originally defined via the Predefined Notes page in the Case Configuration tab. For more information, see Predefined Notes.
