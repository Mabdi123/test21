Cases Views Sorting

This list enables you to define the order of the Case Views as they appear in the Cases menu.

The icons available within the Custom Fields window are as follows:

Icon

Description

Enables you to move down a selected Case View.

Enables you to move up a selected Case View.

To change a position of a Case View:

  1. Select the Case View you want to move.
  2. Click the or icons to move the Case View.
