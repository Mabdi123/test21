Filter

Define the contents of this CWI by selecting a setting in one or more of the filters provided in this page.

The following filters can be set via a drop-down menu:

  • State – Filter by alert state: Closed, Open or Any (All options are selected).
  • Finding – Filter by alert finding: Issue, Non issue, No determination or Any (All options are selected)
  • Step – Filter by step: In Process, Ready, and so on.
  • Aging – Filter the alerts by their age in days.
  • Limit Score Range – Select the score range for the CWIs.

You must click Accept in each filter in order to confirm your settings.
