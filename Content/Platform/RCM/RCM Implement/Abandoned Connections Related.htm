Abandoned Connections Related Parameters

Parameter name

Default value

Description

actimize.pool.substituteAbandonedConnections

1

Possible values: 0 (False), 1 (True)
Relevant only for systems where the number of connections to the repository DB server is limited.
When this parameter is set to 1 (=True), the system kills connections once they exceed the timeout.
When the value is true DB connections that failed to close can be recovered.

actimize.pool.abandonedConnectionTimeout

300

Timeout in seconds for a connection that is considered as an ‘abandoned connection’
