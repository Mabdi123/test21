Configuring Consolidation Criteria

This is a group of conditions that indicate to RCM whether to create a new CWI, or consolidate the TWI into an existing CWI and enrich the sorting process of specific CWI items.

For example, if there are 20 CWIs, this tab enables you to define the initial CWI that is consolidated when a TWI matches more than one CWI for the same relationship.
