Create the Tree Structure

You can create the tree structure as follows.

To create a tree structure:

  1. Click on the icon to create folder(s) and then enter name(s) for the folder(s).
  2. Move specific folders up or down via the and icons.
  3. Move specific folders left or right via the and icons.
  4. Drag Setting(s) and/or Platform Lists from the Available Setting window to a specific folder. For example, a Rating Setting is dragged to the Risk Category folder. You can also drag predefined conditional setting in the same fashion. For information on creating conditional settings, see Define Conditional Settings. For information on creating Platform Lists, see Setting up Platform Lists. For information on defining Settings, see Define Settings.

  1. To change a Setting\Platform List that is already in the Setting Navigation Tree, click the arrow next to the Setting\Platform List and select another Setting\Platform List from the drop-down menu.

  1. You have the option to open the related Platform List by clicking the icon that appears when a Platform List is selected.
