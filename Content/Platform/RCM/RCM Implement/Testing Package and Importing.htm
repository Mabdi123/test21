Testing Package and Importing

You can test your package as follows.

  1. Click the Test button to find out the result of importing the package. A Test Summary screen appears with a heading that indicates whether the test was successful or not and a log of all of the actions that were made. To save a log file of the test, click Save Log File….
  1. Finally, click the Import button to actually import the objects into the RCM. The summary screen of the Import wizard appears with a message indicating that the import procedure has completed successfully and log of all of the actions that were made is displayed. To save a log file of the import procedure, click Save Log File…. Select the Refresh Design Entities Pane to update the objects lists in the Design entities pane that is on the left side of the main screen.

  1. Select the Partial Import option and click Next, the Import File Location screen appears.
