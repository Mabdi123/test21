Edit Guidelines

The $(<parameter>) syntax indicates that the value of the parameter is dependent on another parameter that is in the parentheses. For example, the value $(actimize.workingdirectory) indicates that there is a dependency on the value of the parameter actimize.workingdirectory

Values that are taken from the command line of the Java console (-D<name>=<value> syntax) are entered in the following syntax:

system:<command>

For example, the value ${system:user.home} indicates that the value of the path retrieved from the from value in the system.

If no value is provided for a system value then the default setting is retrieved from the database. Otherwise, value is not resolved.
