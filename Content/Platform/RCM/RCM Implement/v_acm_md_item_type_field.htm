v_acm_md_item_type_fields

Description: This view describes the Item Type fields

Field Name

Description

item_type_join_id

The Item Type Join ID

fl_mandatory

A flag describing that this field is mandatory when creating an Item

default_value

Default value that is added when the Item is created

customized_field_join_id

The Customized field Join ID

display_order

The Field Display order (used in the designer)

e_updatable

Possible Options: Updatable, Not_Updatable, Updatable_on_Creation, Hide_on_Edit, Hidden

fl_cacheable

If set to true then this field will be stored when creating a cached XML of this Item

fl_display_field

If set to true then this field stores the display name of the Item
