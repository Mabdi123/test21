Use the Workflow Asynch Execution Tool

The Workflow Asynch Execution Tool is a utility that runs the workflow escalation process (All RCM utilities and their respective readme file can be found in the \Utilities\bin folder).

The process performs the following actions:

  • Sends an email to the owner on each work item that has an email date that has passed, Note that the work item must be assigned to a user that has an email address
  • Changes the step of a work item whose deadline has passed and was defined to move to another step after the deadline.

Before running the utility, verify that the following jars are in your class path:

  • jaxrpc.jar
  • axis.jar
  • path to your wsclient (web service jar) this can be done by editing the asynch_execution.bat (asynch_execution.sh for Unix) file
