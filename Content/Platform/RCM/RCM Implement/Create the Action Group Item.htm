Create the Action Group Item Type

In the GUI Items wizard you can create a GUI item based on the Action Group type. The action group is a container of actions which are associated to the group type.

The GUI Item Attributes page for the Action Group item type contains the following fields:

Field

Description

Module

Select the Module that you want this GUI item to be added to (For example, Work Items or Cases.)

Name

Enter a name for the Action Group GUI item. (For example Work Items Export).You can display a translatable value by using ${resource string identifier}. For example: ${alerts.dropDown.workItemsExport}.

Type

The following types of action groups are available.

  • Classic Text only: Displays the action as a link on the left pane. This is the type of action group that also appears in previous versions of RCM and is available in all modules. If this group type is not used then the left actions pane does not appear.

  • Toolbar with Icon Buttons: Displays an icon for the action on the actions toolbar. Available only for the currently supported modules.
  • Toolbar with Icon and Text Buttons: Displays an icon and the text.

(sample you can create)

  • Drop-Down Menu with Text Buttons: Displays a drop-down menu with text buttons.

  • Drop-Down Menu with Icon and Text Buttons: Displays a drop-down list item with text and an image for the action on the actions toolbar.

The Drop-down action group is within the Toolbar action group type. The Toolbar container determines how the drop-down type appears (icon or text). And this drop-down determines how its actions appear (action or text)

Drop-down menu action types require an initial icon for the menu. This is the icon that is displayed when the menu is in a collapsed mode.

Visible

Select to make this item visible in the RCM display. Only users with relevant permissions can view displayed GUI items.
