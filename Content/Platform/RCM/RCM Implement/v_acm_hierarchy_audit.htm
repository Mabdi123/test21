v_acm_hierarchy_audit

Description: An audit (history) DB view that store information about BU hierarchy history.

Field Name

Description

hierarchy_join_id

This field is the join id of the BU hierarchy that was audited.
(v_acm_hierarchies2)

action

The action performed on the BU hierarchy

action_date

The date the action took place

user_join_id

This field is the join id of the user who performed the change in the BU hierarchy details. (v_acm_users2)

action_text

Information regarding the audit. For example user email changed
