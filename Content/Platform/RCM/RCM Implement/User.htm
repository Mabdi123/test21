User

User {

String userIdentifier;  (May be defined externally to RCM)

String loginName;

String password;

String fullName;

String email;

String businessUnitIdentifer;

Attributes[] attributes

}

Description: This object contains information regarding a user.

Fields:

  • User Identifier – A unique identifier of a user (i.e. SSN) defined by the customer. (Required.) Note that the group identifier must be unique within both users and groups.  
  • Login Name – The user login name that is used when trying to login to RCM. (Required.)
  • Password – The password a user is using to login to RCM. (Required.)
  • Full Name – The user full name. (Required.)
  • E-mail – The user’s e-mail address (Not required.)
  • Attributes – An array of user attributes. (Not Required.)
