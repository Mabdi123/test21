View Properties for Field

Selecting a specific View Field provides a number of formatting options. (Below the Owner field was selected).

The following formatting options are available for each field:

Option

Description

Column Width:

Enter the width of the field in units:

Pixels – Enter the width of the column in pixels.

Percentage – Enter the width of the column as a percentage of the scrollable list pane (This is the pane that includes the Alerts List table).

Specification applies only to grids (lists)

Alignment

Select to align field text to the right, center, or left. If field data is in language where the text is read from right to left then select the Right to Left checkbox.

Column Title Text Wrap

and

Column Data Text Rap

Wrap the column title text and the column data text with one of the following options:

Cut – Long text is shortened and this is indicated by an ellipses.

Extend – Column is extended to the size fit the text

Wrap – If necessary, text is displayed in multiple lines. Note that text cannot be broken down into lines if there are no space characters.

Additional Styling (CSS)

Select a specific CSS style for the field. For more information on adding CSS style, see Advanced Field Styles.

Column Display Formatter

For details, see Column Display Formatter
