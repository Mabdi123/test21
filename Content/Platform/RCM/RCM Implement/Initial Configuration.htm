Initial Configuration

In order to configure consolidation:

  1. In the Designer Alerts bar General tab, define a CWI by selecting the Can be a consolidation target checkbox.

    An additional tab Consolidation Criteria appears. See Setting up Alert Types for more information about this tab.

  2. The following optional checkboxes may only be selected when the Can be a consolidation target checkbox has been selected:

    1. Reset Consolidated Work Item step to its configured default step – The work item returns to its default when a new item is consolidated to it.

    2. Un-assign Consolidated Work Item user if its state prior to consolidation is closed – This checkbox may only be selected when the Reset Consolidated Work Item step to its configured default step checkbox has been selected.

The Can be used as related item checkbox should be selected for CWI Definitions.
