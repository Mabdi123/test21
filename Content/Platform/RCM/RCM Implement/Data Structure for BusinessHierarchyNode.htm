Data Structure for BusinessHierarchyNode

BusinessHierarchyNode {

int businessUnitId;

BusinessHierarchyNode parentNode;

List childNodesList;

}

Description: This object contains the definition of a single node in a business hierarchy.

Fields:

  • id – RCM's internal unique identifier for the business unit in this node (Required.)
  • parentNode – Parent node / null for root node (Required.)
  • childNodesList – List of all child nodes. (Required.)
