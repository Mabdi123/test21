v_acm_user_audit

Description: An audit (history) DB view that store information about user history.

Field Name

Description

audited_user_join_id

This field is the join id of the user that was audited.(v_acm_users2)

action

The action performed on the user

action_date

The date the action took place

user_join_id

This field is the join id of the user who performed the change in the user details. (v_acm_users2)

action_text

Information regarding the audit. For example user email changed
