v_acm_deployed_rules

Description: Returns the deployed Policy rules.

Field Name

Description

deployed_rule_join_id

The internal Join ID to the deployed Policy Rule.

deployed_rule_set_join_id

The internal Join ID to the rule set

rule_join_id

The internal Join ID to the Policy Rule.

rule_version_join_id

The internal Join ID to the Rule Version.

rule_type

REGULAR or EXCEPTION (=priority rule)

rule_priority

Order within priority rules

conditions_html

The HTML of the rule conditions
