v_acm_bu_attribute_values

Description: This view lists the entire Business unit Attribute Values. Each Business Unit Attribute has only one value. Therefore, it may have one of the following values: Numeric, String, Date, Boolean or Integer.

Field Name

Description

bu_identifier

The business unit identifier

bu_attr_identifier

The Identifier of the attribute

numeric_value

Numeric value of the attribute value

string_value

String value of the attribute value

date_value

Date value of the attribute value

bool_value

Boolean value of the attribute value

integer_value

Integer value of the attribute value
