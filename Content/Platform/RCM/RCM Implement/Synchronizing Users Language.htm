Synchronizing Users Language Settings with LDAP

RCM provides the ability to import user settings from LDAP sources and to synchronize them with RCM data. It is possible to import this attribute as part of the user’s RCM attributes when the user’s representation in LDAP contains the preferred language, In order to import this setting:

  1. The key that is used by the LDAP for the language attribute should be inserted to the ‘actimize_ldap.properties’ file that is used for LDAP synchronization.
  1. The value of the identifier for the language as it appears in the RCM Designer should match the value of the language in the LDAP. When necessary it is possible to edit the RCM language identifier value in the RCM Designer in order to enforce the match.

The key name for mapping a language setting in the actimize.ldap.properties file is ‘language’.
