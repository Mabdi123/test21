v_acm_tasks

Description: This view describes the tasks executed by the scheduled query process.

Field Name

Description

task_join_id

The task’s join ID that is used to join it to other views

process_join_id

The process join ID used in v_acm_processes

start_date

The task’s start date and time

end_date

The task’s end date and time

task_status

The task's step

scheduled_query_join_id

The Scheduled Query Join ID used to join it to v_acm_scheduled_queries

query_sql

The actual SQL that runs in the server
