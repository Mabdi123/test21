PDF Parameters

All PDF-related parameters defined on a nested widget via the DART widget renderer reside under the pdfParams XML node and are relevant only to that specific widget. These parameters are:

maxRows – For each widget, when exporting record widgets to the PDF, this parameter determines the maximum number of rows to export to the PDF. This parameter is set via the renderer info XML, allowing separate configuration per DART widget renderer. The actimize.widget.maximumRecordsForPrint parameter in admin.jsp will be used if this parameter is not defined. If the value is supplied, the lower of the two determines the result.

If the results contain more rows than are allowed by the settings, this is indicated in the PDF immediately after the grid that is truncated.

  • includeInPdf – For each nested widget, this parameter enables or disables PDF printing of the widget.
  • When set to True (or omitted), the widget is included in the PDF.
  • When set to False, the widget is not included in the PDF.
