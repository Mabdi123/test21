The Translation Tab

The Translation tab lists the RCM identifiers and their values.

By default, the Languages Table includes the following columns:

Column

Description

Indicates that the row has been modified

Module

The specific module name

Identifier

The identifier is the internal Id of the Base Value and is part of the metadata. It cannot be modified.

Base Value

The original US - English string for the Identifier

Description

An optional description of the base value.

Last Update Date

Indicate that last time the row has been updated
