Alert Steps List

Use this list to define the various types of alert steps, which can later be associated with a specific alert type.

The icons available within the Alert Steps window are as follows:

Icon

Description

Enables you to add a new alert step.

Deletes a selected alert step.

The options that are available within the Alert Steps page are:

Option

Description

Name

Alert step name.

Identifier

Enables you to enter a reference ID for this alert step.

Description

Alert step description.

State

Enables you to select Open or Closed for alert step state.

Finding

Enables you to select between the following:

Issue

Non Issue

No Determination

Scope

Enables you to indicate whether this alert step will be available for:

All Alert Types - An alert step that has this scope will be associated with all current and future alert types.

Selected Alert Types - Add/remove an alert step that has this scope to specific alert types(s).
