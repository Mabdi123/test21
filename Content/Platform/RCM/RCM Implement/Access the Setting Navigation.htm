Access the Setting Navigation Trees Options

To access the tools and options for defining Setting Navigation Trees:

  • Click on the General bar and then click on the Setting Navigation Trees tab in the Entities pane.

The icons available within the Entities pane for this task are as follows:

Icon

Description

Displays the following tabs on the right side of the screen that enable you to define a new Setting Navigation Tree:

  • General – Enables you to add a name and description for the new Setting Navigation Tree.
  • Tree Structure – Enables you to create the Setting Navigation Tree.

Selecting a Setting Navigation Tree within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected Setting Navigation Tree.

Deletes a selected Setting Navigation Tree. You must close any page associated with this Setting Navigation Tree before you can delete it.

Duplicates the selected Setting Navigation Tree. The copy is named “copy of < Setting Navigation Tree name>.

Refreshes the Setting Navigation Tree list.

Enables you to display the Identifier of the Setting Navigation Tree.
