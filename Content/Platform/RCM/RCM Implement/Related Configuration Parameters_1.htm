Related Configuration Parameters

The following parameters are related to the configuration of the tuning process:

  • actimize.tuning.execution.jobInterval: The interval in minutes when RCM checks with AIS servers whether or not any Execution Plan has finished executing.  
  • actimize.tuning.executionCheck.enabled: Defines if RCM should automatically check with the AIS servers that the Execution Plans have finished executing. By default this setting is enabled.
