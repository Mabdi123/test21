v_acm_item_forms

Description: A view that contains the forms associated with work items.

Field Name

Description

form_join_id

The form join ID that is used to associate it to other views

form_identifier

The identifier of the form

form_name

The name of the form

form_status

The status identifier of the form

reference

The reference of the form

create_date

The date and time when the form was created

last_update_date

The date and time when the form was last updated

form_xml

The form’s XML representation

item_join_id

The item join ID used to associate it to a work item

form_type_join_id

The form type ID

last_update_user_join_id

The last update user ID

create_user_join_id

The create user ID

parent_join_id

The parent form ID

Item_id

The identifier of the work item related to this form

bu_join_id

The business unit of the work item related to this form

owner_join_id

The owner identifier of the work item related to this form

item_type_join_id

The alert type identifier of the work item related to this form

reference_errors

Reference ID of the response received for forms

fl_has_attachments

Indicates whether or not the form has attachments
