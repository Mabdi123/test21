v_acm_rfi_messages

Description: Displays the list of RCM RFI messages

Field Name

Description

rfi_message_join_id

The internal Id of the message

rfi_message_identifier

The unique identifier of the message

create_date

The created date of this RFI.

rfi_join_id

The join id of the RFI.

rfi_message_subject

The subject of the RFI message

rfi_message_from

The ‘from’ field of the RFI message

rfi_message_to

The ‘to’ field of the RFI message

rfi_message_cc

The ‘cc’ field of the RFI message

rfi_message_bcc

The ‘bcc’ field of the RFI message

rfi_message_body

The ‘body’ field of the RFI message

fl_has_attachments

Indicates whether or not the RFI message has attachments

fl_read

Indicates whether or not the RFI message has been read

fl_deleted

Indicates whether or not the RFI message has been logically deleted
