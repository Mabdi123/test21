WorkItemsWidget(widgetId, htmlContainerId, optionalParams)

Creates a Work Items Widget according to the specified arguments, and returns a JavaScript Widget object for managing the Widget:

widgetId: a string representing the ID of the widget on the client side. The ID is used for referencing the client-side Widget object when trying to access its selection or other properties.

htmlContainerId: the ID of the container element that should store the Widget HTML settings (e.g., id of a DIV, SPAN etc.).

optionalParams: a JavaScript object containing zero or more of the following optional parameters (see explanation on the structure of this argument):

  • viewIdentifier: the identifier of the Work Item view that should be used as the basis of the grid display (the view only affects the column layout and not which records are displayed)
  • ddqIdentifier: the identifier of a Drill Down query that should be used for filtering the records (the ddq should return a single list of alert_internal_id values)
  • ddqParams: a standard ddq params string in the following format:
    @@ value 1 @@ value 2 @@ … value n @@

    @@1@@true@@abc@@

    The ddqParams value is ignored if no ddqIdentifier is supplied.

  • actionsGroupIdentifier: the identifier of the GUI items action title that stores the GUI items for this Widget (if none is supplied, no actions combo will appear).
  • pagingSize: a value that controls the number of records per page (default: the value of the config parameter actimize.common.rowsInSegment).
  • selectionMode: a value that controls the behavior of grid selection. The possible case sensitive values are:
  • RCMAPI.GridSelectionModesEnum.MULTI(default): Multiple selection allowed
  • RCMAPI.GridSelectionModesEnum.SINGLE: Single selection allowed
  • RCMAPI.GridSelectionModesEnum.NONE: Record selection is disabled and the selection box is not shown
  • showResetButton: Boolean parameter that controls whether the reset button should be visible to the end-user (default value: true).
  • showRefreshButton: Boolean parameter that controls whether the refresh button should be visible to the end-user (default value: true).
  • showFilter: Boolean parameter that controls whether or not the quick filter section should be visible to the end-user (default value: true).

The entire buttons row is hidden when none of the buttons bar components are displayed (i.e., reset button, refresh button and filter box).

  • stretchGridWidth: Boolean parameter that controls whether or not the widget grid stretches itself. If set to true, the grid occupies the entire container width and the grid columns adjust accordingly (i.e. either shrink or increase in size). Default value is false.
  • autoShrinkHeight: Boolean parameter that controls whether or not the widget grid container automatically adjusts its height according to the widget contents. If set to true, the widget container automatically changes its height after the widget contents has been loaded. Therefore, at the end of this process the container adjusts its height to fit the contents. Default value is false.
