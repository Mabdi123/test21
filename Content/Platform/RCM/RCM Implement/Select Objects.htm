Select Objects

  1. Click the Show Identifiers icon to view/hide the identifiers of the objects that can be imported.
  • To move an object for import, select the object that you want to move and click the icon. The selected object appears at the right pane of the wizard screen.
  • To move all objects for import, click the icon. All the objects appear in the right pane of the wizard screen.
  • To move back an object from the selected objects, select the object and click the icon. To move back all the selected objects click the icon.

  1. Click Next. The Confirmation page appears.
