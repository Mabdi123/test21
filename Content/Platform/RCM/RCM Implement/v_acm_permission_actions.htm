v_acm_permission_actions

Description: A DB View that contains ACM actions

Field Name

Description

action_join_id

The action internal Join ID

permission_model

The module for permissions (i.e. Alerts, Cases…)

action_name

Short descriptive action name (i.e. ‘ADD_NOTE’, ‘VIEW’) 

type

The type of the action (group or single(. Currently group actions are basically ‘All’
