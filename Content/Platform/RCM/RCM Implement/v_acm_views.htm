v_acm_views

Description: RCM views

Field Name

Description

view_join_id

The View Join ID (PK)

view_identifier

The View Identifier

view_name

The View Name

fl_system

Indicates that the view is a System View (Shipped with RCM)

display_order

The View Display Order when presented in a list

Description

The View Description

fl_visibility

Indicates that the View is visible

e_module

Indicates to which Module this View belongs (i.e. Alerts, Cases, etc.)

data_filter_join_id

The Data Filter Join ID

view_group

The View group (used for display)

fl_item_view

Indicates whether or not this view is an Item View
