Assign on the Transition

You are provided with the option to define the assignment behavior of the transition. Select from the following options of assigning the alert once the transition is done

  • Me – The user that is performing the step change.
  • Unassign – The alert becomes unassigned after the transition is made.

NOTE:

  • The automatic transition on touch setting overrides this definition. Meaning if you have selected automatic transition for a step, and one of the “touches” is assign, then the step will automatically change and the assignment will not hold.
  • Recalling or resetting a transition does not reset the assignment of the transition
  • If a transition is done via a plugin call then the alert is assigned to the user performing the step change action.
