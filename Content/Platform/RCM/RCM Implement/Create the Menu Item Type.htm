Create the Menu Item Type

In the GUI Items wizard you can create a GUI item based on the Menu type.

The GUI Item Attributes page for the Menu item type contains the following fields:

Field

Description

Mode

The RCM mode, Main or Administration.

Name

Enter the name that will appear within the new menu of the RCM application.

Create in submenu

Select to create the item in a submenu.

Script

Enables authorized users to enter the script for this new item (i.e. the function name used for performing the action associated with this GUI item).

Once you have entered the required information for the GUI type that you are creating, click Next to continue to the next wizard step.
