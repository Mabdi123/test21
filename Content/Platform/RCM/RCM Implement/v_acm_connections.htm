v_acm_connections

Description: RCM Connections (Connections grid)

Field Name

Description

connection_join_id

The Join ID for the connection

connection_type

Type of connection (Standard / Virtual / Tenant Specifc)

dsn_name

Name of the connection

conn_url

The URL of the connection (this is where the connection is made)

jndi_name

In case the connection is of type JNDI then this is the JNDI data source name

parent_name

If the connection is of type ‘Tenant Specific’ then this is the name of its parent Virtual connection

tenant_name

If the connection is of type ‘Tenant Specific’ then this is the name of the related tenant
