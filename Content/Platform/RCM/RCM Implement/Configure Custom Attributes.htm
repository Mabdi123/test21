Configure Custom Attributes

The custom attributes for RCM users are defined in an XML file that has a specific structure. The location from which this XML is read is defined in the configuration parameter actimize.userAttributes.location. The table in which user attributes are saved is "acm_user_attributes".

After you have configured custom attributes they are then available for editing in the “Update User” screen, and they can be used in the Web Service to add or modify RCM users.

For detailed instructions, refer to the Data Structures for Permissions Management chapter in the RCM Self Developer Guide.
