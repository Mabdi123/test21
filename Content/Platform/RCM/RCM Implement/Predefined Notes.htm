Predefined Notes

Use this list to define the various types of Predefined Notes, which can later be associated with specific Case Type(s).

The icons available within the Predefined Notes window are as follows:

Icon

Description

Enables you to add a new Predefined Note.

Deletes a selected Predefined Note.

The options that are available within the Predefined Notes page are:

Option

Description

Identifier

Enables you to enter a reference ID for this Predefined Note.

Description

Predefined Note description.

Scope

Enables you to select the Cases types that will have this Predefined Note. Select between the following:

All Case Types - A Predefined Note that has this scope will be associated with all current and future case types.

Selected Case Types - Add/remove a Predefined Note that has this scope to specific case types.

Text

Contents of note.
