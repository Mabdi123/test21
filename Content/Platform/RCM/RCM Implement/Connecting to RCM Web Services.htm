Connecting to RCM Web Services

The following code examples show how to connect to RCM web services in java (using Axis) and in C# (using core .Net services). Once you generate stub classes (using your development tool of choice), the basic pattern is the same:

  1. Configure SSL support (optional)
  1. Connect to authentication service
  1. Perform login
  1. Obtain session cookie
  1. Connect to required services maintaining session cookie
  1. Use services
