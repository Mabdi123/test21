Changing the Values of the DDQs in the RCM

The RCM application provides you with the option to change the default values of DDQs used in the Get Next API without using the RCM Designer and while the RCM is in production mode.

In order for a DDQ to appear in the RCM application, you must select the checkbox in the general tab of the DDQ in the RCM Designer.

Go to the Admin tab of the Administration module and expand the Manage Drill Down Queries bar by clicking on its '+' icon. Select a DDQ and click the Edit Drill Down Query action link. A window with the DDQ's default values appears.

Fill in the new default values and click Save.
