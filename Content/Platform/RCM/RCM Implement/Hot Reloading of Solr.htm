Hot Reloading of Solr

If changes are made to solrconfig.xml or schema.xml, solr can hot-reload without restarting the application server( It still serves search requests while it is reloading). This is useful, for example, if you want to re-configure a slave and change its master

Go to
http:/ <host name>:<port>/solr/admin/cores?action=RELOAD&core=rcmCore

For example:

http://localhost:8080/solr/admin/cores?action=RELOAD&core=rcmCore
