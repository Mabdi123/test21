Create RCM Report Display Categories

You can add as many categories for the Report Definitions as you require. These categories appear within the Generate Report window when an RCM user requests to generate an on-demand report. (Only those categories that include reports appear within the Generate Reports window).

You have a set of pre-defined Management reports that are provided for you on your RCM Installation CD, which can be added under the Management category.

To add a category for Report Definitions

  1. Within the Report Definitions toolbar, click .

    The Display Categories window opens. The default category Management Reports automatically appears within the list.

  1. In the Display Categories toolbar, click to add a new row within this window.
  2. Enter the desired name for the category and a description, if appropriate.
  3. Use the Type drop-down option to indicate whether the reports that belong to this category are Normal reports (i.e., reports that are presented only within the Reports Module of the RCM).
  4. Click OK to save the categories.

You can choose, at any time, to view the reports according to these categories or to view all reports in a flat list, without categorization.

To toggle between displaying reports according to categories and displaying a flat view of reports:

  • Within the Report Definitions Design Entities pane toolbar, click .

When the icon appears pushed in, the view shows the categories. When it appears released you can see all reports without reference to their categories.

This option does not affect the way that reports appear within the RCM Generate Reports window.
