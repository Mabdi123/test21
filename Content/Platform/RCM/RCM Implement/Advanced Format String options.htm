Advanced Format String options of Link Formatter

Each link is composed of: <a href=[HREF][STYLE][TARGET]>[DISPLAY]</a>

All parts but the [STYLE] part can be defined in a Format String.

Set URL Style

On field properties, set “Additional Styling (css)” field, to contain the style you want for the URL.

Set Renderer Format string

All other URL parts: HREF, TARGET and DISPLAY can be set in the “Format String” field by using the following format (in any order):

<href>…</href><display>…<display><target>…</target>

<display> and <target> parts are optional. If you did not enter these parts then the URL will contain their default value.

<display> default value: field value

<target> default value: _self

<href> and <display> parts can contain:

  • Hard coded data
  • Any field value by identifier (the term “field” refers to the items defined in the Fields tab of the View)
  • Current field value
Hard Coded Format

Hard coded format can contain any string that is permissible in that URL portion:

<target> values: <window name>, _blank, _parent, _top

<display> values: any value

<href> values: any value that can be in the URL HREF part

Fetching field value

In order to fetch the field value, use the following format:

<fetch>[field identifier]</fetch>

OR

<fetch><value/></fetch>

For example:

<fetch>owner</fetch> returns the alert owner full name.

<fetch>Open Date</fetch> returns the alert open date time.

<fetch><value/></fetch> returns the current field value

Escaping Field Value

If you want to escape the field value, use the following format:

<fetch escape=’js’>[field identifier] OR <value/></fetch>

<fetch escape=’url’>[field identifier] OR <value/></fetch>

<fetch escape=’js-url’>[field identifier] OR <value/></fetch>

Using escape=’js-url’ escape the field value twice. First by URL and then by JavaScript.

Examples:

Format:

<href>javascript:location.href='http://<fetch escape="js-url">str1</fetch>';</href>

Fields value:

str1 = www.google.com

current field: myLink

Result:

<A href=javascript:location.href='http://www.google.com'>myLink</A>

 

Format:

<href>http://<fetch>customField1</fetch></href><display><fetch><value/></fetch></display><target>_blank</target>

Fields value:

customField1= www.google.com

current field: myLink

Result:

<A href='http://www.google.com’ target=”_blank”>myLink</A>
