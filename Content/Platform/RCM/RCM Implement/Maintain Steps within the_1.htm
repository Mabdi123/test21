Maintain Steps within the RCM Designer

All Steps that are available for assignment to a Case Type are written to the Steps Table within the RCM database. Steps within this table may be:

  • Common Steps – These are automatically assigned to all Case Types.
  • Available Steps – These are steps which can, optionally, be assigned to any Case Type using the RCM Designer Case Types à Assigned Predefined Steps page.

These notes are originally defined in the Case Steps page in the Case Configuration tab. For more information, see Predefined Notes.
