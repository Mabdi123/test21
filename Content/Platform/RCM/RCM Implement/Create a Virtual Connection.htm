Create a Virtual Connection

To create a virtual connection, you should initially verify that the RCM is in Multi-Tenant mode. You can verify this by checking that the value for the parameter actimize.isHostedPlatform is Yes (1). After you have verified that the RCM is Multi-Tenant mode, do the following:

  1. Click the New Connection link and select the virtual connection checkbox in the window that opens.
  2. Enter a name for the virtual connection and save your settings. All other fields in the window are greyed out and cannot be filled in.
