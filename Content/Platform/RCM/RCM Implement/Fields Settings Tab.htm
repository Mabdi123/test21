Fields Settings Tab

You can select a platform field and designate relevant fields for display in the Settings Navigator in RCM using the Fields Settings tab. Each field can be defined as read only or updatable and with a default value.

Select a list and click to open the Properties and display the Fields Settings tab.

Click to add a field.

You can configure the field as follows:

  • Updatable – Select to enable editing for the field.
  • True: Enable editing
  • False: Field is read only and cannot be updated. (default)
  • Default Value – Enter a default value for the field. Date fields cannot be assigned a default value.

If you are setting a default value for a BU picker, UDDT or DDQ, use the relevant identifier. For BU picker and UDDT values, the Display Name appears in the RCM UI. For DDQ values, the identifier is displayed in RCM.

  • Field values must be given according to field type. For example an integer field cannot be given a string value.
  • If you add fields to a platform list you must add the field to the View tab View Fields in order for them to appear in the Tree Structure. Then the new fields are added automatically to the fields available for selection in the Fields Settings tab.
  • A field that has a configured value in Fields Settings (in use in the navigation tree) cannot be deleted in the platform list.
  • If a field is a key type, it is recommended to be set as updateable True. (if there is only one key in the list).
