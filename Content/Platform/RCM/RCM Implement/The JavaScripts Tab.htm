The JavaScripts Tab

The JavaScripts tab enables you to define and correlate parameters the RCM result values for the platform list fields. Select a UDDT from the User Defined Data Types tab in the General bar. Click the JavaScripts tab.

You can edit the JavaScript parameters as follows:

Processing Parameters

Enter a function that returns a string with the parameter list for the DDQ query.

The parameters in this script refer to all of the parameters that are defined in the DDQ object. For example, if the DDQ contains three parameters then the script is mandatory and must refer to all three parameters.

Parameters can be defined for both UDDT types: drop-down list and picker fields.

Callback Parameters

Enter a function that assigns the DDQ result values to the platform list fields. The values are passed to the function from the DDQ via the “arr” argument (of type array), where arr[0], arr[1], arr[2]… represents the DDQ result values.

You can view the parameters in RCM where the relevant values are inserted based on the selection. In this example the configuration in the callback determines that when you select a Country, it will map the CityName and DistrictName to the relevant fields.
