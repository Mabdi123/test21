The Base Language Page

The base language of RCM is English (U.S.). The page is composed of the following tabs

  • A read only General tab that includes the name, identifier, description, language code and country code of English (U.S.).
  • A Translation tab that provides tools for adding/removing and exporting/importing resource strings.

Import/export of resource strings is designed to be used only in the base language in order to transfer resource strings from one RCM to another.
