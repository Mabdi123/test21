v_acm_blotters

Description: DB View that stores information and statistics about Trade Blotters.

Field Name Description

blotter_id

The Blotter Identifier

blotter_join_id

The Blotter join ID used when joining other blotter views

blotter_type_join_id

The Blotter type join ID used to connecting to the blotter type view

blotter_date

Business date

state

The Blotter State. This may be Inactive, Active or Materialized

workflow_step

The step of this Blotter

signoff_date

The Signoff date and time

signoff_user_join_id

The user join id of the User who signed off the Blotter

materialized_date

The date and time when the Blotter was materialized

bu_join_id

The Blotter Business Unit join ID

original_bu_join_id

The Blotter Original Business Unit join ID

owner_join_id

The Blotter Owner join ID

trades_count,

The number of Trades associated with this Blotter

reviewed_trades_count

The number of Trades that are already reviewed.

alerted_trades_count

The number of transactions in the blotter that are related to alerts

trades_alerts_count

The distinct number of Alerts associated with all the Blotter’s Trades

trades_notes_count

The number of notes associated with all the Blotter’s Trade

notes_count

The number of the Blotter Notes

description

The description of this Blotter
