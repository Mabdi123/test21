v_acm_wf_tasks

Field Name

Description

task_join_id

Unique id of the workflow task

display_order

The order in which this task is displayed with other tasks

task_text

The name of the task

task_description

The description of the task

task_link_url

The action to run when clicking on task form workflow window

fl_mandatory

Whether or not the task is mandatory

wf_step_join_id

Workflow step associated with this task

audit_join_id

Audit associated with this task

fl_checked

Whether or not the task was checked (done)
