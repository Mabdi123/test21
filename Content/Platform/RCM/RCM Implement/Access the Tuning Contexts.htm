Access the Tuning Contexts Options

To access the tools and options for defining Tuning Contexts:

  • Click the Tuning bar and then click the Tuning Context tab in the Entities pane.

The icons available within the Design Entities pane for this task are as follows:

Icon

Description

Displays the following three tabs on the right side of the screen that enable you to define a new Tuning Context:

  • General – Enables you to add a name, description and to map setting navigation trees..
  • Execution – Enables you to select the AIS connection, Execution Plan and optional arguments.
  • VTL – Enables you to enter the VTL code.

Selecting a Tuning Context within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected Tuning Context.

Deletes a selected Tuning Context. You must close any page associated with this Tuning Context before you can delete it.

Duplicates the selected Tuning Context. The copy is named “copy of <Tuning Context name>.

Refreshes the Tuning Context list.

Toggles the display of the identifier.
