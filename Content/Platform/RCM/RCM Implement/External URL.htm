External URL

When External URL is selected as the Report Provider, reports are provided by means of an External URL Web Service (see the Web Service for External URL in Appendix C for more information).

Report Types that use an External URL provider must be assigned a policy that is defined as follows:

  • Materialization Policy must always be set to never
  • Archiving must always be set to never
  • Purge policy must always be set to never.
  • Notes and Sign off can be set to any of the existing options.

When the report provider is an external URL, the Materialization and Archive formats are not required. Also Report Provider Data and Report Parameters are not required for this definition.

If reports are uploaded from an external URL, it is necessary to specify:

  • Report Name
  • Report Identifier (this can be the same as the name)
  • Display Category
  • An appropriate policy that is defined for reports provided from external URL.

Note that the policy for external URL reports refers only to Notes. An external URL report is never materialized, archived or purged.
