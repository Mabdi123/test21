v_acm_tenants

Description: This view describes the Tenants in RCM.

Field Name

Description

tenant_join_id

The Tenant’s join ID that is used to join it to other views

tenant_identifier

The identifier of the Tenant

tenant_name

The name of the Tenant

description

The description of the Tenant

default_hierarchy

The join ID of the Tenant’s default hierarchy

fl_active

Flag that indicates whether the Tenant is active or not

fl_deleted

Flag that indicates whether the Tenant is deleted or not

tenant_prefix

The prefix of the Tenant
