Language Hierarchy

Defining the required language is made in several levels in RCM. The levels from the lowest to the highest level are:

Level

Mandatory

Calculation

Application level

yes

Can be set from the Designer or via the configuration parameter actimize.language.currentACMLanguage

Server level

no

Set in acm.ini configuration parameter actimize.language.currentACMLanguage

Group level

no

Set in the Group settings of RCM Admin tab.

User level

no

Set in the User settings of RCM Admin tab.

The determination of the language that is displayed to each user is according to the highest level in this hierarchy.

The login page for all users is displayed in the server’s default language. The user views pages in the requested language after logging in.
