User Defined Data Types (UDDT) Overview

UDDTs enable you to create a predefined picker or drop-down list. Fields that have been linked to UDDTs only display the values that are invoked from the UDDT configuration.

  • UDDTs of type list that are created via Designer can only be used in platform lists.
  • UDDTs must only use simple and not complex queries.

In the Designer, click the General bar and click the User Defined Data Types tab.
