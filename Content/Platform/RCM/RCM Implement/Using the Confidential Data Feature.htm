Using the Confidential Data Feature

Common users that do not have this permission cannot view attachments and notes that have been defined as confidential.

To add the confidential data feature do the following:

  1. Go to the Config Parameters tab of admin.jsp and activate the parameter actimize.isConfidentialData by selecting a value of Yes.

  1. Click the Administration link from the main view of the RCM UI.

  1. In the Admin tab select Manage Roles.
  1. Select the relevant role and click the Update Role link.
  1. Select the Confidential Data bar and add the relevant data types (e.g., alerts).

  1. Users with this role can now select a note or an attachment as a confidential level item.

  1. Rows with the confidentiality level are displayed with the icon.

The confidential notes/attachments rows are not displayed to common users.
