The UDDT General Tab

The General tab enables you to create or edit a UDDT for use in platform lists. Select a UDDT from the User Defined Data Types tab in the General bar. The General tab opens.

The General tab contains the following:

Parameter

Description

Name

The name of the UDDT.

Identifier

The identifier of the UDDT.

Description

Description of the UDDT.

Edit

Enables you to edit the description.

UDDT Type

Select the type of UDDT: Drop-Down List or Picker.

Drill Down Query

Select the DDQ you want for the values of the drop-down list or picker.

Click to open the selected DDQ.

Refresh

Click to refresh the data.
