Examples

The following examples describe various scenarios where the help default is used, as well as values with and without template placeholders.

Note that in the examples, RCM’s context path is http://myrcm:8080/rcm5.13:

Help Page

Config Parameter Value

Resulting Help Link

Admin

:help/[page].htm

http://myrcm:8080/rcm5.13/help/Admin.htm

In this example, the default template is used and the help link opens to the default hard coded page (Admin).

Work Items

http://othersite:8082/help?q=[page]

http://othersite:8082/help?q=Work_Items_List

In this example, [page] is used as a URL parameter. The placeholder gives page information to go to a version of a work items help page.

DART

http://othersite:8082/static-help.htm

http://server2:8082/static-help.htm

In this example, no placeholders are used, and therefore the help link opens to a page that was set without context.
