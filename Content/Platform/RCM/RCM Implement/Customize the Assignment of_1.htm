Customize the Assignment of Notes Using the RCM Designer

You can add any Available Note to the list of Selected Notes for the Case Type that you are defining.

All Common Notes appear within the RCM Designer Case Types àAssigned Predefined Notes in the drop-down list of notes that are available to the RCM Web Application end users.
