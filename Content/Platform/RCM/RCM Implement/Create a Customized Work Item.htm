Create a Customized Work Item

You can create a customized work item as follows.

To create a customized item:

  1. Select the actimize.alerts.workItemsCategories parameter from the Config Parameters tab of the RCM Admin Site (admin.jsp).

  1. Select Edit Parameter from the activity bar for this module.

    The Edit window opens with the currently defined value for this parameter.

  1. Enter a list of comma separated work items category names in the Value field. For example: Documents, Issues

  1. Select a work item in the Category field that is in the Alert Type General Tab. For more information on Alert Types, see Setting up Alert Types.
