v_acm_forms

Description: A view that contains the forms.

Field Name

Description

form_join_id

The form join ID that is used to associate it to other views

form_identifier

The identifier of the form

form_name

The name of the form

form_status

The Status of the form

reference

The reference of the form

create_date

The date and time when the form was created

last_update_date

The date and time when the form was last updated

form_xml

The form’s XML representation

case_join_id

The case join ID used to associate it to v_acm_cases

form_type_join_id

The form type join ID used to associate it to v_acm_form_types

last_update_user_join_id

The last update user join ID used to associate it to v_acm_users2

create_user_join_id

The create user join ID used to associate it to v_acm_users2

parent_join_id

The parent form join id used to associate it to v_acm_forms

case_identifier

The case identifier associated to v_acm_cases

bu_join_id

The business unit join ID associated to v_acm_business_units2

owner_join_id

The join id associated to the owner of the case associated with this form

case_type_join_id

The join id of the case type of the case associated with this form
