BU and Calendar Associations

The following illustrates the connection between business units and calendars and how the system selects the relevant calendar to use for a specific BU.

The default hierarchy is always selected if a BU is in two hierarchies.

Hierarchy 1 (Default Hierarchy)

  • BU1 (Calendar 1)
  • BU2 (Calendar 2)
  • BU3
  • BU4
  • BU6

Hierarchy 2

  • BU5 (Calendar 3)
  • BU4
  • BU8

Hierarchy 3

  • BU9 (Calendar 4)
  • BU8

Calendar Scenarios

The following are example scenarios for associating BUs with the correct calendar name.

Scenario 1:

Work item assigned BU5, and the system selects Calendar 3.

Scenario 2:

Work item assigned BU3. BU3 is found in Hierarchy 1 under the parent BU1. The system selects Calendar 1 because it is under the parent BU with that calendar.

Scenario 3:

Work item assigned BU4. BU4 is found in two hierarchies – 1 and 2. The system selects Calendar 1 because it is located in Hierarchy 1 which is the default hierarchy.

Scenario 4:

Work item assigned BU8. BU8 is found in two hierarchies – 2 and 3. There is a conflict because BU8 is associated with Calendar 3 in Hierarchy 2 and Calendar 4 in Hierarchy 3. The system does not select a calendar because there is no default hierarchy and therefore does not use a configured calendar for calculating deadlines (for BU8 the deadline is calculated using Days configuration).

Deadline Recalculations

In certain cases there are changes that affect calendar calculations for step deadlines. These include BU changes and platform list data changes (adding and removing dates).

Example (using the hierarchies above)

Work item assigned BU5, and the system selects Calendar 3.

The step deadlines for the workflow of this work item are configured to business days based on calendar 3. If between two steps the BU for this item is changed to BU 9, then the deadline for the next step is caluculated according to calendar 4. This recalculation only applies to workflow step deadlines and not global deadlines, which remain as originally configured.
