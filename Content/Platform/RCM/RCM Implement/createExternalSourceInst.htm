createExternalSourceInstance

ACMResult createExternalSourceInstance(in ExternalInstanceData instanceData,in String instanceContent)

Description: This method allows you to define a report instance of type external URL that points to the specified URL.

IN

  • ExternalInstanceData – Accepts the following:
  • ExternalTypeEnum (enter value ExternalURL)
  • InstanceName (Required)
  • BusinessUnitIdentifier (Required)
  • GenerationDate
  • BusinessDate(The date format is: "yyyy-MM-dd HH:mm:ss")
  • BusinessUser
  • GeneratingSystem (e.g. AIS, Designer)
  • ReportDefinitionId (Required. This must be a report type that was defined, within the RCM Designer, for an External URL).
  • InstanceContent – The URL that provides the report.
