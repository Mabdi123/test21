Create a New UDDT

You can create a new UDDT in the User Defined Data Types tab.

To create a UDDT in the Designer:

  1. Select New to create a new UDDT.
  1. In the General tab enter the relevant information for the UDDT.
  1. Link the predefined UDDT to the relevant fields.
    • For platform lists see Link a UDDT to a Platform List Field.
    • For settings see Link a UDDT to Settings.
