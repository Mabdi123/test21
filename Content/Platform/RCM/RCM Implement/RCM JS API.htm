RCM JS API

Exposes the user Date and Time format using RCMAPI JavaScript:

RCMAPI.convertToUserDateTimeFormat(dateValue)

RCMAPI.convertToUserDateTimeFormat(dateValue, sourceDateFormat)

Example:

function testFunc(){

var dateFormat = RCMAPI.convertToUserDateTimeFormat("1978-04-15 01:02:03");

alert("My date: " + dateFormat);

}
