General Configuration - Value Lists

Click the General bar and the Configuration tab to access Value Lists.

The following tab is used to define value lists, which can later be linked with specific custom fields.

The icons available within the Value Lists window are as follows:

Icon

Description

Enables you to add a new value list.

Deletes a selected value list.

The options that are available within the Value Lists page are:

Option

Description

Name

Enables you to enter a name for this value list.

Identifier

Enables you to enter a reference ID for this value list, which later can be identified by an external application.

Description

Provides you the option to enter a description for this value list.

Values

Enter values for the value List. click the icon to open the following window:

Click the icon to add and define a value. Repeat this action to add another value. Click the icon to set the highlighted field as the default value in the list. Click the icon to remove the highlighted field as the default value. Once the list has been finished, click the Accept icon.

The value list can then be linked with a specific custom field.

For example, the Custom Field USA Region is associated with the Region Value List.

The RCM user now has an option to select from a list of values for this custom field when editing/creating a work item (alert) that has this Custom Field.
