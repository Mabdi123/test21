Limitations and Guidelines

In order for a JReport to function properly in the RCM environment, various limitations and guidelines should be considered.

Parameter Data Types

Only parameters of the following data types are supported:

  • String
  • Integer
  • Boolean
  • Date
    Date time parameters must have User Format in parameter properties set to “yyyy-MM-dd”.
  • DateTime
    Date time parameters must have User Format in parameter properties set to “yyyy-MM-dd hh:mm:ss”.

To configure the format do the following:

  1. Right click the date parameter and open the Edit Parameter window.

  1. Enter the format in User Defined Format field
