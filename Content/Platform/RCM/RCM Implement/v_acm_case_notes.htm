v_acm_case_notes

Description: Returns all case notes with the following information:

Field Name

Description
case_join_id

The Join ID to the Case to which this note is attached (v_acm_cases)
note

The Note text
audit_join_id

The Join ID to the related audit record associated with this note (v_acm_case_audit)
user_join_id

The Join ID to the user who added the note to the Case (v_acm_users2)
create_date

The date when the note was added
