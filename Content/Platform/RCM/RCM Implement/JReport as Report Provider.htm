JReport as Report Provider

When JReports is selected as the Report Provider, you must indicate the cls report file and the cat catalog file for the Report.

To select the Report Provider files for a report generated by JReport Designer:

  1. Click the Report Provider Data tab to bring this page forward.

  1. Within the JReport Report File section of this page, select the appropriate cls files as follows:

    If you are not using an alternative layout for the archived file, for JReport report file, select the appropriate cls file that was generated when you created the new report within the JReport Designer. (If you did not select to use alternative materialized archiving layout, this is the only option that appears within this section).

    If you selected to also use an alternative layout for the archived file, you must also select the cls file that you want to use for this alternative layout.

  1. Within the JReport Catalog section of this page, enter the appropriate information and select the cat file as follows:

    • Enter a name for this catalog. This enables you to easily select the catalog again.
      • Use the Browse options beside the Catalog file field to select the cat file that was generated when you created this file within the JReport Designer. Make sure that you choose the cat file that is in the same location as the cls file that you selected for this Report Definition (above).
  1. Click Upload to upload the Report files.
