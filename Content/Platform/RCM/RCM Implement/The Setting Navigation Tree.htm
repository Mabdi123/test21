The Setting Navigation Tree Toolbar

The icons available within in the Navigation Tree toolbar are as follows:

Icon

Description

Opens a new folder for the Setting Navigation Tree.

Adds a Platform List to the Navigation Tree. The first Platform List in the list of Available Platform List is added to the tree.

Adds a Setting to the Navigation Tree. The first Setting in the list of Available Setting is added to the tree.

Removes the selected Setting/Platform List.

Moves up in the Setting Navigation Tree.

Moves down in the Setting Navigation Tree.

Moves the selected Setting/Platform List to the left.

Moves the selected Setting/Platform List to the right.

Cut, copy, and paste icons.

Clears the Settings/Platform Lists that contain erroneous fields.

Refreshes the Navigation Tree.
