v_acm_report_audit2

Description: Returns audit history for all the reports with the following information.

Field Name

Description

report_instance_id

The internal ID for each report instance (v_acm_report_instances2)

report_action

The Report actions

action_date

The date when each action was initiated

user_join_id

The Join ID to the user associated with each action (v_acm_users2)

action_text

Additional information associated with an action.
E.g., information about an assigned user action includes the old and new user names
