Use Sprite Images

Sprite images can be inserted in same path as the other images but it is recommended to insert them in sub folder of actions. For example:

\Web root\app_gui_items\rcm\<theme>\actions\sprite_icons_picker

To insert a sprite image, do the following:

  1. Edit or create a GUI item..
  1. Select any customized image from icons picker. For example, my_add_attachment.png.

  1. Finish the process of editing the GUI Item by applying it.
  1. Add the CSS related code according to the following syntax:

.custom-<selected custom image name>{

                background: url('<path to sprite folder>/<name of sprite image>') <pixel coordinates of specific image in sprite file><First coordinate - left and second- top)>;

}

For example

.custom-my_add_attachment{

                background: url('../sprite_icons_picker/icons.png') 32px 32px;

}
