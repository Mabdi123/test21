RoleAssignment

RoleAssignment{

String roleIdentifier;

String subjectIdentifier;

String businessUnitIdentifier;

AdvancedAssignemntEnum advancedAssigmentEnum;

}

Description: This object contains information regarding a role assignment.

Fields:
  • Role Identifier – An identifier of a role that is defined in RCM. (Required.)
  • Subject Identifier – Can be a User Identifier or a Group Identifier. (Required.)
  • Business Unit Identifier – A Business Unit Identifier that overrides a user’s business unit. This field is required only if AdvancedAssignemntEnum is null. In any other case it is ignored.
  • Advanced assignment Enum
  • <all> – Applies to all business units
  • <user> – Use the user business unit identifier
  • <none> – Do not use any business unit
