Overview of Global Search Customization

The Solr server enables the RCM global search functionality. The global search enables using robust research features that also searches text in Microsoft® Office®, PDF, CSV and text file attachments. It is currently available only for Work Items including indexing of notes, RFI and workflow steps. It is based on an external indexing engine where RCM periodically indexes data that is modified and sends it to the indexing engine. Refer to the RCM Installation Guide for more information on installing and configuring the Solr search server. Refer to the RCM Reference Guide for information on how users can take advantage of this mechanism’s advanced search features.
