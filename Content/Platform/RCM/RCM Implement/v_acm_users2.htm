v_acm_users2

Description: Returns user information

Field Name

Description

user_join_id

The user internal Join ID

user_identifier

The user identifier

login_name

The user login name

full_name

The user full name

Email

The user e-mail

bu_join_id

The user business unit internal Join ID (v_acm_business_units2)

is_active

A flag indicating whether the user is active or not

language_join_id

The join ID of the language of this user

ava_role

Contains the role value: interactor, publisher or null (if the user does not have permissions).

fl_ava_require_sync

Describes if the user failed to sync with AVA. After a failed sync the user needs to sync again and then the value is true. Default: false (no need to sync).
