v_acm_trades

Description: DB View that stores Trade information

Field Name

Description

trade_id

Trade Identifier

trade_join_id

Trade join ID used for connecting other blotters

trade_type_join_id

Trade Type join ID used to connecting to other blotters

blotter_join_id

The Trade’s Blotter join ID 

review_date

The date and time the Trade was reviewed

review_user_join_id

The User join ID that reviewed the Trade

trade_date

The Trade’s business date

alert_count

The number of Alerts that are associated with the Trade

open_alert_acount

The number of open Alerts that are associated to this Trade

note_count

The number of notes associated to this Trade
