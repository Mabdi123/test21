Nested Lists Tab

You can define a nested list under platform lists in the RCM.

In the Platform Lists bar of the Entities pane, select a list and select the Nested Lists tab.

You can add up to five nested lists to be displayed under a platform list. You can also add up to four lists to a nested list (5 total lists including the original nested (child) list.

In nested lists, the key field of the original (parent) list must be included within the key field of the nested (child) list. See Define Fields of the Platform List.

To add a nested list:

  1. Click the Add Nested List icon to add a nested platform list.
  2. Add a Description to identify the list.
  3. Using the arrow icon, you can adjust the display order of the nested lists.
  4. Click the Go icon to open the nested (child) platform list and configure the list fields.
