Overview of the Records Widget API

A records widget in the alert details HTML page is declared using the createWidget XSL API inside the alert details XSL. Respectively, the widget for alert details export to PDF is declared using the exportRecordsWidgetToPDFXSL API, inside the alert details XSL-FO.

The Records Widget API is divided to two sections:

  • An XSL API for declaring a widget.
  • A JavaScript API for showing the widget and manipulating it after it appears.
