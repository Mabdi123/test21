countUsersByCriteria

ACMResult countUsersByCriteria(in String userFilter,String businessUnitIdentifierFilter,out int usersCount)

Description: This method accepts a user login name filter and a business unit filter. Based on this, it queries the database and returns the count (the number of users that match the criteria).

IN

  • userFilter – A user login name. If this field is passed as null there will be no filter.
  • businessUnitIdentifierFilter – Business Unit Identifier. If passed as null there will be no filter.

OUT

  • usersCount – The number of users that match the search criteria.
