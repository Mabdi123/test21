Cases Steps

Use this list to define the various types of Case steps, which can later be associated with a specific Case Type.

The icons available within the Case Steps List window are as follows:

Icon

Description

Enables you to add a new case step.

Deletes a selected case step.

The options that are available within the Case Steps page are:

Option

Description

Name

Case step name.

Identifier

Enables you to enter a reference ID for this case step.

Description

Case step description.

State

Enables you to select Open or Closed for case step state.

Issue:

Enables you to select between the following:

Issue

Non Issue

No Determination

Scope

Enables you to indicate whether this case step will be available for:

All Case Types - A case step that has this scope will be associated with all current and future Case Types.

Selected Case Types - Add/remove a case step that has this scope to specific Case Types(s).
