getUsersByCriteria

ACMResult getUsersByCriteria(in String userFilter,in String businessUnitIdentifierFilter, in int startFrom, in int maxRecords,out  User[] users);

Description: This method accepts a name filter and a business unit filter. Based on this, it generates a list of users and returns a section of them starting from a user located in the startFrom location and counting until the max records.

IN

  • userFilter – A user full name. If this field is passed as null there will be no filter.
  • businessUnitIdentifierFilter – Business Unit Identifier. If passed as null there will be no filter.
  • startFrom – Start from number. A valid number must be entered (e.g., 0).
  • maxRecords – The number of users to retrieve. A valid number must be entered (e.g., 1000).

OUT

  • users – An array of User objects.
