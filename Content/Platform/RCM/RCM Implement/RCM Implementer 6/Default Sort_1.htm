Default Sort

This tab enables you to define the default sort of the fields for this CWI.

The icons available within the Default Sort window are as follows:

Icon

Description

Adds a new view field for sorting.

Deletes the selected view field.

and

Moves the selected view field up or down in the list, changing its viewing order.

To add a new view field:

  1. Click the Add a new Field icon.
  2. The next available field (in alphabetical order) appears. (Common fields are displayed before Built In fields.) If this is not the field that you want to add, click the arrow next to this field to display the drop-down menu and select the required field.

  1. Select the sort order for each field in the Sort Order column. The default sort order for each field is ascending. To switch to descending, click the arrow at the right of this field and select Descending.
