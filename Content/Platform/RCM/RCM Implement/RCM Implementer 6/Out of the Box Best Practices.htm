Out-of-the-Box Best Practices

The currently available out-of-the-box Best Practices include the following:

  • Create Alert Type: Includes all the steps for creating an alert type
  • Create Alert Workflow: Includes all the steps for creating an alert workflow
  • Create Business Unit and Hierarchy: Includes all the steps for creating a Business Unit and its related hierarchy.
  • Create DART View: Includes all the steps for creating a DART view.
  • Define Entity Matching: Includes the steps for configuring the entity matching feature in RCM.
