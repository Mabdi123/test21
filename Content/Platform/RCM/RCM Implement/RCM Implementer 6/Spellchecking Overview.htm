Spellchecking Overview

Spellchecking is available for all text boxes in the RCM. Text boxes include notes, description fields and various form fields.

Click the Spellchecking link to start the spellchecking procedure.

Spellchecking has also been integrated into the Rich Text feature. When the Rich Text feature is enabled, click the button to start the spellchecking procedure.

The configuration parameter “actimize.general.spellCheckEnabled” is used to enable/disable the spell checking feature.

The Sentry Spell Checker is used as the spell checking engine. The name and location of this properties file is set in the RCM configuration parameter “spellcheck.properties.path”. The name and location of the file that stores the license key for this spell checker is set in the configuration parameter “spellcheck.license.path”.

For information about spellchecking APIs and overriding styles, refer to the RCM Self Developer Guide.
