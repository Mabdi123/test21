Advanced Settings

Click Advanced in the New Setting window.The following describes the Advanced Features option.

The Advanced Features window includes the following options:

Option

Description

Subtitle

Defines a category that appears within the Settings Activity Bars (if relevant).

Restore defaults script

Enables you to enter an SQL script that is run from a Restore Default button within the Threshold/List/Reference page of the end user application. When you enter a script in this entry box, the Restore Default button within the end user application is displayed.

Import

Use this option to enable/disable an Import Excel/CSV option, within the the RCM. You can select one of the following:

N/A – The Import Excel/CSV option does not appear for the current setting within the end user application

Merge – The user can select to import data from an xls or csv file and the data will be merged into the current data for this setting.

Replace – The user can select to import data from an xsl or csv file and the data from the selected file will replace the current data for this setting. 

When the Import feature is enabled and the end user clicks the Import link the following opens, enabling the user to select the file from which the data will be imported.
