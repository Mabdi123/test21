Set Up Workflows

The RCM Designer enables implementers to set up the alert workflows that are used within the RCM Application.

Setting up an alert workflow involves the following tasks:

  • Access the Workflow Options
  • Workflow General Tab
  • Create a Workflow
  • The Workflow Toolbar
  • Add Steps to the Workflow
  • Add Deadline Parameters to a Step
  • Create a Checklist for the Step
  • Create a List of Restricted Actions
  • Link the Step Nodes
  • Create the Entire Workflow
  • Modify Workflows of an Alert Type
  • Workflow Barriers
  • Transition Tab
  • Set up Reasons for Transition
  • Use the Workflow Asynch Execution Tool
  • Use the RCM Scheduler for Workflow Deadlines Jobs
