getUsersInBU

ACMResult getUsersInBU(in String buisnessUnitIdentifier,in String hierarchyIdentifier, in int startFrom, in int maxRecords,out  User[] users);

Description: This method returns a list of users that are members of a business unit and returns a section of them, starting from a user located in the startFrom location and counting till the max records. If a hierarchy identifier is passed the method will include users that are members in any of the business unit children. The users are ordered by their full name.

IN

  • buisnessUnitIdentifier – A business unit identifier.
  • hierarchyIdentifier – Hierarchy identifier. If passed, the query will include all users that are members of any of the business unit children within the context of the hierarchy.
  • startFrom – Start from number. If passed as null start from 1.
  • maxRecords – The number of users to retrieve. If passed as -1 returns all the users that match the search.
