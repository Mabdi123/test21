getSubjectGroups

ACMResult getSubjectGroups(in String subjectIdentifier, in boolean isRecursive, out String[] groupIdentifiers);

Description: This method returns all user/group assignments to a group.

IN

  • subjectIdentifier – A User Identifier or a Group Identifier.
  • isRecursive – If this Boolean is false then only the directly associated groups will be returned. If this field is true then a recursive list of groups will be returned.

OUT

  • groupIdentifiers – An array of group identifiers.
