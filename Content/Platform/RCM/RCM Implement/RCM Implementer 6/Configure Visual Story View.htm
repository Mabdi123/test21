Configure Visual Story View

The Visual Story View enables a uniform item display view in RCM. You must first configure the view in the Designer as follows.

Configure Visual Story View as Default

You can enable the Visual Story View for item details according to alert type.

To enable the Visual Story View:

  1. In the Alerts bar, select the Types tab and select an alert type from the list.
  2. In the General tab, select the Details Can be Shown in Visual Story checkbox to enable the Visual Story View tab.

    This allows the alert details to be displayed in the Visual Story View in RCM.

  1. In the Alert Details tab select the Use Visual Story View checkbox as the display default for item details.
