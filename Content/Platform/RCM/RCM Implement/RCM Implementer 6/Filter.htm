Filter

You can define the contents of this view by selecting a setting in one or more of the filters that are provided in the following.

The following filters can be set via a drop-down list:

  • Alert Type

    For case items views, make sure to select the alert types for the Case Item Alerts view that will be displayed in the details Case Work Items tab in RCM. For more information, refer to the RCM Reference Guide.

  • State – Closed or Open
  • Finding – issue, non-issue, and no determination
    • Owner
  • Step – In Process, Ready, etc.

Select one of these settings for each of the above filters:

  • Any – All options are selected. This setting is equivalent to not filtering the parameter.
  • Click the check box of one or more of the available options in the drop-down menu

You must click Accept in each filter in order to confirm your settings.

Additional filters include:

Filter

Description

Business Unit

Select between the current business unit and any business unit (all business units).

Aging

Filter the alerts via their age in days. It refers to the alert date field that the user can pick and not to the create date field.

Archived

Select from one of the following options:

  • Show only archived reports
  • Do not include archived reports
  • Include archived reports

Limit Score Range

Enter a minimum and a maximum score for the alert.
