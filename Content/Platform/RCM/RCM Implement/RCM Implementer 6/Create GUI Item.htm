Create GUI Item

You can create a new GUI item by using the GUI wizard.

To create a new GUI item:

  1. Click the Create GUI Item link within the GUI activity bar to open the GUI Wizard.

  1. The first step in the wizard is to select the GUI type of the object that you want to add to the application. The options are:

Option

Description

Action Group

An action group incorporates a number of actions under the same group.

Action

This is the button that you click to perform an action. An action can belong in an action group. Actions can be displayed or hidden according to permission settings. For example, it is possible to assign the permission to the Analyst role to create a new item. A user without this permission is not able to view this action (the New Item button).

Menu

This is a tab name that appears as one of the tabs at the top of the application window.

The menus that appear in your RCM application depend on the requirements of your organization.

Tool

The tools are a method of displaying toolbar buttons. It is recommended to use the Action type instead.

The selection of the GUI item type affects the fields in the wizard pages that follow.
