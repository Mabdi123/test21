Customizing JReport Icons

The following describes how to customize the icons that are displayed in the toolbar of the JReport Web page, which displays RCM reports.

NOTE:

  • This can only be performed following full RCM installation.
  • You must work in RCM development mode.

To customize JReport icons:

  1. In RCM add ./jreport/admin to the end of the web address.
  2. In the JReport administration, click the Profile menu and select Customize Profile.

  1. Click the Page Report Studio tab and in the Features tab click the New Profile link.

  1. In the Basic tab select the icons you want to display or remove. In the example, the Zoom icon was removed.

  1. Click OK and then click OK in the Page Report Studio tab.
