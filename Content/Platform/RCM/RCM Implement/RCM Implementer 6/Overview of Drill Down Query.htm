Overview of Drill-Down Query Setup

NOTE:

Changing a Drill-Down Query or any parameter associated with a query must be done in co-ordination with definitions within the AIS Model that generates the Query.

A Drill-Down Query is an SQL Query that is used in order to expand the Alert Details. Drill-Down Queries are defined during modeling and are written to the drill_down_queries table within the RCM database. All queries, that are already defined, automatically appear within the RCM Designer Queries module. You can view these queries, and certified AIS implementers can use this tool to facilitate the editing of existing queries.

Defining drill-down queries requires the following tasks:

  • Providing a name and description for this Query
  • Selecting the appropriate connector
  • Entering the Query
  • Adding parameters to the Query
