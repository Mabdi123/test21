Group

Group{

String groupIdentifier;

String groupName;

}

Description: This object contains information regarding a group of users / groups / both.

Fields:

  • Group Identifier – A unique identifier of a group. Defined by the customer. (Required.) Note that the group identifier must be unique within both users and groups.
  • Group Name – A unique name of a group. Must be unique within the context of user login name and group name. (Required.)
