Define the Fields of the Cache XML for the Child Alert Types

Define the fields of the cache XML for the Alert Types in the Alert Type Custom Fields tab. These are the fields that are embedded in the XML of the parent alert type. One of the fields must be the alert identifier. You can also select any of the custom fields but you cannot include any other common fields or XML fields.

Select True in the Is Cachable field for all the relevant fields in the Alert Type Fields of the item. These fields are cached in the XML of the parent item.
