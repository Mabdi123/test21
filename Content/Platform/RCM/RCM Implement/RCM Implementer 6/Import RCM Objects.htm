Import RCM Objects

From the main menu, select File and then click the Import option.

This action moves you to the first screen of the import wizard. You have the option of selecting a full or partial import:

  • Full Import: Import all objects from package (file) that have been previously exported.
  • Partial Import: Import specific objects from package (file) that have been previously exported.

NOTE:

Currently, you cannot partially import users/groups – only a full import of users/groups is possible.
