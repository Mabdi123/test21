Access the Alert (Work Item) Views Options

To access the tools and options for the alert view:

  • Click the Alerts bar and then click the Alert Views tab in the Entities pane.

The icons available within the Entities pane for this task are as follows:

Icon

Description

Displays four tabs on the right side of the screen that enable you to define a new alert view. The Alert Types tabs are:

  • General – Enables you to add a name, description, and identifier for the new definition.
  • Fields – Enables you to define the viewable fields and to format their display.
  • Default Sort – Enables you to define the default sorting of the viewable fields for this Alerts View
  • Filter – Enables you to define contents of this Alert View by editing the where of the query on the alerts.

Selecting an alert view within the list and clicking this icon opens the Field page enabling you to edit the details of the selected item/alert view.
