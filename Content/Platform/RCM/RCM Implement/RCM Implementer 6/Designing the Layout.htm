To insert a Custom Field inside the layout:

  • Select a field from the Available Fields pane and drag it to the desired layout cell.
  • Alternatively, click the Auto Populate icon to insert all fields into the layout automatically.
Formatting the Cell

Selecting a specific field provides a number of formatting options. The following formatting options are available for each field:

Option

Description

Cell content type
  • Field value - displays the dynamic value of the field.
  • Field label - displays the name of the field.
  • Free text - displays the text that is entered in the Free Text field of the cell properties that appears when this option is selected.

Bold, Italic, and Underline checkboxes

Select relevant checkbox to format the selected field’s displayed text in bold, italic, or underline.

Alignment

Select to align field text to the right, center, or left. If field data is in language where the text is read from right to left (e.g., Hebrew) then select the Right to Left checkbox.

Word Wrap

Wraps the column title text and the column data text with one of the following options:

  • Cut – Long text is shortened and this is indicated by an ellipses.
  • Extend – Column is extended to fit the size of text
  • Wrap – If necessary, text is displayed in multiple lines. Note that text cannot be broken down into lines if there are no space characters.

Additional Styling (CSS)

Select a specific CSS style for the field. For more information on adding CSS style, refer to Advanced Field Styles, page 1.

Display Formatter

For more information on this field refer to Column Display Formatter, page 1.
