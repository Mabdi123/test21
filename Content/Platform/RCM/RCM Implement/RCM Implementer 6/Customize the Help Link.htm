Customize the Help Link

Using the actimize.common.helpLinkURL configuration parameter as a template, you can customize the help link and define the URL for the help page.

The default value for the parameter is :help/Default.htm#RCM/[page].htm.

The [page] placeholder is replaced with a specific page based on the RCM context, and the colon place holder (:) is replaced with the base URL of the RCM application. Both placeholders are optional.
