Create a Checklist for the Step

NOTE:

Adding a Checklist to a step is optional. A step can be without a Checklist.

The Checklist feature enables you to add tasks for the RCM user during a specific step.

NOTE:

RCM does not enforce the user to complete all the step’s tasks before a move to the next step is allowed.

The Checklist Toolbar

The toolbar includes the following icons:

Description

Clicking this New Task icon creates a new line in the Checklist table.

Select the checkbox of the new text in order to make it a mandatory task.

In the Text column enter the name of the task.

The Action link column provides you with one of the following options:

Select an action from the drop-down list as shown in the following screenshot.

The RCM user can then click on the task’s action link in order to be moved to the relevant screen in the GUI. For example, the Edit Case button moves the user to the Edit Case window.

Refer to the Self Developer Guide for the following actions: the jump to query and the jump to batch actions.

Enter a web address (e.g., http://www.example12345.com). When the RCM user clicks on the task, the web address appears in a browser page. If you would like the URL to appear in a new browser window then enter the following: javascript:window.open("URL"). For example: javascript:window.open("http://www.example12345.com")

The Description column provides you with the option to enter a description for the task.

NOTE:

Action links are currently available only for alert workflows.
