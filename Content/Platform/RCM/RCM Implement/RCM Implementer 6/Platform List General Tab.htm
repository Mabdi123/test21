Platform List General Tab

The Platform List General tab enables you to define the platform list name and description.

The General tab includes the following:

Field

Description

Name

Provide a name for this platform list.

Identifier

Modify the identifier of the list.

Description

Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field box.

Four Eye Review

Select to enforce an approval for every new record that is added to the list by the RCM user

Support multi-tenant mode

Select the Support multi-tenant mode checkbox to enable support for the list in a multi-tenant environment.

Support test values

Select to enable support for test values in this list.
