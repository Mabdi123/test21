Delete All Documents in the Index

Using the curl command on Linux, or on Windows with Cygwin. For example:

curl http://localhost:8983/solr/a0b1c2d3/update?commit=true -H "Content-Type: text/xml" --data-binary '<delete><query>*:*</query></delete>'

Curl sends a POST request. An alternative is a GET request pasted directly in the Web browser address. The parameters need to be URL encoded. For example:

http://localhost:8983/solr/update?commit=true&stream.body=%3Cdelete%3E%3Cquery%3E*%3A*%3C%2Fquery%3E%3C%2Fdelete%3E
