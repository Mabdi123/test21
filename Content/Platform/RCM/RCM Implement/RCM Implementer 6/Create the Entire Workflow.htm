Create the Entire Workflow

Add step Nodes and Transition arrows in the same fashion that was described in the previous steps until the workflow is completed. Verify that there is only one Default Start Node and save the workflow.

  • An arrow that has clock icon indicates that is initiating step has a deadline.
  • An arrow with a note icon indicates that the user must add a note in order to move to the next step.

A transition is displayed in one of the following colors:

  • Matte blue – Standard transition
  • Orange – Allow automatic transition when touched is also selected
  • Dark blue – Barrier transition
  • Pink – A deadline transition when Change step to is selected

If the transition definition includes more than one color then transition color is determined according to the priority of the transition color. Pink transition color is the highest priority color and matte blue is the weakest priority color (Pink > Dark blue > Orange > Matte blue).
