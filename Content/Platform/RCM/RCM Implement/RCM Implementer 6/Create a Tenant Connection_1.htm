Create a Tenant Connection

To create a tenant connection, do the following:

  1. Select an AIS virtual connection from the Connection list that will be linked to the tenant connection that you want to create.
  1. Click the 'Add Tenant AIS Connection' action link. The new Tenant connection window opens.

  1. Select a tenant from the 'Select Tenant' dropdown menu. Only tenants that do not already use this connection appear in the menu.
  1. Enter a unique identifier name in the Identifier field and fill in all the other fields of the connection. These fields are identical to the fields of a standard AIS connection.

NOTES:

  • AIS functions in a tenant connection are read only and cannot be edited.
  • You cannot change the tenant when you are editing an AIS tenant connection.
