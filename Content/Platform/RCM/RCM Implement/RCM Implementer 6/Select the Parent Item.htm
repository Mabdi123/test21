Select the Parent Item

For the Action and Menu types proceed to the Select Parent Item page of the GUI Wizard.

The parent item (any Action Group type) indicates the location and display type of the action. Select the parent item for the action then click Next to continue to the next wizard step.

NOTE:

There may be definitions where the selected parent item must also have a parent item. All drop-down menu items within a drop-down menu item have a parent item and a grandparent item.
