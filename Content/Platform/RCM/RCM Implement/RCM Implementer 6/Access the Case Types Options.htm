Access the Case Types Options

To access the tools and options for the Case Types:

  • Click the Cases bar and then click the Case Types tab in the Design Entities pane (on the left side of the RCM Designer Screen).

The icons available within the Design Entities pane for this task are:

Icon

Description

Displays five tabs on the right side of the screen that enable you to define a new Case Definition. The Case Types tabs are:

General – Enables you to add a name, description, and ID for the new definition.

Case Type Custom Fields – Enables you to define the custom case fields for this case.

Case Type Fields Layout – Enables you to design the layout of the Case Type Custom fields when the Case details are viewed

Assigned Predefined Notes – Enables you to select possible notes for a specific Case Type.

Assigned Steps – Enables you to select possible steps for a specific Case Type.

Selecting a Case Type within the list and clicking this icon opens the General page on the right side of the screen, enabling you to edit the details of the selected Case Type.

Deletes a selected Case Type. You must close any page associated with this Case Type before you can delete it.

Duplicates the selected Case Type.

Refreshes the Case Type Screen.
