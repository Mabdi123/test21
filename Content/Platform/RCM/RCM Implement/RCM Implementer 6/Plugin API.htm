Plugin API

Exposes the following APIs to PlgCommonServices in order to expose to Plugin:

getUserDateTimeFormat() – Returns the user datetime format (both date and time) as defined in the local language.

getUserDateFormat() – Same as above, but only for the date format

getUserTimeFormat() – Same as above, but only for the time format

convertToUserDateTimeFormat(dateValue) – Converts a given date value from the input date format to the user expected date format.

convertToUserDateTimeFormat(dateValue, sourceDateFormat) – The second parameter is optional. If it is not present, then format is default RCM format (yyyy-MM-dd HH:mm:ss)

Example:

String dateTimeFormat = PlgCommonServices.getUserDateTimeFormat();
