RCM Web Services

RCM exposes a set of web services enabling external systems, such as the RCM Designer or the AIS, to access the RCM repository and to perform selected operations. RCM Web Services allow other systems to integrate with RCM, to query it for data and to perform various actions such as:

  • Managing RCM permissions
  • Managing users
  • Managing user groups
  • Managing authorization for various actions
  • Synchronizing RCM with the organizational structure
  • Defining business units
  • Defining business unit inter-relationships
