Assign Steps

The following describes the process of assigning steps in the RCM Designer.

Maintain Steps within the RCM Designer

All steps that are available for assignment to an Alert Type are written to the Steps Table within the RCM database. Steps within this table may be:

  • Common Steps – These are automatically assigned to all Alert Types.
  • Available Steps – These are steps which can, optionally, be assigned to any Alert Type using the RCM Designer Alert Types à Assigned Predefined Steps page.

These steps are originally defined via the Alert Steps page in the Cases à Configuration tab. For more information, see Alert Steps List.

Assign the Workflow of Steps or the Available Steps

The following window enables you to assign the step workflow.

The Steps Workflow tab enables you to select one of the following options.

Select Step Workflow Definition

Assign a predefined workflow to the alert type. This option is achieved by selecting a workflow from the Step Workflow Definition drop-down menu. For more information, see Set Up Step Workflows for Cases.

The Workflow Assignments Rule Table allows having more than one workflow for the same alert type. The alert/item instance can be mapped to different workflows not only by the alert type but also by the Business Unit. For example, an alert instance from an alert type that was opened on BU1 can have one workflow while another alert instance from the same alert type can be linked to BU2 with a different workflow. The prioritization of the Workflow Assignments Rules avoids any mapping conflicts (for example, a BU on one hierarchy appearing on another hierarchy).

To define a new Workflow Assignments Rules:

  • Click the Add new workflow rule icon .
  • Select a workflow for the rule from the drop-down menu.
  • Select whether or not include descendants and map a BU hierarchy to the workflow rule.
  • If you selected to include descendants in the previous column then select the BU hierarchy for the rule.
  • Select the Business Unit that will be mapped to the rule.
  • Add additional rules and then use the and icons to prioritize the rules.

Select Possible Steps Assignment

Add any Available Step to the list of Selected Steps for the alert type that you are defining. All Common and Selected Steps appear within the drop-down list of steps that are available to the RCM Web Application end users. All Common Steps appear within the RCM Designer Alert Types àAssigned Predefined Steps.

To add a step to the list of selected steps, select the step that you want to add from the list of Available Steps and click Add. The step is added to the list of Selected Steps.
