Create a New Connection

When JNDIConnectionManager is set as the connection management option, the following window opens when you create a new connection:

To create a new connection:

  1. Assign an internal name to the connection within the Connection Name field.
  1. Select the name that was configured within the WebLogic console for this connection.
  2. Select the database type.

NOTE:

The Database Type must always be configured.

  1. Click to validate the connection information.
  1. Save the connection.
  • If the connection is successful, the connection becomes active immediately after clicking Save.
  • If the connection fails, the information is saved within the list of Connections.
