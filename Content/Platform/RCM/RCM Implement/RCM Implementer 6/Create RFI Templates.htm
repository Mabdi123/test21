Create RFI Templates

This feature enables you to create and manage templates for RFI messages that are then used by the RCM user. The RCM user can select a predefined template and use it for an RFI message. These templates expedite and simplify the use of RFI messages. VTL code can be added to the RFI template in order to insert the relevant work item details when the RCM user creates an RFI.

Refer to the Administration chapter of the RCM Reference Guide for more information.
