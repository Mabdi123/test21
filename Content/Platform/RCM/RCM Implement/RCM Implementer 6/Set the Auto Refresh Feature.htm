Set the Auto Refresh Feature of Alerts (Work Items)

To set the auto refresh feature of alerts, go to the Config Parameters tab of admin.jsp and define the following parameters:

  • actimize.alerts.autorefresh.state – Enter one of the following definitions:
  • default_on – The autorefresh features appears in the Alerts tab and is enabled. The user has the option to disable it.
  • default_off – The autorefresh features appears in the Alerts tab and is disabled. The user has the option to enable it.
  • disabled – The autorefresh features does not appear in the Alerts tab.
  • actimize.alerts.autorefresh.stopTimerInMinutes – Enter the time in minutes before the auto refresh mode turns off. Value must be between 1 and 1440 minutes (24 hours). The default value is 45 minutes.
  • actimize.alerts.autorefresh.intervalInSeconds – Enter the time in seconds between screen refreshments. Value must be between 30 and 3600 seconds (1 hour). The default value is 1 minute.
