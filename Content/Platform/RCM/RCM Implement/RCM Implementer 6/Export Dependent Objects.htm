Export Dependent Objects

Some of the objects that can be exported have dependent objects. For example, a specific alert view depends on alert settings such as built in fields. The RCM Designer provides you with the option to automatically include the dependent object(s) of the object(s) that you select for export. This is achieved by selecting the checkbox of this option.

If this checkbox is cleared then when you select an object that has dependent object(s), a three dots icon appears at the end of the object’s name. Clicking on the icon opens a menu that enables you to select the dependent object(s) you want to export. If you want to select more than one dependent object then select the objects while pressing the Ctrl key. You can also click the Select All button at bottom of the menu to export all the dependent object(s).
