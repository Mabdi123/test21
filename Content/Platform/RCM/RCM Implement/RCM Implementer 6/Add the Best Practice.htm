Add the Best Practice

Once you are content with the functionality of the Best Practice that you created, you can add it to the Best Practices Menu. Close the RCM Designer and then copy the JSON file and folder that you created to the Best Practices folder of the RCM Designer. For example.

C:\Program Files (x86)\Actimize\CaseManagerDesigner\Bin\Best Practices

Open the RCM Designer. In the following example the Create a Platform List file and folder has been copied to the Best Practice folder and therefore this Best Practice appears on the menu.
