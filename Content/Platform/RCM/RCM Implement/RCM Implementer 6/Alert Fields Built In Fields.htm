Alert Fields - Built In Fields

The following are the common fields that appear in the Alert Fields - Built In Fields window.

You cannot remove or add built in fields. The only properties of these fields that can be modified are the display title and the description. Built In Fields appear in the Advanced Filter window and may optionally be defined in specific Alert Views. See Set up Alert Views.

The options that are available within the Built In Fields page are:

Option

Description

Field

A view only property that displays the field name in the database.

Table

A view only field that indicates the table where this field is stored. All tables are stored in the database of the RCM.

Display Title

Built In Field’s display title. This property can be modified.

Description

Built In Field’s description. This property can be modified.

Data Type

A view only property that displays data type (String, Integer, Real, Date, and Datetime).
