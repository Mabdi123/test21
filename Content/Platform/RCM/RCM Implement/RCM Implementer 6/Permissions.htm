Permissions

The Modules section of the Role window includes the Tuning module.

A Tuning Context object has the following permissions:

  • All – All actions are permissible
  • View – User can view the Tuning Context, and view test results.
  • Edit – User can create, modify and delete test values. User can also revert or apply them.
  • Execute – User can execute the test, and can cancel it.

NOTE:

Editing test items also requires editing permissions on the related Platform List.
