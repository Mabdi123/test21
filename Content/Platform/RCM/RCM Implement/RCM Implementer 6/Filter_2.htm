Filter

Define the contents of this view by selecting a setting in one or more of the filters that are provided in this page.

The following filters are set in the drop-down menu:

Case Type

  • State – Closed or Open
  • Finding – issue, non issue, and no determination
    Owner
  • Step – In Process, Ready, etc.

Select one of these settings for each of the above filters:

  • Any – All options are selected. This setting is equivalent to not filtering the parameter.
  • Select a check box of one or more of the available options in the dropdown menu

You must click Accept button in each filter in order to confirm your settings.

Additional filters include:

Filter

Description

Business Unit

Select between the current business unit and any business unit (all business units).

Aging

Filter the cases via their age in days. Refers to the field ‘Case_date’ that the user can select and not to field ‘create_date’.

Archived

Select from one of the following options:

Show only archived reports

Do not include archived reports

Include archived reports

Limit Score Range

Enter a minimum and a maximum score for the alerts.
