Create a Workflow

The following enables you to create a new workflow or assign steps.

To create a new alert workflow:

  1. Using the options within the General tab, provide a name for this workflow.
  1. Enter the alert workflow Identifier.
  1. Provide a description, if required.
  1. Select the checkbox to add a deadline to a workflow and enter the relevant options.

See Workflow General Tab for the available configuration options.

When creating a deadline, verify the following:

The work item (alert) should belong to an alert type that has been associated with a workflow definition. Even though a deadline may be defined for a step, it will only apply to an alert in that step when the alert type is associated with the workflow.

For Send Email Notification:

  • All the mail.smtp. RCM configuration parameters must be defined.
  • The alert must have an owner.
  • The alert owner must have a defined email address.
  • The email automatically goes to the work item owner. If the owner does not have a valid email address or the alert does not have owner, it is mandatory to enter an address in the TO field.
