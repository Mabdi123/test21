Setting Up Outgoing Form Types

NOTE:

Using Outgoing Forms in the RCM requires a license.

Setting up and configuring Outgoing Forms in RCM Designer requires an additional license.

This module enables the implementer to create and edit automatic regulatory forms for the RCM user (e.g., SAR and CTR forms). These forms are XML based and include validation and mapping capabilities to ensure that the correct data is entered and processed. They can be easily modified in order to be compatible with the most up-to-date internal and regulatory forms.

For more information refer to the RCM Outgoing Forms Implementer’s Guide.
