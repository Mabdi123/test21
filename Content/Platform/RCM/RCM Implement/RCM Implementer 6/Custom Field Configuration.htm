Custom Field Configuration for Global Search

You can enable specific field searches in the RCM Global Search by adding the relevant data type endings to the custom field in the Identifier parameter. For example if the custom field is a text field with the identifier of first_name you would enter first_name_custom_txt.

Identifiers must not contain any blank spaces. All empty spaces will have an underscore added.

 

Data Type

Added to Custom Field

Text

<identifier>_custom_txt

Integer

<identifier>_custom_ti

Double

<identifier>_custom_td

Date

Calendar/Clock

<identifier>_custom_tdt
