Mapping Tag Types to Alert Types and/or Transaction Types

The Tag Types tab appears in Alert Types and Transaction Types and Blotter Transaction Types.

To map a Tag Type to a specific Alert Type or Transaction:

  1. Click the Add Tag Type icon.
  1. Click the arrow next to the tag type and select the Tag Type.

  1. If required repeat the procedure to map additional Tag Types.

The Tag Types appear as input fields when a note is added or edited:

The View Notes window provides information on all the Tag values and enables the viewer to use the Tag values as links to more information about a specific value.

For example, when the user clicks on John Smith Broker value, all the relevant notes about John Smith are displayed.
