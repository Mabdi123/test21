Fields Filter Tab

The filtering mechanism enables you to filter a Platform List according to a specific property. Select a list and click to open the Properties and display the Fields Filter tab.

You can filter the same Platform List more than once. Each filter can then appear as a separate node of the tree. The following screenshot illustrates the Tree view according to the filtering:

The options of the filtering mechanism are identical to those of built in alert fields. For more information on using all the options of field filtering, see Fields Filtering.
