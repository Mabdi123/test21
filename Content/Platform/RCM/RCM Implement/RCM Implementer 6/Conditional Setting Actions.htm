Conditional Setting Actions

The Settings Activity Bar includes the following actions:

The options are as follows:

Action

Description

New Conditional Setting

You must create at least one conditional Setting Template via the New Setting action in order to use this feature. See Define Settings.

Note that this action cannot be done for Lists and Thresholds in the RCM. Click the New Conditional Setting action link. The New Conditional Setting window opens.

Select a template for the Conditional Setting. A Conditional Setting Template is created via the New Setting action. See Define Settings.

Enter the Conditional Setting name and identifier. This is the name that can later be added to the Settings Navigation Tree.

Enter a value for the conditional setting parameter(s). Note that there can be more than one parameter in a specific template.

Click Save. The Conditional Setting now appears in the list.

The Conditional Setting now appears in the Available Settings Window of the Designer and can be added as a setting to a tree.

 

For more information on creating Settings Navigation Trees, see Setting and Platform Lists Navigation Trees Setting and Platform Lists Navigation Trees, page 1.

Edit Conditional Setting

Enables authorized personnel to edit selected conditional setting definitions. This action should only be done in consultation with authorized Actimize IT professionals.

Delete Setting

Deletes a selected Conditional Setting definition. This action should only be done in consultation with authorized Actimize IT professionals.
