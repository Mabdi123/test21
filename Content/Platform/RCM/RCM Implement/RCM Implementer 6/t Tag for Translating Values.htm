t Tag for Translating Values

<rcm:t select="/some/xpath" />

<rcm:t byId="true" text="<identifier>" escapingType=”js”/>     

<rcm:t module=”<module_name>” select="/some/xpath" />

  • select – An xpath or variable (same as the xsl:value-of tag) of values that need to be translated
  • text – A constant value that needs translation
  • byId – True means the given value is an identifier, false means it is a base value. Indication for both select and text attributes. Default is false.
  • module – A module name (optional). Default is ‘Actimize‘or the module that was set by setModule tag.
  • escapingType – ‘html’, ‘js’, 'pdf', ‘none’. Default is ‘html’. 'pdf' is the default when the tags are used in an XSL that creates PDF files

For example:

<xsl:for-each select="/enrichedAlert">

<table><tr><td><rcm:t select="@alertType" /></td></tr></table>
