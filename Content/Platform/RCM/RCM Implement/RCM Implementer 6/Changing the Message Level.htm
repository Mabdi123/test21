Changing the Message Level for RCM Designer

RCM Designer log messages provide detailed information about any connection problems between the RCM Designer and the RCM Database and Web Services. The logs are maintained, according to the application version number, under the following location:

C:\Documents and Settings\All Users\Application Data\Actimize\RCM Designer\<version number>

Authorized users can change the information level of database messages and web service messages, in the log, as follows:

To change the level of messages within the Designer log:

  1. Open the Actimize.Applications.CaseManagerDesigner.exe.config file, which resides under the same directory as the application itself.
  1. Locate the following <switches> section:

<switches>

<add name="DatabaseMessagesSwitch" value="0" />

<add name="WebServicesMessagesSwitch" value="0" />

</switches>

  1. To suppress the detailed messages within the log and provide only high-level information, enter the value of 0 for either of the switches.

For full detailed messages about connections, enter the value - 1 for either switch.

Since the detailed log might include confidential information ensure it is secured and therefore not accessible to unauthorized users.
