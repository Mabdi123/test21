Platform Lists and Setting Properties

Unlike Settings, Platform Lists can appear multiple times within the same folder, possibly with different filters.

Therefore, the leaf nodes that refer to a platform list have an editable identifier. This identifier must be unique within the scope of its folder.

The following table summarizes the properties that the user is able to edit on a leaf node:

Leaf Object:

Viewable Fields

Editable Fields

Setting and Conditional Setting

identifier, name

both are read-only

Platform List, with or without filter

identifier and name are editable
