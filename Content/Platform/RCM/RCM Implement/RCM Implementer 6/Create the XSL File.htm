Create the XSL File

Create XSL files for each alert type that will include a widget. The XSL file includes:

  • JavaScript command that creates the widget
  • Instructions for combining the widget in the display (HTML) for Internet Explorer

XSL file example with comments inside it:

  <xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

- <!--

HTML DEFINITION

  -->

  <head>

- <!--

Creating a widget while adding the optional parameters

  -->

  <script language="javascript">function loadWorkItemsWidget() { var WorkItemsWidget1 = new WorkItemsWidget('w1', 'div1', { viewIdentifier: "Ch_View", ddqIdentifier: "AlertIdByOwner", ddqParamValues: ["admin", "09-22-04", 251], selectionMode: RCMAPI.GridSelectionModesEnum.MULTI, showResetButton: true, showRefreshButton: true }); WorkItemsWidget1.draw(); } window.attachEvent('onload',loadWorkItemsWidget);</script>

  </head>

  <body>

- <!--

Calling for the widget as div1

  -->

  <div id="div1" style="border: 2px solid blue; width: 100%; height: 100%" />

  </body>

  </xsl:template>

  </xsl:stylesheet>

All XSL files should be inserted in the app_gui_items folder (the path can be changed in the config parameter actimize.alerts.pathToXSL).
