Create New Language Settings

The RCM Designer enables implementers to set up additional languages that are used within the RCM Application.

Setting up a language requires the following tasks:

  • Providing a name, description, and ID for this language.
  • Selecting a language code and country code.
  • Entering the translations in the Translations window and/or optionally exporting a comma delimited file of the translation strings in order to be translated by a professional translator and then importing the comma delimited file back to the Translations window.
