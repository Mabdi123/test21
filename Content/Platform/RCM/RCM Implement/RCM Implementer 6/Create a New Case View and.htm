Create a New Case View and Assigning General Information

The following enables you to create a new case view

To create a new case view:

  1. From the Views tab, click the icon.
  1. The screen on the right side changes to include four tabs: General, Fields, Default Sort, and Query tabs.
  1. Using the options within the General tab, provide a name for this view.
  1. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description box.
  1. Enter the Case View Identifier that you want to appear within the RCM activity bar. The Case View Identifier is a string that represents the Case View within the Case-view table of the RCM database.
  1. Select the checkbox to enable selecting this view in the User Roles Settings of the case Manager.
