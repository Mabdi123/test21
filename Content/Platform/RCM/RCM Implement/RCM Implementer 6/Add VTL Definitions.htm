Add VTL Definitions

This tab is enabled only if you selected an embedded VTL file and enables to customize the display via VTL code.

During the test run the value of $tuningTest.lastRunStatus is null and in order to view the null value a function should be added to the VTL tab. For example, the following function could be written:

<div>lastRunstatus :

            #if( ! $tuningTest.lastRunStatus )

                  null

            #else

                  $tuningTest.lastRunStatus

            #end

</div>-->

Otherwise the RCM user view the text “$tuningTest.lastRunStatus” after running a test.

$tuningTest is an object to be used in the VTL:

  • $tuningTest.testIdentifier – String
  • $tuningTest.tenantIdentifier – String
  • $tuningTest.contextIdentifier – String
  • $tuningTest.executionStatus – String. “Not_Running”, “Aborting”, “Running”.
  • $tuningTest.lastRunStatus – String. “Succeeded”, “Failed”, “Aborted”.
  • $tuningTest.parameters – Array of Strings. The parameters that RCM passed to the execution.
