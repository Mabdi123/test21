Customize the Header

You can customize the RCM application header using the VFS.

In the VFS, under the /app_gui_items/rcm/images/logo folder, you can replace the SVG file with the logo you want to add.

  • The file must be an SVG file and the name must be logo.svg (lowercase letters only).

  • The file width must be 71 px and the height 50 px.

  • You must clear the cache in the RCM application web browser.
