Cases Fields - Built In Fields

These are the common fields, which appear in the Advanced Filter window. You cannot remove or add Built In Fields.

The only properties of these fields that can be modified are the display title and the description. Built In Fields appear in the Advanced Filter window and may optionally be defined in specific Case Views. See Set up Case Views.

The options that are available within the Built In Fields page are:

Option

Description

Field

A view only property that displays the field name in the database.

Display Title

Built In Field’s display title. This property can be modified.

Description

Built In Field’s description. This property can be modified.

Data Type

A view only property that displays data type (String, Integer, Real, Date, and Datetime).
