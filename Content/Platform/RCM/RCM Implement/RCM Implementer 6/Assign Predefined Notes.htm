Assign Predefined Notes

You can assign predefined notes for alerts as follows.

Maintain Predefined Notes within the RCM Designer

All notes that are available for assignment to an alert type are written to the Notes Table within the RCM database. Notes within this table may be:

  • Common Notes – These are automatically assigned to all alert types.
  • Available Notes – These are notes which can, optionally, be assigned to any alert type using the Designer Alert Types à Assigned Predefined Notes page.

These notes are originally defined via the Predefined Notes page in the Alert Configuration tab. For more information, see Alert Predefined Notes.

Customize the Assignment of Notes

You can add any Available Note to the list of Selected Notes for the alert type that you are defining.

All Common Notes appear in the RCM Designer Alert Types àAssigned Predefined Notes path within the drop-down list of notes that are available to the RCM Web Application end users.

To add a note to the list of selected notes:

  • Select the note that you want to add from the list of Available Notes and click Add.

Available Notes only appear within this list if there are notes, defined within the Notes table of the RCM database, that are assigned the value 0 for the For_all_types field.

The note is added to the list of Selected Notes. You can also click on the Use all Resolutions checkbox in order to use all resolutions that are currently included within the Resolutions table of the RCM database.
