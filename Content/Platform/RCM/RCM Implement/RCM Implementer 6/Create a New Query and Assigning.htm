Create a New Query and Assigning General Information

The following describes how to create a new query.

NOTE:

If queries already exist in the RCM database, the existing queries appear automatically within the list of queries.

To create a new query:

  1. From the Queries bar, Queries tab click the icon.

The screen on the right side changes to include three tabs: General, Query Definition and Result Display tabs.

  1. Using the options within the General tab, provide a name for this query. Select the checkbox to enable a user with the relevant permissions to edit the values of the DDQ via the RCM application. See Changing the Values of the DDQs in the RCM.
  2. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field.
  3. Click the Query Definition tab to bring this page forward and continue with the definition of the query.
