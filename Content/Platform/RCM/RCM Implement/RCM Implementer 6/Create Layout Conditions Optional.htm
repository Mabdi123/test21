Create Layout Conditions (Optional)

The Layouts tab enables you to define and format layout conditions. A layout condition is a single Boolean expression that determines whether its layout must be used or not.

Click the new Layout icon and define an identifier, description and condition for the layout. The first layout in the table has the highest display priority. The priority is the order in which the conditions are evaluated. The first layout condition that is true is the one that is used. For example if the first of three condition returns false and the second condition returns true, then the layout that is used and displayed will be the layout with the second priority. (The third condition is not evaluated).

If you would like to display all conditions that return true, select the Enable multiple layouts checkbox.

NOTE:

It is possible to add a default layout by adding the word true in the condition field.
