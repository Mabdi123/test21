Create Groups of Sensitive Data Fields

Sensitive Data Groups are used for allocating permissions to the RCM application users. When the user has permissions to view a specific Sensitive Data Group, they are able to see the unmasked value of the field(s) of that group.

Click the General bar and then click the Configuration tab in the Entities pane. Select the Groups page of Sensitive Data.

To create a group:

  1. Click the New Group icon:
  1. Define a name, unique identifier and optionally a description for the group.
  1. In the Fields column, click the icon to open the list of Alert Custom Fields that have a defined mask.
  1. Select the custom fields that you want to add to this group and click the Accept button.
