Removal of HTML Elements

The removal of the HTML elements (tags) and attributes can be made via JavaScript (hen text is pasted), AJAX call (when text is pasted) or in Java code (on submit).

The following HTML elements that are removed in the rich text feature:

  • <a> - creates a link to another document, usually referred to as a link or a hyperlink.
  • <applet> - defines an embedded applet.
  • <button> - defines a push button.
  • <embed> - defines a container for an external application or interactive content (a plug-in).
  • <form> - is used to create an HTML form for user input.
  • <frame> - defines one particular window (frame) within a <frameset>.
  • <frameset> - defines a frameset, holds one or more <frame> elements.
  • <iframe> - specifies an inline frame.
  • <img> - defines an image in an HTML page.
  • <input> - is used to enter user information.
  • <link> - defines the relationship between a document and an external resource.
  • <object> - is used to include objects such as images, audio, videos, Java applets, ActiveX, PDF, and Flash.
  • <script> - is used to define a client-side script, such as a JavaScript.
  • <select> - is used to create a select list (drop-down list).
  • <textarea> - defines a multi-line text input control.
  • <q> - defines quotes.
  • The following HTML attributes that are removed in the rich text feature:
  • onclick - event occurs when an object gets clicked.
  • ondblclick - event occurs when an object gets double-clicked.
  • onmousedown - event occurs when a mouse button is clicked.
  • onmouseup - event occurs when a mouse button is released.
  • onmouseover - event occurs when the mouse pointer moves over a specified object.
  • onmousemove - event occurs when the mouse pointer is moved.
  • onmouseout - event occurs when the mouse pointer moves away from a specified object.
  • onkeypress - event occurs when a keyboard key is pressed or held down.
  • onkeydown - event occurs when a keyboard key is pressed.
  • onkeyup - event occurs when a keyboard key is released.
