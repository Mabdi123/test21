ACMResult

ACMResult {

boolean Step;

LogMessage[] logMessages;

}

Description: This object is used as a return value of every web service method.

Fields:

  • Step (true/false) – Whether the web service method was successful or not.
  • LogMessages – One or more log message objects.

NOTE:

Web services are transactional, thus any failure performs complete rollback of transaction unless otherwise noted in method documentation.
