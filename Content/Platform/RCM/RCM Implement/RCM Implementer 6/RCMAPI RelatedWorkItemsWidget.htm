RCMAPI.RelatedWorkItemsWidget(widgetId, htmlContainerId, optionalParams)

This API is similar to the WorkItemsWidget but is intended for a very specific task. This API can only be used from the Work Item details page and it can only show the children of that item rather than allowing the basis for selecting the work items on a generic drill down query.

For example, if the details page shows item 'A' that has two related items 'B' & 'C', then only these two items will be shown in the widget grid. This restriction allows for a much simpler API call and setup. No drill down query needs to be created and the API arguments list is shorter.

Specifically, the two options ddqIdentifier & ddqParams are no longer needed. A sample call to this API may include the following code:

var myRelatedWorkItemsWidget = new RCMAPI.RelatedWorkItemsWidget('w1', 'div1', {

viewIdentifier: "allAlerts",

actionsGroupIdentifier: "workItemsWidgetActions"

});

myRelatedWorkItemsWidget.draw();
