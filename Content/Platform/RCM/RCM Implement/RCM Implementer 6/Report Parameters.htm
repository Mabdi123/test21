Report Parameters

Once you have uploaded the Report files, you can define the parameters. Click the Report Parameter tab to bring this page forward.

All parameters that were defined for this report catalog appear within the list of Report Parameters.

The Parameter Name and Type display according to the definition set for the report. You can (optionally) change the Display Name to a name that is more appropriate for display purposes.

The Hidden column provides you with the option to hide the parameter from the RCM end user. This is useful when you prefer a default setting for the parameter that should not be modified.

NOTE:

If a parameter is defined as Required in JReport then this definition cannot be changed to Not Required in the Designer.

If the Required option is selected then the parameter must be filled in. A parameter that is defined as both hidden and required indicates that the user cannot modify its value and that the implementer must supply this value in the Designer.
