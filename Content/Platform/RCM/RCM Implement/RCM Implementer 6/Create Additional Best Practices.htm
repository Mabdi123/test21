Create Additional Best Practices

There are two types of Best Practices:

  • Out-of-the-box Best Practices that are released with RCM Designer.
  • Additional self developed Best Practices. This describes how to create additional Best Practices.

Best Practices are located in the RCM Designer bin folder

(e.g., C:\Program Files (x86)\Actimize\CaseManagerDesigner\Bin\Best Practices). Each Best Practice is composed of a JSON file and a folder that has the same name. For example, Create Alert Type.json and the folder Create Alert Type. The JSON file includes all the definitions for the Best Practice and the related folder includes a folder for each step which contains the More Details help Images.
