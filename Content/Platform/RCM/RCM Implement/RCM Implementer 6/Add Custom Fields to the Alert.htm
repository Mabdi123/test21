Add Custom Fields to the Alert Type

Alert Type Custom Fields are the additional fields, which may appear in a specific details view and in the Advanced Filter window. For more information on adding Custom Fields see Alert Fields - Custom Fields.

You have the option of adding these fields into the alert details of this Alert Type in the RCM. These fields are then placed in the layout of this Alert Type via the XSL definitions. For more information about this window, see Identify the XML File and XSL Location.

The icons available within the Alert Type Fields window are:

Icon

Description

Enables you to add a new custom field to the alert type.

Deletes a selected custom field.

and

Move a specific alert type custom field up or down in the list in order to change its viewing order.

Toggle the display of the identifiers of the fields.

The options that are available within the Alert Type Custom Fields page are:

Option Description

Field

Select a custom field by clicking the arrow next to this field.

Is Mandatory

Select between True and False:

  • True – User must fill in the field
  • False – User does not have to fill in the field

Update Mode
  • Not Updateable – User does not have the option to update this field.
  • Updateable – User has the option to update this field.
  • Updateable On Creation – User may edit this field only when a manual alert is created.
  • Hide on Edit – The field does not appear when the user edits the alert.
  • Hide Always – The user cannot view this field in all modes.
