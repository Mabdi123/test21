APF Comparer Tool

The APF Comparer tool enables you to compare two RCM APF files to see where objects have been changed, updated, added or removed.

To compare APFs:

  1. Navigate to File -> Compare APFs…

  1. In the Compare APFs window enter the following:
  • APF 1: - Enter or browse to select the first file you want to compare.
  • APF 2: - Enter or browse to select the second file you want to compare.
  • Remove Workflow Definitions – Select this option to remove the workflow comparison for alerts and cases to avoid showing these workflow differences. Because workflows are displayed in hierarchies, identical workflows can be represented differently across environments. This can appear as many differences in the APF comparison even though the workflows are identical.
  • Comparison Utility – Click the icon to expand the Comparison Utility section to configure the tool parameters.
  • Path – Enter or browse to the path for the compare tool you want to use. You can also drag an exe file from Windows Explorer to the Path field.
  • Arguments – You can add any supported arguments in this section. The only mandatory arguments are $input_file1 and $input_file2 which are placeholders for the input files.

For example:·$input_file1 $input_file2

  1. Click Compare to begin the compare process in your selected tool.

NOTES:

All APF Comparer window settings are saved per user even when Designer is closed

Only the content.xml file in the APF is compared, and not the header.xml and the users.xml.
