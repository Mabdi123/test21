Quick Filter Guidelines

Quick filter uses a different format since a database specific SQL code is required for the filtering to speed up the loading of Alerts / Cases screens. Since there is no trivial way to translate the Java date format to a specific database format, the following configuration parameter was added:

actimize.common.quickFilterDateConversionString

This parameter is used to format date columns when using Quick Filter.
‘?’ is a placeholder for the column name

Examples:

Formatting a date in the form Month/Day/Year and time in 12 hour format in Oracle
to_char(?,'MM/DD/YYYY HH12:MI:SS AM')
Format a date in the form Month/Day/Year and time in 24 hour format in MSSQL: (convert(varchar,?,101) + ' ' + convert(varchar,?,108)).
