Add Parameters to the Query (Optional)

You can (optionally) add parameters to the Query that you defined for the current catalog.

To add parameters:

  1. Edit the Query to include the required parameter(s) (as shown in the example below):

  1. From the Catalog Browser, right mouse click on Parameters and select Add Parameter from the resulting pop up menu.

The Create Parameter window opens, enabling you to enter the parameter name, value type and the value options that you want available for this parameter (as shown in the example below):

  1. Click OK within the Create Parameter window to accept the new parameter.
  1. From the JReport Designer main menu options, select File à Save Catalog to save the catalog.
