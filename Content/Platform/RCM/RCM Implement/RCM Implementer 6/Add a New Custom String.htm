Add a New Custom String

Add a custom resource string by clicking ‘New Custom Resource String’ button.

A new row in the base language appears with the ‘ACTIMIZE’ module by default. Enter a base value for the resource string and if required change the module name.

The default identifier is set to <module>.<base value>. If the length of the string <module>.<base value> is more than 590 characters then the <base value> in the identifier is trimmed.

Changing the base value does not change the identifier name. You can modify the identifier but the module prefix that has to be the same as the module name and cannot be modified. The rows that contain the custom resource strings include a white background which indicates that they are editable.

NOTE:

‘RCM’ and ‘RCMMD’ are reserved module names for the built in and metadata resource strings.

Follow these rules regarding Resource strings and module definitions:

  • The identifier must be unique, not empty and contains only lower case letters.
  • The module name contains only upper case letters.
  • The module name consists of alphanumeric characters and cannot be empty.
  • The module name cannot contain more than 50 characters.
  • The base value cannot be empty
  • The base value cannot contain more than 1049 characters.
  • The identifier cannot contain more 590 characters.
