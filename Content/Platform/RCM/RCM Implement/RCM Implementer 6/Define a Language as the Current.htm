Define a Language as the Current RCM Language

The following describes setting up the current RCM language.

In order to have a specific Non-English language displayed in the RCM you must do the following after the translation has been completed:

  1. Set the selected language as the current language by right clicking on the specific language node and selecting the Set As Current option.
  1. The RCM users must logout of their current session and clear the cache of their Web browser. After these actions are done, they can login again in order to view the RCM in the selected language.
  1. Update the value of the parameter actimize.language.currentACMLanguage via the Config parameters tab of admin.jsp. The value of the parameter should be the identifier of the language that you want to be displayed in the RCM. In order to support multiple languages, configure this parameter in the acm.ini file for each RCM server.
