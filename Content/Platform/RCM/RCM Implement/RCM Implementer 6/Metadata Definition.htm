Metadata Definition

To access the DART Widget Renderer, click the Queries bar and open the Layouts tab. In the Display Formatter field of the Cell Properties pane section of the the Layouts tab, select DART Widget Renderer. This enables you to set the secondary layouts used by the records widget.

The format string should include an XSL API to create the internal widget. The XSL code encapsulates the same XSL used for the creation of the Records Widget, but uses additional tags to retrieve the rest of the required information.

NOTE:

It is permitted to use <fetch> tags in the widget renderer's format string.
