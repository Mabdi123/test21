Set up Case Views

NOTE:

You must logout and then login to RCM in order to view the changes that you made in a view.

The RCM Designer enables implementers to set up the Case Views that are used within the RCM Application. Each view is usually composed of Built In Fields and Custom Fields.

There are two types of Case Views:

  • System Views: Views that are readily available after the RCM Designer installation.
  • Defined Views: Defined Views that can be added by the implementer.

Setting up Defined Case Views requires the following tasks:

  • Providing a name, description for this Case View
  • Defining the fields, which will be viewed, and the format of their display
  • Defining the default sort of the Cases View, and/or defining a filter for it via an SQL query
  • Saving the newly configured view
