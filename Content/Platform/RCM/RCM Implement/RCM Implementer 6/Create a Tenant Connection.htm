Create a Tenant Connection

To create a tenant connection, do the following:

  1. Select a virtual connection from the Connection list that will be linked to the tenant connection that you want to create.
  1. Click the New Tenant Connection action link. The new Tenant connection window opens.

  1. Select a tenant from the Select Tenant dropdown menu and enter a unique name in the Connection Name field.
  1. Fill in all the other fields of the connection. These fields are identical to the fields of a DBCP Connection.

NOTES:

Creation fails when the selected Virtual Connection and tenant already exist.
