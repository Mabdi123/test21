About this Guide

This guide has been designed for authorized administrators and IT personnel. It describes all tasks required to install both Risk Case Manager (RCM) applications and to start receiving the first alerts. Administrators should be familiar with the AIS Visual Modeler application before customizing the RCM application.

Advanced development options, for the Risk Case Manager features, are not covered in this guide. For information about customization of RCM integration with the AIS Models and with external data sources, please refer to your RCM Technical Support. Certain features and functionality described in this manual are available only when licensed as part of specific Business Solutions, ERCM, or with the version of additional Value Added Products. These features and functionality may not be supported or available in your implementation.

This document may describe features and capabilities that require a separate license and are not covered by the standard license. For information about licensing please contact your Actimize representative.
