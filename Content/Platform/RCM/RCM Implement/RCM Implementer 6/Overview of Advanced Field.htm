Overview of Advanced Field Styles

The Advanced Field Styles (CSS) page enables you to define display styles that you can later select when defining the format for each field of Alert and Case Views. For example, this page enables you to define the style Bold 14 for a specific field.

This is achieved by defining Cascading Style sheet Attributes and then later associating them with a specific field within the Field tab of an alert or case view, as shown in the screen shot below.

For More information on formatting a view for a field, see View Properties for Field.
