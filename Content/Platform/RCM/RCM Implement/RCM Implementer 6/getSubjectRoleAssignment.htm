getSubjectRoleAssignments

ACMResult getSubjectRoleAssignments(in String subjectIdentifier, out RoleAssignment[] roleAssignments);

Description: This method gets all role assignments of a subject.

IN

  • subjectIdnetifier – A User Identifier or a Group Identifier. If a null value is passed, the return value will include all role assignments.

OUT

  • roleAssignments – An array of Role Assignments.
