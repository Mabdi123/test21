Create a New Navigation Tree and Assigning General Information

The following describes how to create a new Navigation Tree.

To create a new Setting Navigation Tree:

  1. From the Setting Navigation Tree tab, click the icon.

The screen on the right side changes to include two tabs: General and Tree Structure tabs.

  1. Using the options within the General tab, provide a name for this tree.
  2. Enter a unique ID for the Setting Navigation Tree. This ID represents the Setting Navigation Tree within the Setting Navigation Tree table of the RCM database.
  3. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field box.
  4. If you want the tree to be supported in Tuning Context(s) then select the Support Tuning Module checkbox. See also Setting up Tuning Contexts, page 1.
