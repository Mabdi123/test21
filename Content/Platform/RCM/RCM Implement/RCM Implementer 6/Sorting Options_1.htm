Sorting Options

Basic Sorting

Each column provides with a sorting feature. Click the arrow on the column heading to create an ascending or descending table sort according to the column.

Advanced Sorting

Click the small arrow on the column heading to open a dropdown menu that provides you with advanced sorting options such as sorting using a particular string or by only using blank fields.

Grouping by columns

The Find and Replace Utility

The Find and Replace Utility includes the following fields and icons:

Field

Description

Find what

Enables you to enter the text that you want to find.

Replace with:

Enables you to enter the text that you want to replace with the text in the Find what field.

Find Next

Clicking this icon moves you to the next row where the text in the Find What field appears.

Replace

Clicking this icon replaces the highlighted text with the text in the Replace with field.

Find Previous

Clicking this icon moves you to the previous row where the text in the Find What field appears.

Replace All

Replaces all the instances that match the text in the Find What field with the text in the Replace with field.

Match Case

Selecting this checkbox enables the utility to distinguish between uppercase and lowercase characters. It then searches only for text that has the same uppercase and lowercase characters as in the Find What field.

Search only translations

Selecting this checkbox enables a text search only in the Translations column
