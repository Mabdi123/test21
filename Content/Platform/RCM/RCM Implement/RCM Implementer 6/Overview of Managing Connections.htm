Overview of Managing Connections

Authorized IT professionals can set up connections for retrieving data from various sources (e.g. connections to data sources that provide Reference information, Thresholds, Trade Blotter data, etc.). This is done using the Connections tab within the RCM Admin Site.

The information in the Connections tab depends on the ConnectionManager option that is currently configured within the actimize_applicationContext.xml file (see your RCM Installation Guide for more information about the ConnectionManager and the actimize_applicationContext.xml).

The Connection Manager, in this version, can be set to DBCPConnectionManager or to WebLogicConnectionManager (using jndi).
