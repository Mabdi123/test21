SMTP Parameters

For SMTP support (sending mail notifications) edit the following configuration parameters:

  • mail.smtp.adminMail – The email address for sending administrative mails. This is the Email that will be presented as the sender's email address - mandatory for sending mails.
  • mail.smtp.host – The IP or hostname of the SMTP server - mandatory for sending mails.
  • mail.smtp.password – The password of the SMTP server.
  • mail.smtp.user – The username of the SMTP server.

Note that the last two parameters are relevant only when the user who started the application server does not have the required permissions to the SMTP server.
