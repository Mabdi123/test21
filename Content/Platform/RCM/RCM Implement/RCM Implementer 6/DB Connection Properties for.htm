DB Connection Properties for RCM DBCP

If you are using a JNDI connection then these lines remain with the comment characters

If you are using an SSL connection with an Oracle DB then you should leave this section commented. Remove the comment characters from section “(4) Enable SSL protocol for Oracle connections” and enter the appropriate values for the oracle.jdbc.ssl parameter.

Name

Description

actimize.repository.defaultCatalog

The name of the database (also referred to as the catalog) that is being used. For example, if the name of the database is ‘northwind’ then enter:
actimize.repository.defaultCatalog = northwind

actimize.repository.username

The username for the database that is being used. For example, if the username is sa then enter: actimize.repository.username = sa

actimize.repository.password

The encrypted password for the database that is being used. In order to find the encrypted result of a password you must use the Encryption Tool: run_encryptor.bat. For more information on run_encryptor.bat, refer to the RCM Installation Guide. For example, if the password is left blank then enter: actimize.repository.password = JbZhuHIo2KVYACuJ.

For more information on using an external credential provider for this value, refer to Encrypt values for External Configuration Files in the RCM Installation Guide.
