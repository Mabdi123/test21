Related Configuration Parameters

  • actimize.platformLists.multiRecordsEditMaxBatchSize - an integer indicating the number of records that are included in each batch when saving multiple items. Default value is 1,000.For example, if the user edits 5,000 items and the value of the parameter is 1,000 then the items are saved to the database in 5 batches where each batch saves 1,000 items.
