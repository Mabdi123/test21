Assign Group Permissions

Group permissions are defined in the Update Role window in RCM Administration.

These specific permissions are defined in the Sensitive Data bar. Select the Groups that users with this role can view. For example, If user A has permissions to view the AML officers group then they will be able to view all the sensitive data of this group unmasked (e.g., field that includes a credit card number shall display the entire number to the user).
