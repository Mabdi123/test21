v_acm_alert_notes

Description: Returns the notes for all Alerts, with the following information for each Note:

Field Name

Description

alert_join_id

The Join ID to the Alert to which this note is added (v_acm_alerts2)

note

The Note text

audit_join_id

The Join ID to the related audit record associated with this note (v_acm_alert_audit2)

user_join_id

The Join ID to the user who added the note (v_acm_users2)

create_date

The date when the note was added
