Move Objects for Export

  • To move an object for export, select the object that you want to move and click the icon. The selected object appears at the right pane of the wizard screen.
  • To move all objects for export, click the icon. All the objects appear at the right pane of the wizard screen.
  • To move back an object from the selected objects, select the object and click the icon. To move back all the selected objects click the icon.
