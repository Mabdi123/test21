v_acm_ava_user_view_permission

Description: Provides the flattened permissions of AVA users, along with the flattened BU hierarchies involved. That is, it restricts what AVA users can access via their BU, item type, and assignment permissions.

Field Name

Description

user_join_id

The numeric ID of the AVA-roled user

user_identifier

The identifier of the AVA-roled user

bu_join_id

The numeric ID of the business unit to which this permission entry applies

-100 means all BUs

item_type_join_id

The numeric ID of the alert type to which this permission entry applies

-1 means all alert types

item_assignment

The numeric representation of the required assignment relationship between the user and the defined work item.

0: unassigned work items

1: work items assigned to the user in question

2: work items assigned to other users

3: all of the above
