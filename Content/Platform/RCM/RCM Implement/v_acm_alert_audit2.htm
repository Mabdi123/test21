v_acm_alert_audit2

Description: Returns the audit trail for all actions on all alerts (items), with the following fields for each action.

Field Name

Description

audit_join_id

The Join ID for the audit record of all actions performed on the alert (this join ID is typically used to join to Notes).

alert_join_id

The Join ID for the alert associated with the Audit record (v_acm_alerts2)

alert_action

Audit history for all actions performed on the alert (e.g., Change Step, Assign to User).

action_date

The date and time when the action was initiated

user_join_id

The Join ID to the user who initiated the action (v_acm_users2)

action_text

Additional information associated with an action, e.g., information about a Change Step action includes the old and new step names and the user who changed the step

original_identifier

Used in transient consolidation and merge feature, for storing the deleted alert identifier
