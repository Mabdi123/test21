Cases Configuration

These configuration settings enable you to maintain different tables that define full lists of Case Types’ attributes. These attributes can later be assigned to a specific Case Type, Case View and to the Advanced Filtering window. For example, you can add, edit, and delete notes in the Predefined Notes Page and then assign some of the notes to a specific Case Type.

To setup the Cases configuration:

  • Click the Cases bar and then click the Configuration tab.

The pages available within the Cases Configuration menu are:

  • Case Steps List – Define and edit the list of Case Steps.
  • Cases Predefined Notes – Create and edit notes that can be attached to cases.
    Cases Fields:
  • Built in fields – Customize the display titles of the cases’ permanent fields.
  • Custom Fields – Define and edit customized fields to be used in cases.
  • Cases View Sorting – Define the order of the Case Views as they appear in the Cases menu.

Fields within the list that are grayed out cannot be changed. For each field that can be edited (i.e., that appears in black text) you can perform the following:

  • Change the Column Header Display Name – This is the name that appears within the RCM Case table for this field.
  • Set the width of the column – You can select the width size and the way in which the width is measured (e.g., in pixels or in percentage).
  • Indicate whether you want this column to be visible (i.e., to appear within the Cases table)
