Alerts View Sorting

This list enables you to define the order of the alert (work item) views as they appear in the Alerts menu.

Select Alerts View Sorting to sort views in the RCM.

For example, if My Items view is sorted as the top view and My BU Items is sorted after it then the Work Items menu appears in the following way: 

The icons available within the Alert Views Sorting window are as follows:

Icon

Description

Enables you to move down a selected alert view.

Enables you to move up a selected alert view.

To change a position of an alert view:

  • Select the alert view you want to move.
  • Click the or icons to move the alert view.
