Upgrade Tool GUI Items

A customized tool GUI Item is upgraded via the Copy to Action action. The following example illustrates the steps for upgrading a customized tool GUI Item.

  1. The following screenshot displays a customized GUI item for the Work Items (HO_5501_Tool_WI).

  1. The GUI item of this action is selected and the action Copy to Action is selected.

The Parent Item Custom Actions is selected for this customized GUI item. This parent item is an out of the box option that does not have to be predefined.

  1. The original GUI Item's visibility is disabled by clearing the Visible checkbox of the GUI Item attributes screen.

  1. Click Finish. The action now appears as an icon of the Work Items toolbar.
