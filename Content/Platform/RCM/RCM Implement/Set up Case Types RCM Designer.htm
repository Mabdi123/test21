Set up Case Types – RCM Designer

The RCM Designer enables implementers to set up the Case Types that are used within the RCM Application.

Setting up a Case Type requires the following tasks:

  • Providing a name, description, and ID for this Case Type.
  • Defining attributes of Case Type by selecting Custom Case fields.
  • Designing the layout of the Case Type Custom fields when the case details are viewed.
  • Assigning the Predefined Notes and steps for the Case Type.
