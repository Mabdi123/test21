Remove Meta Data Resource Strings

To remove RCMMD module resource strings, click the Remove button. A window appears:

By default, the resource strings are grouped by the following columns: Entity, Type and Item name. You have the option to group the resource strings by additional columns. This is achieved by dragging a column heading into the grouping panel. Select the specific resource strings that you want to remove by checking the checkbox in the Remove Resource String column. For example, in this screenshot a specific case type is selected:

Additional options in this wizard window include:

- Show/Hide Grouping panel

- Check all values – All resource strings are checked

- Check selected rows – The highlighted resource strings are checked

- Uncheck all values – All resource strings’ checkboxes are cleared

After you have selected all the resource strings that you wanted to remove, click Next. A summary screen displays all the selected resource strings. You have the option to group the resource strings by the value column. After you have reviewed the resource strings that have been selected, click Remove.

The RCMMD resource strings that have been selected are removed from the main list of resource strings. All other language lists are then synchronized with the updated base language resource string list.
