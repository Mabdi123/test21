Create the Action GUI Item Type

In the GUI Items wizard you can create a GUI item based on the Action type.

The GUI Item Attributes page for the Action item type contains the following fields:

Field

Description

Module

Select the Module that you want this GUI item to be added to (e.g. Lists, Thresholds, etc.)

Name

Enter a name for the GUI item. (For example, Change Step for the action item performChangeStep().)

Tooltip

The text that will appear as a tooltip for the item.

Type

Select the relevant action Type:

  • Action: An action icon that runs a specific javascript call when the user clicks on it.
  • Separator: A visual indicator that separates between icons or menu items. When you select a Separator the icon and script fields do not appear. A name is required for the identifier of the separator and the tooltip field can be left blank.

Scope

Select the screens where the action appears:

  • All: The action appears in both list and details screens of the module.
  • Details: The action only appears in details screen of the module.
  • List: The action only appears in the list screen of the module.

Icon

Select an icon from the icon picker. It is recommended to select an icon that has the same name as the action. For example, selecting add_attachment.png for an action named Add Attachment.

Script

Enter the relevant JavaScript based function for the action.

Selection

This property provides the ability to display the action as disabled until the user selects an item. The following options are available:

  • Any: No selection required. The action is always displayed as enabled regardless of item selection.
  • Single: Only a single item selection is required in order to enable the action. Selection of more than one item disables the action.
  • Multiple: One or more item selection is required in order to enable the action. Select the visible checkbox in order for the item to be displayed.

Visible

Select to make this item visible in the RCM display. Only users with relevant permissions can view displayed GUI items.
