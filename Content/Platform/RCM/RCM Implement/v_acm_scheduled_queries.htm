v_acm_scheduled_queries

Description: The view that describes the scheduled query.

Field Name

Description

scheduled_query_join_id

The scheduled query join ID that is used to join it to other views (v_acm_tasks)

Identifier

The scheduled query identifier

Name

The scheduled query name

Description

The scheduled query description

where_clause

The SQL ‘where’ clause

src_eds_join_id

The external data source identifier used for executing the query.

target_eds_join_id

The query results are written to this external data source identifier

original_eds_join_id

The external data source the query was built on.

period_from

The period’s earliest date and time

period_to

The period's latest date and time

owner_join_id

The creator’s Join ID that is used when joining to v_acm_users

create_date

The date the scheduled query was created

last_execution_date

The last time the query was executed

e_schduling_mode

The Scheduling mode (Daily, Weekly, etc.)

alert_type_join_id

The alert type Join ID that is used to create the DART alert.

Score

The score value the user wants to give to the DART alert

bu_join_id

The BU Join ID that is assigned to the DART alert.

end_date

The final date and time that the query can be executed.

day_of_week

The day of the week that the query should be executed when the mode is Weekly

day_of_month

The day of the month that the query should be executed when the mode is Monthly

query_identifier

The Identifier of the query that was used to generate the scheduled query

start_date

The day this scheduled query should start being a part of the external data source process.
