Export Tab

In this tab, you define what XSL will be used in order to render the alert as a PDF.

The format used for PDF generation is XSL-FO, thus the XSL needs to perform a transformation to that syntax. An example can be found in the RCM Self Developer Guide.

To specify the XSL location:

If you specify that the alert details are retrieved using an XML, then you must specify the XSL location.

  • Enter the XSL file name that you want use for exporting the details.

You can also select the Use global VTL checkbox to enable adding generic details, such as item history and attachments to the PDF as an additional page.
