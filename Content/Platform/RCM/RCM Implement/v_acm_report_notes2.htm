v_acm_report_notes2

Description: Returns all the report notes, with the following information:

Field Name

Description

report_instance_id

The Identifier for each report instance (v_acm_report_instances2)

create_date

The date when the note was added

user_join_id

The Join ID to the user associated with the note (v_acm_users2)

note_text

The Note Text

segment_identifier

The identifier for the part (segment) of the report to which this note refers.
