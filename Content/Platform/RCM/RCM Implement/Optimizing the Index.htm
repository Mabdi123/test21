Optimizing the Index

To optimize the index, open this URL in the browser:

http://<host name>:<port>/solr/update?optimize=true

Example

http://localhost:9090/solr/update?optimize=true

An optimization command needs to be executed twice. When optimizing the index the first time, Solr needs double the disk space. E.g., if the index size is 200MB then after the optimization it will be 400MB, because Solr temporarily copies the files.

Opening the above URL again will cause the extra files to be deleted, and the index size will be approximately 200MB again.
