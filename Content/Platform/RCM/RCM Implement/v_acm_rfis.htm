v_acm_rfis

Description: Displays the list of RCM RFIs.

Field Name

Description

rfi_join_id

The join id of the RFI.

rfi_identifier

The identifier of this RFI.

create_date

The created date of this RFI.

rfi_title

The title of the RFI.

rfi_guid

Unique ID used to identify messages belonging to the RFI

owner_join_id,

The internal ID of the owner

fl_has_unread_messages

Indicates whether or not the RFI has unread messages.
