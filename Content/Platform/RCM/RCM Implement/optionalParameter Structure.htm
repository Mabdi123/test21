optionalParameter Structure

The optionalParameter object is a standard JavaScript object defined as a set of name-value attributes (the order of the attributes is ignored).

For example, the following code defines a JavaScript object with two properties – a and b – where a is the integer value 1 and b is the string value “hello”:

var obj = {a: 1, b: “hello”}

Similarly, the following code is an example of a valid optionalParam object for the Work Items API(the parameter “showResetButton” was omitted, so the widget drawn with these settings would have used the default value of “showResetButton=true”):

var optionalParam = {

viewIdentifier: “allAlerts”,

ddqIdentifier: “alertsByIdRange”,

ddqParams: [4, 7],

actionsGroupIdentifier: “alertWidgetActions”,

pagingSize: 6,

selectionMode: “single”,

showRefreshButton: false

}
