The XSL API

The XSL API is basically an <xsl:value-of> node that contains a custom RCM XSL function for generating the widget. The purpose of this API is the following:

  1. Generate the widget object on the server
  1. Write JavaScript code into the HTML document for manipulating the widget on the client.

The select attribute of the XSL node consists of a function call with arguments that supply the RCM server with the necessary settings in order to create the widget.
