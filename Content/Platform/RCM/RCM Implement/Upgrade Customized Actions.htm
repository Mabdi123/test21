Upgrade Customized Actions

To upgrade a customized action you should change its parent Action Group. The following example illustrates the steps for upgrading an action.

  1. The Get Next - Grouping action link is initially displayed on the left pane of the Work Items module.

  1. Select the relevant GUI item is selected and click the Edit GUI Item link.

  1. The Parent Item Custom Actions is selected for this customized action. This parent item is an out of the box option that does not have to be predefined.

  1. Click Finish. The action now appears as an icon of the Work Items toolbar.

See also Select the Parent Item, page 1.
