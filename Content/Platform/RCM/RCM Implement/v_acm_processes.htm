v_acm_processes

Description: This view is used for monitoring of scheduled query processes

Field Name

Description

process_join_id 

The process Join ID (needed for v_acm_tasks)

eds_identifier

The external data Source identifier (log) used by this process

reference_date

The data used by this process as a reference for calculating which scheduled query runs and is also embedded in the query.

start_date

The date and time the Process started

end_date

The date and time the Process ended

Step

The Process step.
