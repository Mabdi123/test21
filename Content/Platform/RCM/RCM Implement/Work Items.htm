Work Items

The Workbench tab, contains the sub-tabs Work Items (Alerts), Blotters (if licensed) and Reports. Under Workbench, views of different categories of work items can be displayed such as My Work Items, All Work Items, Documents and any user-defined categories that are required at your site. See Create a Customized Work Item for instructions on creating a customized work item. Note that if the default set of tabs and subtabs is not compatible with your organization’s terminology and usage of RCM, then the labels on tabs are still customizable.
