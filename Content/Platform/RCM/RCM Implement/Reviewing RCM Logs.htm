Reviewing RCM Logs

The logs can be reviewed via the Log Viewer. To see the Log Viewer, enter the LogViewer.jsp page by changing the URL to <URL Root>/LogViewer.jsp.

The log view and other actions can be controlled from the menu filters and buttons:

  • Use the Show last drop-down menu to select the number of most recent records to display.
  • To update the displayed results, click Refresh.
  • To set the in-memory log to store and display the desired level of messages, select the level from the Set the log level to drop-down menu.
  • To apply the new setting, click Set.
  • You can enter a free text value in the Filter field to dynamically filter the displayed results.
  • To remove the free text filter, click the button next to the Filter field.
  • For the list of diagnostics, click the Diagnostics button to display the Diagnostics window.
  • Clicking Create Report creates a spreadsheet file with diagnostic information and log entries.
  • Clicking Clear Cache clears the hibernate cache.
