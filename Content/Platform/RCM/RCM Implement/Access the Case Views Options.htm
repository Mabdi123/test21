Access the Case Views Options

To access the tools and options for the Case View:

  • Click the Views tab of the Cases bar in the Entities pane.

The icons available within the Entities pane for this task are as follows:

Icon

Description

Displays four tabs on the right side of the screen that enable you to define a new case view. The Case Types tabs are:

  • General – Enables you to add a name, description, and identifier for the new definition.
  • Fields – Enables you to define the viewable fields and to format their display.
  • Default Sort – Enables you to define the default sorting of the viewable fields for this Cases View
  • Filter – Enables you to define contents of this Case View by editing the ‘where’ of the query on the cases.

Selecting a Case View within the list and clicking this icon opens the Field page on the right side of the screen, enabling you to edit the details of the selected Case View.

Deletes a selected Case View. You must close any page associated with this Case Definition before you can delete it.

Duplicates the selected Case View.
