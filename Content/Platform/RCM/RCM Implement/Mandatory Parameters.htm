Mandatory Parameters

Name

Description

actimize.repository.url

The repository connection string. Remove the comment characters (##) from the connection string that you want to use. For more information refer to the RCM Installation Guide.

actimize.repository.type

Remove the comment characters (##) from the Repository Type that you are using. This is determined according to the type of database server that is used (e.g., Oracle, MS SQL server or Sybase).

For example, if you are using an MS SQL server then the comment characters should be removed from the following:

actimize.repository.type = sqlserver
