Workflow General Tab

Select an item workflow in the Step Workflow tab of the Entities pane.

The following options are available in the workflow General tab.

Field

Description

Name

Enter a name for the workflow.

Identifier

Enter the alert/item workflow identifier.

Description

The workflow description. You can access a larger text area for the description by clicking the Edit button.

Global Deadline section – configure the workflow deadlines for the following features.

NOTE:

Business days are calculated according to the Business Calendar – Days and Business Calendar – BU Associations OOTB platform lists in the RCM UI. For details, see Business Calendar Lists.

Send Email Notification

Enables you to indicate the number of Business Days/Days/Hours/Minutes before the deadline in which an email is sent to the owner of the alert.

The options for email are:

  • Standard email, for example:

  • RFI template based email. For information, refer to the RCM Reference Guide, in Managing RFI Templates.

You must enter the relevant RFI identifier in the actimize.workflow.deadline.defaultRFITemplate config parameter. For details, see Parameter Descriptions.

Create Reminder

Enables you to indicate the number of Business Days/Days/Hours/Minutes before the deadline in which a reminder about the deadline is displayed to the user.

Change Step to

Enables you to select the step of the alert after the deadline is reached. The required escalation step needs to be added to the workflow first. The selection of the escalated step creates an automatic Transition between the currently in focus step and the selected escalated step. Note that the Allow manual transition checkbox for the transition arrow is enabled when this parameter is defined.
