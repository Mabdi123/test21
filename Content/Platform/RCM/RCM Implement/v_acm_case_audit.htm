v_acm_case_audit

Description: Returns audit history for each Case with the following information:

Field Name

Description
audit_join_id

The Join ID to the audit record
case_join_id

The Join ID to the Case associated with this audit (v_acm_cases)
case_action

The audit history for all actions performed on the Case

action_date

The date and time when the action was initiated
user_join_id

The Join ID to the user who initiated the action (v_acm_users2)
action_text

Additional information associated with an action.
E.g., information about a Change Step action includes the old and new step names and the user who changed the step
