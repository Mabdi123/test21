Create a New Case Workflow and Assigning General Information

The following enables you to create a new case workflow.

To create a new case workflow:

  1. Click the Cases bar and then click the icon.
  2. The screen on the right side changes to include two tabs: General and Workflow tabs.
  3. Using the options within the General tab, provide a name for this workflow.
  4. Enter the Case Workflow Identifier.
  5. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field box.
  6. You have the option to define the Global Deadline definitions. The Global deadline occurs when a case is not in a specified Step after a defined amount of time.

Configure the following options:

  1. Select the checkbox Workflow reaches global deadline to enable this feature and enter the number of days that must pass after the Case is opened. The case will automatically move to a specified step after the defined number of days.
  1. In the Send email notification checkbox define when an email notification will be sent to the user. The email informs the user that there are only a specific number of days before the global deadline occurs.
  1. In the Create Reminder checkbox define when a reminder will be displayed to the user. A Reminder is an indication on the instance of the case’s data (e.g., indication on the case details on the deadline’s occurrence). The reminder informs the user that there are only a specific number of days before the global deadline occurs.
  1. Select the Change Step to checkbox and indicate the workflow’s step after the global deadline occurs.
