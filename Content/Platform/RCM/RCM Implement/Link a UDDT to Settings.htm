Link a UDDT to Settings

You can define User Defined Data Types (UDDT) as a picker or a drop-down list, see Configuring User Defined Data Types. After configuration, you can link the UDDT type to a specific setting field.

To link a UDDT to a setting:

  1. Go to admin.jsp and click the Settings tab.
  2. Select the relevant setting and click the New Setting or Edit Setting link. The relevant window opens.

For details on editing or defining settings see Setting Actions.

  1. Click Next. The Fields definition page opens.

  1. In the Validation section, select the predefined UDDT from the Input Type field for each Display field and click Save.

Make sure the selected UDDT input type (Char, Numeric) is appropriate for the actual setting field type in RCM.

In this example in the RCM UI, the continent display field has a UDDT (drop-down list) linked to it (countries_continents_uddt).
