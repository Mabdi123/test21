Create a New Tuning Context and Assigning General Information

The following describes how to create a new Tuning context.

To create a new Tuning Context:

  1. From the Entities pane – Tuning Contexts, click the icon. The screen on the right side changes to include three tabs: General, Execution and VTL Display tabs.
  1. Using the options within the General tab, provide a name, identifier and an optional description for this Tuning Context.
  2. Select one or more Setting Navigation Trees to map to the Tuning Context. You have the option to click the Refresh button in order to view an updated view of the available trees. For more information on creating Setting Trees that are enabled for Tuning Context, see also Create a New Navigation Tree and Assigning General Information.
  3. Select the type of VTL definition:
  1. Embedded – Enter the VTL code via the VTL tab.
  1. From Virtual File System – Select a predefined VTL file. When this option is selected the VTL tab cannot be used and is greyed out.
