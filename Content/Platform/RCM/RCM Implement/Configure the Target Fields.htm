Configure the Target Fields Table

Target fields are the built-in and custom fields of the CWI. For a relationship, they are the “target” of the data mapping, as data from the related TWIs is aggregated into those fields.

Add target fields to the table via one of the following options:

  • Click the Add New Target Field icon.
  • Click the Add all Mandatory Fields icon. All the custom fields that must be populated by RCM user appear in the table.
  • Click the Add all available Fields icon. All the custom fields appear in the table.
