Overview of Migration

RCM Designer provides you with the option to export and import objects:

Export of RCM Designer and Admin objects in order to import them later on from a different server or from an updated version.

Import RCM Designer and Admin objects in order to preserve application definitions or reuse objects such as alert types, case types, case views, etc.

The Designer provides you with these import options:

  • Import all objects that have been previously exported
  • Select to import specific objects that have been previously exported.
