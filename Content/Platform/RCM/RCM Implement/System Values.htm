System Values

These values are automatically filled in the selected parameters by the system during the report generation. For example, a date/time parameter may be defined for the date value of the Report Generation Date. In the following screen shot the parameter p_date_to with the display name Date To is provided with the default value of the Report Generation Date.

Therefore, the date value of the parameter p_date_to is defined when the RCM user generates the report. For example, if the user generated the report on the date June 26, 2006 then the value 6/26/06 is provided for the parameter p_date_to.
