Access the Report Definition Options

To access the tools and options for the Report Definitions:

  • Click the Queries bar and then click the Reports tab of the Entities pane.

The icons available within the Entities pane for this task are as follows:

Icon

Description

Displays three tabs on the right side of the screen, enabling you to define a new Report. The Report Definition tabs are:

  • General – Enables you to enter general information for this report
  • Report Provider Data – Enables you to select the relevant files according to the Report Provider that is indicated (e.g., cat and cls files).
  • Report Parameters – Enables you to map report parameters.

Selecting a Report Definition within the list and clicking this icon enables you to edit the details of the selected Report Definition.

Deletes a selected Report Definition.

Creates a copy of the Report Definition that is currently selected within the list. The copy is assigned the name “Copy of <name of selected Report Definition>’

Toggles the display to show or hide the Display Categories. When pushed in, the display categories appear within the list. When released, the list of currently defined reports displays in a flat (uncategorized) list.

Refreshes the list of Report Definitions.

Toggle the display of the identifiers of the Reports

Opens the Display Category window that enables you to add a new category or edit a selected category.

Opens the Policies window that enables you to define policies that are available when setting up the Report Definitions. These policies set the rules for actions such as when the report is materialized, archived, purged etc. The system comes with a default set of policies, and you can add as many policy sets as are required by your organization.

Opens the Generate Report Wizard that enables you to test any report that is defined within the RCM Designer.

Exports the selected Report Definition to an .arp file. The importreport.bat utility, which is provided on your RCM Installation CD (under the Utilities/ReportMigrationTool folder), enables you to import an exported report into a different environment.
