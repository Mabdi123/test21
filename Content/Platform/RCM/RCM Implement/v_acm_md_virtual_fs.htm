v_acm_md_virtual_fs

Description: DB View that shows the content of the virtual file system.

Field Name

Description

virtual_fs_join_id

The join id of this virtual file.

virtual_fs_identifier

The identifier of this virtual file.

content

The content of this virtual file.

owner_join_id

The join ID of the owner of this virtual file.

create_date

The create date of this virtual file.

last_update_user_join_id

The join ID of the user that last updated this virtual file.

last_update_date

The last update date of this virtual file.

content_size_in_bytes

The size of this virtual file.
