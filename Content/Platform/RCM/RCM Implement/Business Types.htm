Business Types

A Business Type is a category that enables you to group Custom Fields in order to allow the RCM end users to search for an entire category or specific field(s). The Business Type and its Custom Fields appear in the Advanced Filter window. Custom Fields are the additional fields, which may appear in a specific alert view or case view and in the Advanced Filter window. (Each view can be composed of Built In Fields and Custom Fields). You can remove, add or modify Custom Fields. See Alert Fields - Custom Fields.

For example: The Silver Accounts Business Type is associated with Custom Fields Silver Standard and Silver Plus.

The end user can later select to filter all silver accounts or a specific silver account via the Advanced Filter window.

Use this list to define the various types of Business Types, which can later be associated with specific Custom Fields(s).

The options that are available within the Business Types page are:

Option

Description

Name

Business Type Name.

Identifier

Enables you to enter a reference ID for this Business Type, which later can be identified by an external application.

Data Type

Select from, String, Integer, Real, Date, and Datetime (Date and time).

Description

Business Type description.

Has Range

True – User will view two fields that allow entering data for a specific range.

False – One field appears for end user.
