v_acm_item_snapshots

Description: Returns the snapshots for all items, with the following information for each snapshot:

Field Name

Description

item_snapshot_join_id

The snapshot internal Join ID

alert_join_id

The Join ID to the Item to which this snapshot is belongs (v_acm_items)

child_snapshot_xml

a snapshot of the xml data of the related child item, taken when the relation was created

create_date

Snapshot creation date
