Save the Selection for Export

The export wizard provides you with the option to save the objects you want to export to a file and then load them later on. This option is useful when a newly created object needs to be exported together with additional objects that were exported previously. Instead of selecting all the objects again with the newly created object, you can select to load the file that defines these previously defined exported objects and then only select the newly created object with it.

  • To save the objects that are selected for export to a file, click the icon. This file can be loaded in a future export procedure by clicking the icon and selecting the previously saved file.
  • After you have selected the objects for export click the Next button. This action moves you to the Export File Path screen
