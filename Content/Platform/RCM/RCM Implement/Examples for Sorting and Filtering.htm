Examples for Sorting and Filtering

Example 1: Selects only CWI items that have an Open State and Issue Finding.

Combining this with the sort, which is Create Date (desc.), indicates that RCM will choose the most recently created CWI that has an Open and Issue status. The example assumes that there are several CWIs that are candidates for the specific TWI item.

Example 2: Selects only CWI items that are assigned to the “id All” BU, and have a score between 80 and 90. The items were created within the last 8 days.
