Sort the Items

The last screen of the wizard enables you to select the order of the actions in the group. Select a specific item and click the up and down arrows in order to move the item in the list.

If the new item is a menu (tab) this determines the position of the tab, where the item at the top of the Order items list is the menu that appears on the far left side of the window.

If you are adding a new action, the order determines the position of this item within the list, and if you are adding a new tool, the order determines the position of the button from left to right.

Click Finish to save and apply changes.
