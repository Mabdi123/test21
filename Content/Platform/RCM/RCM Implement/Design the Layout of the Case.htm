Design the Layout of the Case Type

This window enables you to set the Case Type Custom fields that have been defined previously. (see Add Custom Fields to the Case Type).

Example of a Case Type layout can be viewed in the screen shot below. Note that the three Custom Type fields (Country, Currency, and region) are displayed according to the way they were laid out in the Designer.

The icons available within the Case Type Fields Layout window are:

To insert a Case Type Custom Field inside the layout:

  • Select a field and drag it to the desired layout cell.

  • Alternatively, click the Auto populate icon to insert all fields into the layout automatically.
