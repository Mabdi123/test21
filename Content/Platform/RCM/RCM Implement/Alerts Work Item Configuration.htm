Alerts (Work Item) Configuration

These configuration settings enable you to maintain different tables that define full lists of Alert Types attributes. These attributes can later be assigned to a specific Alert Type, Alert View and to the Advanced Filtering window. For example, you can add, edit, and delete notes in the Predefined Notes Page and then assign some of the notes to a specific Alert Type.

To set up Alert Configuration:

In the Configuration bar, select Alerts Configuration and then select an item.

For example, select Alert Step List.

Alternatively, click on the Alerts bar in the left pane, select the Configuration tab. and then select an item.

For information regarding the Relation Types configuration, see Define the Relation Types.

Fields within the list that are grayed out cannot be changed. For each field that can be edited (i.e., that appears in black text) you can perform the following:

  • Change the Column Header Display Name. This is the name that appears within the RCM Alert table for this field.
  • Set the width of the column. You can select the width size and the way in which the width is measured (e.g., in pixels or in percentage).
  • Indicate whether you want this column to be visible (i.e., to appear within the alert table).

The pages available within the Alerts Configuration tab are as follows:

  • Alert Steps List – Define and edit the list of alert steps.
  • Alert Predefined Notes – Create and edit notes that can be attached to alerts.
    Alert Fields:
  • Alert Fields - Built In Fields – Customize the display titles of the alerts’ permanent fields.
  • Alert Fields - Custom Fields – Define and edit customized fields to be used in alerts and cases.
  • General Configuration - Value Lists– Define and edit the various types of value lists, which can later be associated with specific custom fields.
  • Alert Score Ranges – Define the score ranges that reflect the severity of alerts.
  • Alerts View Sorting – Define the order of the alert views as they appear in the alerts menu.
