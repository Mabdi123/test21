Configure Encryption for Email Collaboration

The SSL/TSL certificate must be added to RCM’s trust store. Create a keystore (.jks) file and add it to RCM via the following configuration parameters:

  • actimize.authentication.truststoreLocation – The path to the .jks file.
  • actimize.authentication.truststorePassword – The password for the key-store (to access the jks file).
  • actimize.authentication.truststoreType – The type of keystore (generally JKS).

Refer to the RCM Installation Guide for more information on 15BSecure Socket Layer (SSL) connections.
