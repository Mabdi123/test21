VTL

convertToUserDateTimeFormat(dateValue, sourceDateFormat)

Converts a given date value from the input date format to the user expected date format.
The second parameter is optional. If it is not present, format is yyyy-MM-dd HH:mm:ss.

Usage:

$vtlUtils.convertToUserDateTimeFormat(“2012-06-27 15:28:00”)

Or

$ vtlUtils.convertToUserDateTimeFormat(“27/06/2012 15:28:00”, “dd/MM/yyyy HH:mm:ss”)
