Set Up Alert Type Relations

Setting up Alert (Item) Type Relations includes the following steps:

  1. Setting up the relevant alert type (person, incident, etc.) and defining whether or not an alert type is a child or parent item.
  1. Defining the fields of the Cache XML for the child alert types
  1. Defining the views of all child alert types
  1. Defining the relation types.
  1. Defining the relation attributes for the alert type
  1. Adding VTL code in the relevant form and selecting that form as the custom layout form of the item type (alert type).
