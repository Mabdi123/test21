v_acm_md_customized_fields

Description: A custom fields view

Field Name

Description

customized_field_join_id

The internal join ID of the customized field

field_join_id

The field internal Join ID

customized_field_identifier

The identifier of the customized field

display_name

The display name of the customized field

meaning_join_id

The join ID of the field meaning

fl_range

Does this field have a range

description

Description of the customized field

e_module

Indicates to which Module this customized field belongs (i.e. Alerts, Cases, etc.)

svl_join_id

The join ID of the suggested value list

column_path

The location of the field in the source Table or DB view

e_data_type

The data type of the field (e.g. Integer, String, Date)

picker_join_id

The internal join ID to the type of picker used for this customized field

fl_common

It is a common field

field_size

The size limit of the field in characters

display_order

The field display order as defined in its item type

sub_module

Secondary module (Blotters and Transactions only)

structure_xml

The layout of the customized field when rendered in Policy Manager's expression builder.

fl_set_field

Is it a set field (Outgoing Forms only)

sd_masking_join_id

The internal join ID of Sensitive Data Masking definition.
