Create a New Case Type and Assigning General Information

The following enables you to create a new workflow or assign steps.

This feature is available as part of the ERCM license.

To create a new Case Type:

  1. From the Types tab click on the icon.
  2. The screen on the right side changes to include five tabs: General, Case Type Custom Fields, Case Type Fields Layout, Assigned Predefined Notes, and Assigned Steps tabs.
  3. Using the options within the General tab, provide a name for this hierarchy.
  4. Provide a description, if required. You can access a larger text area for the description by clicking the Edit button beside the Description field box.
  5. Enter a unique ID for the case. This ID represents the Case Type within the Case -type table of the RCM database.
