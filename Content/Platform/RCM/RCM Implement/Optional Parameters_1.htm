Optional Parameters

The structure of the optional parameters is identical to the XSL API of the Records widget. The default values of the optional parameters are:

  • actionsGroupIdentifier – No actions are displayed.
  • preventNewConditions – true.
  • showRefreshButton – true.
  • showResetButton – false.
  • disableFilterByExample – true.
  • filterDisplayMode – tableAlways.
  • selectionMode – none.
  • showBaseQueryConditions – false.
  • autoShrinkHeight –false.
  • hideGridHeader – false.
  • secondaryLayoutDisplayMode – collapsed.
  • stretchGridWidth – false.
  • adjustSecondaryLayoutWidth – false.
