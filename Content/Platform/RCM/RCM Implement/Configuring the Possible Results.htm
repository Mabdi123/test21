Configuring the Possible Results

The following heartbeat related parameters are available in the Config Parameters tab of the RCM admin.jsp page:

  • actimize.general.heartBeatOK – The response string of heartBeat.jsp when the RCM and the RCM repository are up and running.
  • actimize.general.heartBeatFail – The response string of heartBeat.jsp when the database (RCM repository) is down or not connected properly. This string value can be edited.
