XSL API

Exposes the option to convert date value to XsltExtensionsFunctions:

convertToUserDateTimeFormat(dateValue)

convertToUserDateTimeFormat(dateValue, sourceDateFormat)

The XSL must contain ‘xmlns:actimize’ in the header (see example for more details)

Example:

<?xml version="1.0" encoding="UTF-8" ?>

- <xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform" xmlns:actimize="com.actimize.infrastructure.util.XsltExtensionsFunctions">

- <xsl:template match="/">

- <html>

- <xsl:for-each select="/enrichedAlert">

- <table width="100%">

- <tr>

- <td colspan="2">

- <table width="100%">

- <tr>

- <td colspan="4" bgcolor="#BBFFFF">

  <b>Alert Attributes:</b>

  </td>

  </tr>

- <tr bgcolor="#C2C2C2">

- <td width="15%">

  <b>Name:</b>

  </td>

- <td width="40%">

  <b>Value:</b>

  </td>

- <td width="20%">

  <b>Date:</b>

  </td>

  </tr>

- <xsl:for-each select="attributes/attribute">

- <tr>

- <td>

  <xsl:value-of select="@name" />

  </td>

- <td>

  <xsl:value-of select="@value" />

  </td>

- <td>

  <xsl:value-of select="actimize:convertToUserDateTimeFormat('2012-07-31 11:00:00')" />

  </td>

  </tr>

  </xsl:for-each>

  </table>

  </td>

  </tr>

  </table>

  </xsl:for-each>

  </html>

  </xsl:template>

  </xsl:stylesheet>
