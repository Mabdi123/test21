v_acm_item_attachments_and_content

Description: Returns the attachments and content for all Items, with the following information for each attachment

Field Name

Description

item_join_id

The Join ID to the Item to which this attachment is added (v_acm_items)

file_name

The file name attached to item

file_original_path

Full path of the item attached file

file_date

The file date attached to item

first_attach_date

The time that this file was first attached to item

user_join_id

The Join ID to the user who added the note (v_acm_users2)

attachment_type_join_id

The Join ID to the attachment type 

description

The attachment description

file_size

The attachment file size (in bytes)

audit_join_id

The Join ID to the related audit record associated with this attachment (v_acm_alert_audit2)

additional_info

Attachment additional information

file_content

Binary content of attachments
