Create a Virtual Connection

To create a virtual connection, you should initially verify that the RCM is in Multi-Tenant mode. You can verify this by checking that the value for the parameter actimize.isHostedPlatform is Yes (1). After you have verified that the RCM is in Multi-Tenant mode, do the following:

  1. Click the New Connection link and select the Virtual connection checkbox in the window that opens.
  1. Enter an identifier name for the virtual connection and add the AIS functions.
  1. Save your settings. All other fields in the window are greyed out and cannot be filled in. The ‘Test Connection’ button is disabled for virtual connections.
