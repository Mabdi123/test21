Translation Example

A Spanish language has been added in the RCM Designer and set as the current language. The following translations have been made to these RCMMD module base values:

Base Value Spanish Translation

Account

Cuenta

Item Type

Tipo de Articulo

Closed Items

Puntos Cerrados

All other base values have not been translated yet. All of the translated strings appear when a user selects to view Closed Items.

The translation of Closed Items appears when the user opens the menu of the available views from the My Items view. Base values that have not been translated remain in English (U.S.)
