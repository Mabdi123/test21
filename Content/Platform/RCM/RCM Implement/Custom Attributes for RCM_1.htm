Custom Attributes for RCM Users (User Attributes)

In addition to the out-of-the-box attributes of RCM users (such as user identifier, login name, password, e-mail), you have the option to define custom attributes.

For example, add the LOB (Line of business) attribute for each RCM user and then add it to the RCM user interface in order to be entered and modified.

Such custom attributes can be used in the security table of DART, as parameters for reports or in RCM plugins.

RCM offers a dedicated database view v_acm_user_attributes2 to be used, for example, for dashboards.
