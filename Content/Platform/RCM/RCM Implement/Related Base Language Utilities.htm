Related Base Language Utilities

The following utilizes also provide import and export actions for the base language:

  • Import Resource Strings Utility - Imports base language resource strings from a CSV file into the RCM repository
  • Import Resource Strings by Value Utility - Imports base language values as new resource strings from a CSV file into the RCM repository. Resource string identifiers are created during the import process of this utility.
  • Export Resource Strings Utility - Exports base language resource strings from the RCM repository to a zipped CSV file.

For more information on these utilities refer to the RCM Utilities Guide or to the utility readme files.
