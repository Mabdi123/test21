v_acm_form_attachments_content

Description: Returns the identifiers of form attachments.

Field Name

Description

form_join_id

The form join ID that is used to associate it to other views

attachment_join_id

Unique id of the attachment

file_name

The name of the file

file_original_path

The original path of the form attachment

file_date

The date the file was attached

first_attach_date

The date and time when the file was attached

user_join_id

The User Join ID that performed the Attachment

attachment_type_join_id

The Attachment Type Join ID

description

The Attachment description

file_size

The file size

additional_info

Stores information when used with NICE integration

file_content

Binary content of attachment
