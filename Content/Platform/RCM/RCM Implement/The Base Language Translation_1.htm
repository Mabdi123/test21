The Base Language Translation Toolbar

The buttons available within the Base Language Translations toolbar are as follows:

Icon

Description

Enables you to filter the identifier based on the string that is entered in the adjacent textbox. For example, If you enter ‘Account’ in the textbox then only rows with this string will be displayed in the list of resource strings.

Export and Import >
Export Translations

Exports the language list to a comma delimited (CSV) file. This is usually done in order to provide a translator with a simple file for entering the translations. CSV files can be opened by popular spreadsheet programs such as Microsoft® Excel®.

Export and Import >
Export Marked Translations

Exports only the selected (marked) rows of the language list to a comma delimited (CSV) file.

Export and Import >
Export Resource Strings

Exports all the Resource Strings to a comma delimited (CSV) file. These include only Resource strings that belong to other modules than the RCM module.

Export and Import >
Export Marked Resource Strings

Exports all the marked Resource Strings to a comma delimited (CSV) file. Resource strings belong to other modules other than the RCM module.

Export and Import >
Import Resource strings

Imports the language file composed of Resource Strings into the language list.

Selects all rows

Marks the selected row for export. The icon is then used for exporting only the marked rows.

Clears the selected row as marked for export

Creates a new custom Resource String. See also Add a New Custom String.

Opens a wizard that enables you to import undefined meta data resource strings that belong to the RCMMD module. See also Add Meta Data Resource Strings.

Opens a wizard that enables you to remove undefined meta data resource strings that belong to the RCMMD module. See also Remove Meta Data Resource Strings.

Deletes the resource string from the selected row. You can only delete resource strings that do not belong to the RCM and RCMMD modules.

Groups the list via the selected column. The selected column must be dragged to the specified area in the page as shown in the following.

Opens a menu that provides you with the option to drag additional columns to the table. To remove a column form the table, drag the selected column heading to the Select Columns window.

The following columns can be added:

Allow Update – Displays whether or not the resource string can be modified.

Screen Modules – Displays the screen modules where the resource string appears.

Translation Max. Length – The maximum length of each base value’s translation string. For example, the base value Case has a Translation Max. Length of 5 characters. Therefore, the translation for Case in any language should not exceed 5 characters. Note that if the length is exceeded then the string may not be displayed properly in the RCM UI. If you plan on exceeding a specific string length, then you should view the GUI to verify that it is displayed properly in the RCM UI.

Description, Modules, and Screens columns include the relevant metadata on the string (the Base Value). This metadata is received from the Risk Case Manager.

Step - displays the state of each row (invalid/valid identifier). Provides you with the option to group all the invalid identifiers.

Opens a font window that enables you to customize the font attributes of the text that is displayed in the list.
