addBusinessUnit

ACMResult addBusinessUnit (in BusinessUnit businessUnit,
out int businessUnitId);

Description: This method accepts a BusinessUnit object and saves it in the RCM repository.

IN

  • businessUnit – A business unit that should be added to the repository. The name and identifier of the new object should be unique, and the internal id should be negative.

OUT

  • businessUnitId – The ID of the created business unit.
