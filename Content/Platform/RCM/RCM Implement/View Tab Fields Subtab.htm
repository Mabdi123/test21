View Tab Fields Subtab

This subtab enables you to define the fields that will be available for this view and the method in which these fields will be displayed.

The icons available in View Field area are as follows:

Icon

Description

Enables you to add a new View Field.

Displays the field’s properties

Clicking this icon shows only the fields configured to be viewed (e.g., Visible is set to true).

Clicking this icon shows only the fields configured to be sortable (e.g., Sortable is set to true).

Clicking this icon will show only the fields configured to be filterable (e.g., Filterable is set to true).

Clicking this icon will show only the fields configured to be exportable (e.g., Exportable is set to true).

To add a new View Field:

  1. Click the Add a new Field icon.
  2. The next available field appears in an alphabetical order (Common fields are displayed before Built In fields). If this is not the field that you wanted to add then click the arrow next to this field. This arrow enables you to scroll and select the specific field.
  3. Select the following properties for each field:
  • Visible – Select True for field to be visible in Risk Case Manager.
  • Sortable – Select True for field to be sortable in Risk Case Manager.
  • Filterable – Select True for field to be filterable in Risk Case Manager.
  • Exportable – Select True for field to be exportable in Risk Case Manager.
